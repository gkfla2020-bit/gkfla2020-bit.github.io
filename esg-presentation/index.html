  • 수출환어음 매입 시점 = 자연스러운 ESG 체크 접점

수익 모델 — 하나금융에 어떤 이익인가

직접 수익

  • 수출환어음 수수료: ESG 체크 포함 시 프리미엄 수수료
  • B2B SaaS: 기업 자가검진 월 30만원 (14일 무료 체험)
  • ESG 리포트: 기업별 맞춤 규제 대응 보고서 유료 제공
  • ESG 우대 금리: 컴플라이언스 우수 기업 우대 대출

간접 수익

  • 고객 유입: "하나은행 = ESG 자동 체크" 차별화
  • 외환 거래: 수출기업 록인 → 환전/파생상품 증가
  • 무역보험: ESG 리스크 데이터 기반 보험 상품
  • 글로벌: EU → 미국 CCA, 일본 GX 확장
