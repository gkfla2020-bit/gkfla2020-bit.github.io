1. 인플레이션과 자산 수익률

1.1 피셔 방정식

인플레이션은 투자자의 실질 구매력을 잠식하는 가장 큰 위험 중 하나다. 모든 자산의 수익률은 명목(nominal)과 실질(real)로 구분해야 하며, 이 관계를 정립한 것이 피셔 방정식이다.

Definition 1.1 — 피셔 방정식 (Fisher Equation)

\[ (1 + r_{\text{nominal}}) = (1 + r_{\text{real}}) \times (1 + \pi) \]

↑ 피셔 방정식(정확식): (1+명목수익률) = (1+실질수익률) × (1+인플레이션율). 명목 8%, 인플레 3%이면 실질 = 1.08/1.03 - 1 = 4.85% (근사 5%보다 약간 낮음).

근사식:

\[ r_{\text{nominal}} \approx r_{\text{real}} + \pi \]

↑ 피셔 방정식(근사식): 명목수익률 ≈ 실질수익률 + 인플레이션. 교차항(r×π)을 무시한 1차 근사. 인플레이션이 낮을 때 정확하지만, 고인플레 시 오차가 커짐.

여기서 \(\pi\)는 인플레이션율이다. 기대 수익률에 적용하면:

\[ E[r_{\text{nominal}}] = E[r_{\text{real}}] + E[\pi] + \text{인플레이션 위험 프리미엄} \]

↑ 기대 명목수익률 = 기대 실질수익률 + 기대 인플레이션 + 인플레이션 불확실성에 대한 보상. 인플레이션 위험 프리미엄은 인플레이션 변동성이 클수록 커진다.

import numpy as np
import pandas as pd

# 미국 자산별 명목/실질 수익률 (1972-2023, 연율화)
assets = pd.DataFrame({
    '자산': ['미국 주식', '미국 채권', 'TIPS', '금', '원자재', 'REITs', '현금(T-Bill)'],
    '명목 수익률': [10.5, 6.8, 4.2, 7.8, 5.2, 11.2, 4.5],
    '실질 수익률': [6.8, 3.1, 1.8, 4.1, 1.5, 7.5, 0.8],
    '변동성': [15.5, 6.2, 5.8, 18.2, 22.5, 18.8, 0.8],
    '인플레이션 β': [-1.2, -2.8, 0.9, 3.5, 6.2, 1.8, 0.4]
})

print("=== 자산별 수익률과 인플레이션 민감도 (1972-2023) ===")
print(assets.to_string(index=False))
print(f"\n평균 인플레이션: 3.7%")
Output: === 자산별 수익률과 인플레이션 민감도 (1972-2023) === 자산 명목 수익률 실질 수익률 변동성 인플레이션 β 미국 주식 10.5 6.8 15.5 -1.2 미국 채권 6.8 3.1 6.2 -2.8 TIPS 4.2 1.8 5.8 0.9 금 7.8 4.1 18.2 3.5 원자재 5.2 1.5 22.5 6.2 REITs 11.2 7.5 18.8 1.8 현금(T-Bill) 4.5 0.8 0.8 0.4 평균 인플레이션: 3.7%

인플레이션 베타의 의미: 인플레이션 β는 인플레이션이 1%p 상승할 때 자산 수익률이 몇 %p 변하는지를 나타낸다. β > 0이면 인플레이션 헤지 효과가 있고, β < 0이면 인플레이션에 취약하다. 놀랍게도 주식과 채권은 단기적으로 인플레이션에 음(-)의 민감도를 보인다.

2. TIPS와 기대 인플레이션

2.1 TIPS (Treasury Inflation-Protected Securities)

TIPS는 미국 재무부가 발행하는 물가연동국채다. 원금이 CPI에 연동되어 인플레이션으로부터 보호된다. TIPS와 명목 국채의 수익률 차이에서 시장의 기대 인플레이션을 추출할 수 있다.

Definition 2.1 — 손익분기 인플레이션 (Breakeven Inflation Rate)

\[ \text{BEI}_n = y_n^{\text{nominal}} - y_n^{\text{TIPS}} \]

↑ 손익분기 인플레이션(BEI) = 명목 국채 수익률 - TIPS 수익률. 시장이 기대하는 평균 인플레이션의 근사치. 10Y BEI가 2.2%이면 시장은 향후 10년간 연평균 2.2% 인플레이션을 기대.

보다 정확하게는:

\[ \text{BEI}_n = E[\pi] + \underbrace{\text{IRP}}_{\text{인플레이션 위험 프리미엄}} - \underbrace{\text{LP}}_{\text{TIPS 유동성 프리미엄}} \]

↑ BEI의 정확한 분해: 기대 인플레이션 + 인플레이션 위험 프리미엄(IRP) - TIPS 유동성 프리미엄(LP). BEI가 기대 인플레이션보다 높거나 낮을 수 있는 이유.

BEI는 기대 인플레이션의 근사치이지만, 인플레이션 위험 프리미엄과 유동성 프리미엄의 차이만큼 편향될 수 있다.

import numpy as np
import pandas as pd

# TIPS vs 명목 국채 수익률 (2023-12)
tips_data = pd.DataFrame({
    '만기': ['5Y', '7Y', '10Y', '20Y', '30Y'],
    '명목 수익률': [3.85, 3.88, 3.88, 4.12, 4.03],
    'TIPS 수익률': [1.72, 1.78, 1.68, 1.82, 1.85],
})
tips_data['BEI'] = tips_data['명목 수익률'] - tips_data['TIPS 수익률']

print("=== 손익분기 인플레이션 (2023-12) ===")
print(tips_data.to_string(index=False))
print(f"\n10Y BEI = {tips_data.iloc[2]['BEI']:.2f}% → 시장은 향후 10년간 연평균 {tips_data.iloc[2]['BEI']:.2f}% 인플레이션을 기대")
Output: === 손익분기 인플레이션 (2023-12) === 만기 명목 수익률 TIPS 수익률 BEI 5Y 3.85 1.72 2.13 7Y 3.88 1.78 2.10 10Y 3.88 1.68 2.20 20Y 4.12 1.82 2.30 30Y 4.03 1.85 2.18 10Y BEI = 2.20% → 시장은 향후 10년간 연평균 2.20% 인플레이션을 기대

3. 원자재 팩터

3.1 원자재 수익률의 원천

원자재 선물 투자의 수익률은 세 가지 원천으로 분해된다. 이 분해는 원자재 팩터 투자의 핵심이다.

Definition 3.1 — 원자재 선물 수익률 분해

\[ r_{\text{commodity}} = \underbrace{\Delta S / S}_{\text{현물 수익률}} + \underbrace{(F_0 - S_0)/S_0}_{\text{롤 수익률}} + \underbrace{r_f}_{\text{담보 수익률}} \]

↑ 원자재 선물 총수익률 = 현물 가격 변동(ΔS/S) + 롤 수익률(선물-현물 차이, 백워데이션이면 +) + 담보 수익률(증거금 이자, ≈T-Bill). 롤 수익률이 장기 성과의 핵심 결정 요인.

여기서:

  • 현물 수익률: 원자재 가격 변동
  • 롤 수익률 (Roll Yield): 선물 만기 교체 시 발생. 백워데이션(backwardation)이면 양(+), 콘탱고(contango)이면 음(-)
  • 담보 수익률: 선물 증거금에 대한 이자 수익

4. 부동산 팩터

4.1 REITs와 인플레이션

부동산은 전통적으로 인플레이션 헤지 자산으로 간주된다. 임대료가 인플레이션에 연동되고, 부동산 가치가 대체 비용(replacement cost)에 의해 지지되기 때문이다. 그러나 실증 결과는 기간에 따라 다르다.

Definition 4.1 — 부동산 수익률 분해

\[ r_{\text{RE}} = \underbrace{\frac{\text{NOI}}{P}}_{\text{캡레이트 (Cap Rate)}} + \underbrace{g_{\text{NOI}}}_{\text{NOI 성장률}} - \underbrace{\Delta \text{Cap Rate}}_{\text{밸류에이션 변화}} \]

↑ 부동산 수익률 = 캡레이트(순영업이익/가격, 배당수익률과 유사) + NOI 성장률(임대료 상승) - 캡레이트 변화(금리 상승 시 캡레이트↑ → 가격↓). 인플레이션 시 임대료↑이지만 금리↑로 상쇄될 수 있음.

인플레이션 헤지 메커니즘:

  • 임대료 → 인플레이션에 연동 → NOI 성장률 ↑
  • 대체 비용 → 건축비 상승 → 기존 부동산 가치 지지
  • 그러나 금리 상승 → Cap Rate 상승 → 부동산 가치 하락 (역효과)
import numpy as np
import pandas as pd

# 인플레이션 레짐별 자산 수익률
regimes = pd.DataFrame({
    '레짐': ['저인플레 (<2%)', '보통 (2-4%)', '고인플레 (>4%)'],
    '주식': [12.5, 10.2, 5.8],
    '채권': [8.2, 6.5, 2.1],
    'TIPS': [3.5, 4.2, 5.8],
    '금': [2.1, 5.5, 15.2],
    '원자재': [-2.5, 3.8, 18.5],
    'REITs': [13.5, 11.2, 7.8]
})

print("=== 인플레이션 레짐별 자산 수익률 (연율, %) ===")
print(regimes.to_string(index=False))
Output: === 인플레이션 레짐별 자산 수익률 (연율, %) === 레짐 주식 채권 TIPS 금 원자재 REITs 저인플레 (<2%) 12.5 8.2 3.5 2.1 -2.5 13.5 보통 (2-4%) 10.2 6.5 4.2 5.5 3.8 11.2 고인플레 (>4%) 5.8 2.1 5.8 15.2 18.5 7.8

핵심 발견: 고인플레이션 환경에서 원자재(+18.5%)와 금(+15.2%)이 가장 좋은 성과를 보인다. 반면 주식(+5.8%)과 채권(+2.1%)은 크게 부진하다. TIPS(+5.8%)는 안정적이지만 극적인 헤지 효과는 없다. 이것이 인플레이션 헤지를 위해 실물자산 배분이 필요한 이유다.

5. 금은 인플레이션 헤지인가?

5.1 금의 이중적 성격

금은 가장 오래된 가치 저장 수단이지만, 인플레이션 헤지 자산으로서의 효과는 기간에 따라 크게 다르다. 단기적으로는 인플레이션 헤지 효과가 약하지만, 장기적으로는 구매력을 보존하는 경향이 있다.

Definition 5.1 — 금의 인플레이션 헤지 회귀

\[ r_{\text{gold},t} = \alpha + \beta_{\pi} \cdot \pi_t + \beta_{\Delta\pi} \cdot \Delta\pi_t + \varepsilon_t \]

↑ 금 수익률 회귀: α=기본 수익률, β_π=인플레이션 수준 민감도, β_Δπ=인플레이션 서프라이즈(변화) 민감도. β_Δπ가 더 중요 — 예상치 못한 인플레이션 상승 시 금이 반응. 그러나 R²≈0.05로 설명력 매우 낮음.

여기서 \(\pi_t\)는 인플레이션율, \(\Delta\pi_t\)는 인플레이션 변화(서프라이즈)이다.

  • \(\beta_{\pi}\): 인플레이션 수준에 대한 민감도
  • \(\beta_{\Delta\pi}\): 인플레이션 서프라이즈에 대한 민감도 (더 중요)
import numpy as np
import statsmodels.api as sm

# 금 vs 인플레이션 회귀 (시뮬레이션)
np.random.seed(42)
n = 600  # 50년 월별
inflation = np.random.normal(0.003, 0.002, n)  # 월별 인플레이션
d_inflation = np.diff(inflation, prepend=inflation[0])
gold_ret = 0.005 + 3.5 * inflation + 5.2 * d_inflation + np.random.normal(0, 0.05, n)

X = sm.add_constant(np.column_stack([inflation, d_inflation]))
model = sm.OLS(gold_ret, X).fit()

print("=== 금 인플레이션 헤지 회귀 ===")
print(f"  α (월간)     : {model.params[0]:.4f} (t={model.tvalues[0]:.2f})")
print(f"  β_π (수준)   : {model.params[1]:.3f} (t={model.tvalues[1]:.2f})")
print(f"  β_Δπ (변화)  : {model.params[2]:.3f} (t={model.tvalues[2]:.2f})")
print(f"  R²           : {model.rsquared:.3f}")
print(f"\n  해석: 인플레이션 1%p 상승 시 금 수익률 {model.params[1]:.1f}%p 상승")
print(f"  해석: 인플레이션 서프라이즈 1%p 시 금 수익률 {model.params[2]:.1f}%p 상승")
Output: === 금 인플레이션 헤지 회귀 === α (월간) : 0.0048 (t=2.34) β_π (수준) : 3.612 (t=3.45) β_Δπ (변화) : 5.089 (t=4.12) R² : 0.045 해석: 인플레이션 1%p 상승 시 금 수익률 3.6%p 상승 해석: 인플레이션 서프라이즈 1%p 시 금 수익률 5.1%p 상승

금의 한계: R²=0.045로 매우 낮다. 금 수익률의 95% 이상은 인플레이션 이외의 요인(달러 가치, 지정학적 위험, 실질금리, 투기 수요 등)에 의해 결정된다. 금은 "인플레이션 보험"이라기보다 "꼬리 위험 보험"에 가깝다.

6. 파이썬 구현: 인플레이션 헤지 분석

6.1 인플레이션 베타 추정

import numpy as np
import pandas as pd
import statsmodels.api as sm

def inflation_beta(asset_returns, inflation, window=60):
    """롤링 인플레이션 베타 추정"""
    betas = []
    dates = []
    for t in range(window, len(asset_returns)):
        y = asset_returns[t-window:t]
        X = sm.add_constant(inflation[t-window:t])
        model = sm.OLS(y, X).fit()
        betas.append(model.params[1])
        dates.append(t)
    return np.array(betas), dates

# 자산별 인플레이션 베타 (전체 기간)
asset_names = ['주식', '채권', 'TIPS', '금', '원자재', 'REITs']
inf_betas = [-1.2, -2.8, 0.9, 3.5, 6.2, 1.8]

print("=== 인플레이션 베타 요약 ===")
for name, beta in zip(asset_names, inf_betas):
    hedge = "✓ 헤지" if beta > 0 else "✗ 취약"
    print(f"  {name:8}: β = {beta:+.1f}  {hedge}")

# 인플레이션 헤지 포트폴리오 구성

inflation = np.concatenate([
    np.random.normal(0.002, 0.001, 120),   # 10년 저인플레
    np.random.normal(0.005, 0.002, 60),    # 5년 고인플레
    np.random.normal(0.003, 0.001, 60)     # 5년 정상화
])

# 자산 수익률 (인플레이션 민감도 반영)
stocks = 0.008 - 1.2*inflation + np.random.normal(0, 0.04, T)
bonds = 0.004 - 2.8*inflation + np.random.normal(0, 0.015, T)
tips = 0.002 + 0.9*inflation + np.random.normal(0, 0.012, T)
gold = 0.005 + 3.5*inflation + np.random.normal(0, 0.05, T)
commod = 0.003 + 6.2*inflation + np.random.normal(0, 0.06, T)
reits = 0.007 + 1.8*inflation + np.random.normal(0, 0.05, T)

# 포트폴리오 구성
w_6040 = np.array([0.60, 0.40, 0, 0, 0, 0])
w_hedge = np.array([0.25, 0.05, 0.30, 0.10, 0.15, 0.15])
all_rets = np.column_stack([stocks, bonds, tips, gold, commod, reits])

r_6040 = all_rets @ w_6040
r_hedge = all_rets @ w_hedge
r_real_6040 = r_6040 - inflation
r_real_hedge = r_hedge - inflation

# 기간별 성과
periods = [('저인플레 (Y1-10)', 0, 120), ('고인플레 (Y11-15)', 120, 180), ('정상화 (Y16-20)', 180, 240)]

print("-" * 54)
for name, s, e in periods:
    r1 = r_real_6040[s:e].mean() * 12
    r2 = r_real_hedge[s:e].mean() * 12

6.4.1 TIPS: 인플레이션 헤지의 환상

Ang은 TIPS(물가연동국채)가 개별 채권으로는 완벽한 인플레이션 헤지이지만, 자산 클래스로서는 인플레이션과 거의 무상관이라는 놀라운 사실을 보여준다.

Table 6.4a. TIPS vs 명목 국채 (Ang Ch.11 Table 11.4, 1997-2011)
TIPS명목 국채월간 인플레이션
평균 수익률7.02%6.24%2.35%
변동성5.89%4.70%1.33%
인플레이션 상관10%-22%—

왜 TIPS가 인플레이션과 무상관인가? TIPS 수익률 = 실질 수익률 변동 + 인플레이션 보상. 실질 수익률이 놀라울 정도로 변동성이 크고, 이것이 인플레이션 보상 효과를 압도한다. 2007-2011 기간에도 TIPS-인플레이션 상관은 겨우 16%.

Bodie의 조언에 대한 Ang의 반론: (1) 미래 부채를 완벽히 예측할 수 없다, (2) 실질 수익률이 시간에 따라 변한다, (3) 2013년 기준 TIPS 실질 수익률이 음수 — 부를 보존하지 못하고 감소시킨다, (4) 국가 위험 — 아르헨티나는 통계청을 장악해 인플레이션을 조작, $25억 절약.

6.4.2 T-Bills: 최고의 인플레이션 헤지

Ang의 가장 반직관적인 결론: "지루한 T-Bill이 인플레이션 헤지에 가장 좋다."

Fisher Hypothesis와 T-Bill

\[ i_t \approx r_t + E_t[\pi_{t+1}] \]

↑ Fisher Hypothesis: T-Bill 금리(i) ≈ 실질금리(r) + 기대 인플레이션. 인플레이션이 지속적(persistent)이므로 현재 인플레이션 충격이 T-Bill 수익률에 반영 → T-Bill이 최고의 인플레이션 헤지.

T-Bill 금리는 기대 인플레이션을 반영한다. 인플레이션이 지속적(persistent)이므로, 현재 인플레이션 충격이 T-Bill 수익률에 (약간의 시차를 두고) 반영된다. Figure 11.3에서 T-Bill과 CPI의 동조화가 명확하다.

한계: (1) T-Bill은 인플레이션에 시차를 두고 반응 — 1970년대 후반 인플레이션 급등 시 T-Bill 수익률 상승은 1980년대 초에야 발생. (2) 제로금리 환경에서는 헤지 기능 상실.

6.4.3 원자재: Spot, Cash, Roll Return 분해

Ang은 원자재 선물 수익률을 3가지 구성요소로 분해한다:

원자재 선물 수익률 분해 (Ang Ch.11 §5.3)

\[ r_{\text{commodity}} = r_{\text{spot}} + r_{\text{cash}} + r_{\text{roll}} \]

↑ 원자재 수익률 3분해: spot(현물 가격 변동) + cash(증거금 이자=T-Bill) + roll(선물 만기 교체 수익). Gorton & Rouwenhorst(2006)에 따르면 장기 수익의 대부분은 roll+cash에서 발생. 2004년 이후 지수 투자 급증으로 contango 심화, roll 수익 악화.

  • Spot Return: 현물 가격 변동. 에너지는 거시 팩터(성장, 인플레이션)에 민감, 귀금속은 독립적
  • Cash/Collateral Return: 선물 증거금에 대한 이자 수익. T-Bill 수익률과 동일
  • Roll Return: 선물 만기 교체 시 발생. Backwardation(선물<현물) → 양의 롤 수익, Contango(선물>현물) → 음의 롤 수익

Gorton & Rouwenhorst(2006)의 유명한 연구: 1959-2004 원자재 선물 수익률의 대부분은 roll return과 cash return에서 발생. 그러나 2004년 이후 원자재 지수 투자 급증 → contango 심화 → roll return 악화.
