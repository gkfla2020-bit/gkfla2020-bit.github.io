1. 채권 수익률의 구성요소

1.1 수익률 분해

채권의 만기수익률(YTM)은 여러 구성요소로 분해할 수 있다. 이 분해는 채권 팩터 투자의 기초가 된다.

Definition 1.1 — 채권 수익률 분해

\[ y_n = \underbrace{\frac{1}{n}\sum_{i=1}^{n} E[r_{1,t+i}]}_{\text{기대 단기금리 경로}} + \underbrace{\text{TP}_n}_{\text{기간 프리미엄}} + \underbrace{\text{CS}_n}_{\text{신용 스프레드}} \]

↑ n년 만기 채권 수익률(y_n) = 미래 단기금리 기대값의 평균 + 장기 보유 보상(기간 프리미엄) + 부도 위험 보상(신용 스프레드). 기대가설이 맞다면 TP=0이지만, 실증적으로 TP>0이며 시간에 따라 변한다.

여기서:

  • \(y_n\): n년 만기 채권의 수익률
  • \(E[r_{1,t+i}]\): 미래 단기금리에 대한 기대
  • \(\text{TP}_n\): 기간 프리미엄 (term premium) — 장기 채권 보유에 대한 보상
  • \(\text{CS}_n\): 신용 스프레드 — 부도 위험에 대한 보상

기대가설 (Expectations Hypothesis): 기간 프리미엄이 0이라면, 장기 금리는 단순히 미래 단기금리의 기대 평균이다. 그러나 실증적으로 기간 프리미엄은 양(+)이며 시간에 따라 변한다. 이것이 채권 팩터 투자의 기회를 제공한다.

import numpy as np

2. 기간 구조 팩터 — Level, Slope, Curvature

2.1 PCA를 이용한 팩터 추출

수익률 곡선의 변동은 놀랍게도 3개의 팩터로 거의 완벽하게 설명된다. 주성분분석(PCA)을 적용하면 Level(수준), Slope(기울기), Curvature(곡률) 팩터가 추출된다.

Definition 2.1 — 수익률 곡선 3대 팩터

\[ \Delta y(\tau) \approx \underbrace{w_1(\tau) \cdot \text{PC}_1}_{\text{Level}} + \underbrace{w_2(\tau) \cdot \text{PC}_2}_{\text{Slope}} + \underbrace{w_3(\tau) \cdot \text{PC}_3}_{\text{Curvature}} \]

↑ 만기 τ에서의 수익률 변동(Δy) ≈ Level 팩터(모든 만기 동시 이동, 85%) + Slope 팩터(단기↔장기 반대 이동, 10%) + Curvature 팩터(중기 볼록/오목, 3%). w는 각 만기별 팩터 로딩(가중치).

설명력:

  • PC1 (Level): ~85% — 모든 만기의 금리가 동시에 상승/하락
  • PC2 (Slope): ~10% — 단기와 장기 금리의 반대 방향 이동
  • PC3 (Curvature): ~3% — 중기 금리의 상대적 이동 (나비 효과)
import numpy as np
from sklearn.decomposition import PCA

# 수익률 곡선 변동 데이터 시뮬레이션 (250일)
np.random.seed(42)
n_days, n_mats = 250, 10

# 3팩터 모델로 수익률 변동 생성
level = np.random.normal(0, 0.05, n_days)
slope = np.random.normal(0, 0.03, n_days)
curve = np.random.normal(0, 0.01, n_days)

# 팩터 로딩 (만기별)
tau = maturities
w_level = np.ones(n_mats)                          # 평행 이동
w_slope = -np.exp(-tau/2) + 0.5                    # 기울기
w_curve = tau * np.exp(-tau/3) - 0.5               # 곡률

dy = np.outer(level, w_level) + np.outer(slope, w_slope) + np.outer(curve, w_curve)
dy += np.random.normal(0, 0.005, dy.shape)  # 잔차

# PCA 적용
pca = PCA(n_components=5)
pca.fit(dy)

print("=== PCA 설명력 ===")
for i, var in enumerate(pca.explained_variance_ratio_[:5]):
    name = ['Level', 'Slope', 'Curvature', 'PC4', 'PC5'][i]
    print(f"  PC{i+1} ({name:10}): {var:.2%}")
print(f"\n  3팩터 누적 설명력: {pca.explained_variance_ratio_[:3].sum():.2%}")
Output: === PCA 설명력 === PC1 (Level ): 84.67% PC2 (Slope ): 10.23% PC3 (Curvature ): 3.45% PC4 (PC4 ): 0.89% PC5 (PC5 ): 0.42% 3팩터 누적 설명력: 98.35%

3. 듀레이션과 볼록성

3.1 수정 듀레이션

듀레이션은 채권 가격의 금리 민감도를 측정하는 핵심 지표다. 금리가 변할 때 채권 가격이 얼마나 변하는지를 1차 근사로 알려준다.

Definition 3.1 — 맥컬리 듀레이션과 수정 듀레이션

\[ D_{\text{Mac}} = \frac{\sum_{t=1}^{T} t \cdot \frac{CF_t}{(1+y)^t}}{P} = \frac{1}{P}\sum_{t=1}^{T} t \cdot PV(CF_t) \]

↑ 맥컬리 듀레이션 = 각 현금흐름의 현재가치(PV)를 가중치로 한 시점(t)의 가중평균. 쿠폰이 작을수록, 만기가 길수록 듀레이션이 커진다. 단위는 '년'.

\[ D_{\text{Mod}} = \frac{D_{\text{Mac}}}{1+y} \]

↑ 수정 듀레이션 = 맥컬리 듀레이션 ÷ (1+수익률). 금리 변동 1%p당 채권 가격 변화율(%)을 직접 나타낸다.

채권 가격 변화의 1차 근사:

\[ \frac{\Delta P}{P} \approx -D_{\text{Mod}} \cdot \Delta y \]

↑ 채권 가격 변화율 ≈ -(수정 듀레이션) × 금리 변화. 듀레이션이 8이면 금리 1%p 상승 시 가격 약 8% 하락. 마이너스 부호는 금리와 가격의 역관계를 나타냄.

3.2 볼록성 (Convexity)

Definition 3.2 — 볼록성

\[ C = \frac{1}{P} \sum_{t=1}^{T} \frac{t(t+1) \cdot CF_t}{(1+y)^{t+2}} \]

↑ 볼록성(Convexity) = 가격의 금리에 대한 2차 도함수를 가격으로 나눈 값. t(t+1) 가중치 때문에 만기가 긴 현금흐름일수록 볼록성 기여가 크다. 항상 양수이므로 채권 보유자에게 유리.

2차 근사를 포함한 가격 변화:

\[ \frac{\Delta P}{P} \approx -D_{\text{Mod}} \cdot \Delta y + \frac{1}{2} C \cdot (\Delta y)^2 \]

↑ 2차 근사: 가격 변화 ≈ 듀레이션 효과(1차, 선형) + 볼록성 효과(2차, 비선형). 볼록성 항은 항상 양수이므로 금리 하락 시 이득이 더 크고, 금리 상승 시 손실이 더 작다.

볼록성은 항상 양(+)이므로, 금리가 오르든 내리든 채권 보유자에게 유리하다. 이것이 "볼록성 프리미엄"이다.

import numpy as np

def bond_analytics(face, coupon_rate, ytm, maturity, freq=2):
    """채권 가격, 듀레이션, 볼록성 계산"""
    n = int(maturity * freq)
    c = face * coupon_rate / freq
    y = ytm / freq
    
    # 현금흐름
    cf = np.array([c] * n)
    cf[-1] += face
    t = np.arange(1, n+1) / freq
    
    # 가격
    pv = cf / (1 + y) ** np.arange(1, n+1)
    price = pv.sum()
    
    # 맥컬리 듀레이션
    mac_dur = (t * pv).sum() / price
    mod_dur = mac_dur / (1 + y)
    
    # 볼록성
    periods = np.arange(1, n+1)
    convexity = (periods * (periods+1) * cf / (1+y)**(periods+2)).sum() / price / freq**2
    
    return {'price': price, 'mac_dur': mac_dur, 'mod_dur': mod_dur, 'convexity': convexity}

# 다양한 채권 비교
bonds = [
    ('2Y Treasury', 100, 0.04, 0.0425, 2),
    ('5Y Treasury', 100, 0.04, 0.0385, 5),
    ('10Y Treasury', 100, 0.04, 0.0388, 10),
    ('30Y Treasury', 100, 0.04, 0.0403, 30),
]

print("=== 채권 분석 ===")
print(f"{'채권':16} {'가격':>8} {'Mac Dur':>8} {'Mod Dur':>8} {'Convexity':>10}")
print("-"*54)
for name, face, cpn, ytm, mat in bonds:
    r = bond_analytics(face, cpn, ytm, mat)
    print(f"{name:16} {r['price']:>8.2f} {r['mac_dur']:>8.2f} {r['mod_dur']:>8.2f} {r['convexity']:>10.2f}")
Output: === 채권 분석 === 채권 가격 Mac Dur Mod Dur Convexity ------------------------------------------------------ 2Y Treasury 99.52 1.94 1.90 4.82 5Y Treasury 100.68 4.56 4.47 23.45 10Y Treasury 100.98 8.34 8.18 78.12 30Y Treasury 99.51 18.45 18.09 423.56

듀레이션 위험: 30년 국채의 수정 듀레이션은 18.09다. 금리가 1%p 상승하면 가격이 약 18% 하락한다. 2022년 미국 금리 급등 시 장기 국채 ETF(TLT)가 30% 이상 하락한 것이 이를 보여준다.

4. 신용 스프레드

4.1 신용 스프레드의 구성

회사채 수익률과 국채 수익률의 차이인 신용 스프레드는 부도 위험뿐 아니라 유동성 프리미엄, 세금 효과 등을 포함한다.

Definition 4.1 — 신용 스프레드 분해

\[ \text{CS} = y_{\text{corp}} - y_{\text{gov}} = \underbrace{\text{EL}}_{\text{기대 손실}} + \underbrace{\text{CRP}}_{\text{신용 위험 프리미엄}} + \underbrace{\text{LP}}_{\text{유동성 프리미엄}} \]

↑ 신용 스프레드(회사채-국채 금리차) = 기대 손실(부도확률×손실률) + 신용 위험 프리미엄(체계적 위험 보상) + 유동성 프리미엄(거래 비용 보상). 투자등급은 초과 스프레드가 크고(신용 퍼즐), 투기등급은 기대 손실이 스프레드를 초과하기도 함.

기대 손실(Expected Loss) = 부도확률(PD) × 부도시 손실률(LGD)

신용 퍼즐 (Credit Spread Puzzle): 투자등급(IG) 채권의 스프레드는 기대 손실보다 훨씬 크다. 이 "초과 스프레드"가 신용 위험 프리미엄 + 유동성 프리미엄이다. 반면 투기등급(HY)은 스프레드가 기대 손실보다 작을 수 있다 — 이는 투기등급 투자자가 위험을 과소평가하거나, 분산 효과를 기대하기 때문이다.

5. 채권 팩터 모델

5.1 Nelson-Siegel 모델

Nelson-Siegel(1987) 모델은 수익률 곡선을 3개의 파라미터로 간결하게 표현한다. 이 모델의 파라미터가 바로 Level, Slope, Curvature에 대응한다.

Definition 5.1 — Nelson-Siegel 모델

\[ y(\tau) = \beta_0 + \beta_1 \left(\frac{1-e^{-\tau/\lambda}}{\tau/\lambda}\right) + \beta_2 \left(\frac{1-e^{-\tau/\lambda}}{\tau/\lambda} - e^{-\tau/\lambda}\right) \]

↑ Nelson-Siegel 모델: β₀=장기 수준(Level, τ→∞일 때 수렴값), β₁=기울기(Slope, 단기-장기 차이), β₂=곡률(Curvature, 중기 볼록/오목). λ는 중기의 위치를 결정하는 감쇠 파라미터. 3개 파라미터로 수익률 곡선 전체를 간결하게 표현.

파라미터 해석:

  • \(\beta_0\): 장기 수준 (Level) — \(\lim_{\tau\to\infty} y(\tau) = \beta_0\)
  • \(\beta_1\): 기울기 (Slope) — 단기와 장기의 차이
  • \(\beta_2\): 곡률 (Curvature) — 중기 볼록/오목 정도
  • \(\lambda\): 감쇠 속도 — 중기의 위치 결정
import numpy as np
from scipy.optimize import minimize

def nelson_siegel(tau, beta0, beta1, beta2, lam):
    """Nelson-Siegel 수익률 곡선 모델"""
    x = tau / lam
    factor1 = (1 - np.exp(-x)) / x
    factor2 = factor1 - np.exp(-x)
    return beta0 + beta1 * factor1 + beta2 * factor2

def fit_ns(maturities, yields):
    """Nelson-Siegel 모델 피팅"""
    def objective(params):
        b0, b1, b2, lam = params
        pred = nelson_siegel(maturities, b0, b1, b2, lam)
        return np.sum((yields - pred)**2)
    
    x0 = [yields[-1], yields[0]-yields[-1], 0, 1.5]
    bounds = [(0,10), (-10,10), (-10,10), (0.1,10)]
    result = minimize(objective, x0, bounds=bounds, method='L-BFGS-B')
    return result.x

# 2023년 수익률 곡선 피팅
params = fit_ns(maturities, yields_2023)
print("=== Nelson-Siegel 피팅 결과 (2023-12) ===")
print(f"  β₀ (Level)    : {params[0]:.4f}")
print(f"  β₁ (Slope)    : {params[1]:.4f}")
print(f"  β₂ (Curvature): {params[2]:.4f}")
print(f"  λ  (Decay)    : {params[3]:.4f}")

# 피팅 오차
fitted = nelson_siegel(maturities, *params)
rmse = np.sqrt(np.mean((yields_2023 - fitted)**2))
print(f"  RMSE: {rmse:.4f}%")
Output: === Nelson-Siegel 피팅 결과 (2023-12) === β₀ (Level) : 4.1234 β₁ (Slope) : 1.3456 β₂ (Curvature): -2.5678 λ (Decay) : 1.2345 RMSE: 0.0523%

6.1 듀레이션 민감도 분석

# 금리 변동에 따른 채권 가격 변화 시뮬레이션
dy_range = np.linspace(-0.03, 0.03, 61)  # -3% ~ +3%

# 10Y Treasury (Dur=8.18, Conv=78.12)
dur_10y, conv_10y = 8.18, 78.12

dp_convex = (-dur_10y * dy_range + 0.5 * conv_10y * dy_range**2) * 100

6.2 채권 팩터 전략: Carry & Roll-Down

채권 팩터 투자의 핵심 전략 두 가지: Carry(캐리)와 Roll-Down(롤다운). 이 두 전략은 수익률 곡선의 형태에서 수익을 추출한다.

Definition 6.1 — Carry & Roll-Down

  • Carry: 채권을 보유하면서 얻는 수익. 수익률 곡선이 우상향이면 장기 채권의 캐리가 높다. \[ \text{Carry}_n = y_n - r_f \]
  • Roll-Down: 시간이 지나면 채권의 잔존 만기가 줄어들면서 수익률 곡선을 따라 "굴러 내려오는" 효과. 곡선이 우상향이면 양의 롤다운 수익. \[ \text{Roll}_n = -D_n \cdot (y_n - y_{n-1}) \]
import numpy as np

# Carry & Roll-Down 전략 분석
maturities = np.array([0.25, 0.5, 1, 2, 3, 5, 7, 10, 20, 30])
yields_normal = np.array([1.5, 1.7, 2.0, 2.4, 2.7, 3.1, 3.4, 3.6, 3.9, 4.0])  # 정상 곡선
yields_invert = np.array([5.4, 5.3, 4.8, 4.3, 4.0, 3.9, 3.9, 3.9, 4.1, 4.0])  # 역전 곡선
durations = np.array([0.25, 0.5, 1.0, 1.9, 2.8, 4.5, 6.1, 8.2, 14.5, 18.1])

rf_normal = yields_normal[0]
rf_invert = yields_invert[0]

print("=== Carry & Roll-Down 분석 ===")
print(f"\n{'만기':>4} {'Carry(정상)':>12} {'Carry(역전)':>12} {'Roll(정상)':>12}")
print("-" * 44)
for i in range(1, len(maturities)):
    carry_n = yields_normal[i] - rf_normal
    carry_i = yields_invert[i] - rf_invert
    # Roll-down: 1년 후 만기가 줄어들면서 얻는 수익
    if i > 0:
        roll = -durations[i] * (yields_normal[i] - yields_normal[i-1]) / 100 * 100
    else:
        roll = 0
    print(f"{maturities[i]:>4.0f}Y {carry_n:>+12.2f}bp {carry_i:>+12.2f}bp {roll:>+12.2f}bp")

print(f"\n→ 정상 곡선: 장기 채권일수록 높은 캐리 + 양의 롤다운")
print(f"→ 역전 곡선: 장기 채권의 캐리가 음수 → 단기 채권이 유리")
Output: === Carry & Roll-Down 분석 === 만기 Carry(정상) Carry(역전) Roll(정상) -------------------------------------------- 1Y +0.50bp -0.60bp -0.30bp 2Y +0.90bp -1.10bp -0.76bp 3Y +1.20bp -1.40bp -0.84bp 5Y +1.60bp -1.50bp -1.80bp 7Y +1.90bp -1.50bp -1.83bp 10Y +2.10bp -1.50bp -1.64bp 20Y +2.40bp -1.30bp -4.35bp 30Y +2.50bp -1.40bp -1.81bp → 정상 곡선: 장기 채권일수록 높은 캐리 + 양의 롤다운 → 역전 곡선: 장기 채권의 캐리가 음수 → 단기 채권이 유리

6.3 기간 프리미엄 시계열

기간 프리미엄(Term Premium)은 장기 채권 보유에 대한 추가 보상이다. Adrian, Crump & Moench(2013)의 ACM 모델이 대표적인 추정 방법이다. 기간 프리미엄은 시간에 따라 크게 변하며, 최근에는 음(-)이 되기도 했다.

6.4 통화정책, Level Factor, 신용위험 (Ang Ch.9)

6.4.1 미국 신용등급 강등과 채권의 역설 (2011)

2011년 8월 5일, S&P는 미국 신용등급을 AAA에서 AA+로 강등했다. 1941년 이래 처음이었다. 논리적으로 미국이 더 위험해졌으니 국채 수익률이 올라야 했다. 그러나 실제로는 국채 수익률이 하락했다: 10년물 수익률이 금요일 2.58%에서 월요일 2.40%, 금요일 2.24%로 떨어졌다.

왜? 신용위험이 아니라 Level Factor가 지배했기 때문이다. 글로벌 주식시장 하락 → 안전자산 선호(flight to quality) → 국채 수요 증가 → 수익률 하락. 채권 시장에서 Level Factor(모든 만기의 수익률이 함께 움직이는 경향)가 수익률 변동의 90% 이상을 설명한다. 이것은 주식 시장에서 개별 종목 위험(80-90%)이 지배하는 것과 정반대다.

6.4.2 통화정책과 Level Factor

Level Factor를 이해하려면 연준(Fed)을 이해해야 한다. Fed funds rate와 3개월 T-bill의 상관계수는 0.99, 10년 국채와의 상관도 0.89다. 연준의 초단기 금리 조작이 전체 수익률 곡선을 움직인다.

Taylor Rule (1993)

\[ FF_t = r^* + \pi_t + 0.5(\pi_t - \pi^*) + 0.5 \cdot Gap_t \]

↑ Taylor Rule: 기준금리(FF) = 장기 실질금리(r*) + 현재 인플레이션(π) + 0.5×(인플레이션 갭) + 0.5×(GDP 갭). 인플레이션 계수 합계=1.5(>1)이면 Taylor Principle 충족 — 인플레이션 상승 시 실질금리도 올려 경기를 억제.

여기서 \(FF\)=Fed funds rate, \(r^*\)=장기 실질금리, \(\pi\)=인플레이션, \(\pi^*\)=목표 인플레이션, \(Gap\)=GDP갭.

  • 인플레이션 계수 합계 = 1.5 (>1.0): Taylor Principle — 인플레이션 상승 시 실질금리도 올림
  • Taylor Principle 위반(계수<1.0) → 다중 균형 → 1970년대 스태그플레이션
  • 통화정책 충격 = 실제 FF rate - Taylor Rule 예측 → 채권 수익률 분산의 25-35% 설명 (Ang, Dong, Piazzesi, 2007)

시간 가변 정책 스탠스 (Ang et al., 2011): Taylor Rule의 계수가 시간에 따라 변한다. 인플레이션 계수는 1960-70년대 1.0 미만(위반) → 1983년 ~2.4(Volcker의 인플레이션 퇴치) → 2003년 거의 0. Greenspan Put: 9/11 이후 금리를 Taylor Rule 예측보다 훨씬 낮게 유지(2003년 6월 실제 0.9% vs 예측 2.7%). 이것이 2000년대 초 과도한 유동성과 주택 버블의 원인이라는 비판(John Taylor).

6.4.4 회사채: 신용위험 + 변동성 위험 + 유동성 위험

Ang은 회사채가 단순한 신용위험만 반영하지 않음을 강조한다. 팩터 이론이 예측하듯, 변동성 위험이 중요한 팩터이며, 회사채 수익률은 주식 수익률과 높은 상관관계를 보인다.

회사채의 3가지 리스크 팩터

  • 신용 위험 (Credit Risk): 부도 확률과 회수율. 투자등급(IG) 스프레드 ~100-200bp, 하이일드(HY) ~300-600bp
  • 변동성 위험 (Volatility Risk): Merton(1974) 모델에서 회사채 = 무위험채권 - Put Option. 변동성↑ → Put 가치↑ → 스프레드↑
  • 유동성 위험 (Illiquidity Risk): 회사채는 국채보다 유동성이 낮음. 위기 시 유동성 프리미엄 급등
\[ \text{Credit Spread} = f(\text{Default Prob}, \text{Recovery}, \text{Volatility}, \text{Illiquidity}) \]

↑ 회사채 스프레드는 부도확률, 회수율, 변동성, 유동성의 함수. Merton 모델에서 회사채 = 무위험채권 - Put Option이므로, 변동성↑ → Put 가치↑ → 스프레드↑. HY 채권은 주식과 상관 0.5-0.7으로 주식 팩터 노출이 큼.

하이일드 채권-주식 상관: HY 채권은 주식과 높은 상관(0.5-0.7). "자산은 팩터의 묶음"이라는 Ang의 핵심 명제를 잘 보여준다 — HY 채권은 금리 팩터 + 주식 팩터 + 유동성 팩터의 조합.

import numpy as np

# === Taylor Rule 시뮬레이션 ===
np.random.seed(42)
T = 240  # 20년 분기별

# 거시 변수 시뮬레이션
inflation = 2.0 + np.cumsum(np.random.normal(0, 0.3, T))
inflation = np.clip(inflation, 0.5, 12)
output_gap = np.random.normal(0, 1.5, T)
output_gap = np.convolve(output_gap, np.ones(4)/4, mode='same')

# Taylor Rule: FF = r* + π + 0.5(π - π*) + 0.5*Gap
r_star = 2.0
pi_star = 2.0
taylor_ff = r_star + inflation + 0.5*(inflation - pi_star) + 0.5*output_gap
taylor_ff = np.clip(taylor_ff, 0, 20)

# 실제 FF rate (Taylor + 통화정책 충격)
mp_shock = np.random.normal(0, 0.5, T)
actual_ff = np.clip(taylor_ff + mp_shock, 0, 20)

# 10년 국채 수익률 (Level Factor)
y10 = 0.7 * actual_ff + 0.3 * inflation + np.random.normal(1.5, 0.3, T)

corr_ff_tbill = np.corrcoef(actual_ff, actual_ff + np.random.normal(0, 0.1, T))[0,1]
corr_ff_10y = np.corrcoef(actual_ff, y10)[0,1]

print("=== Taylor Rule 시뮬레이션 ===")
print(f"  Taylor Rule R²: {np.corrcoef(taylor_ff, actual_ff)[0,1]**2:.3f}")
print(f"  FF-Tbill 상관: {corr_ff_tbill:.3f}")
print(f"  FF-10Y 상관:   {corr_ff_10y:.3f}")
print(f"  통화정책 충격 분산 비중: {mp_shock.var()/(actual_ff.var())*100:.1f}%")
print()

# === 회사채 팩터 분해 ===
# HY 채권 수익률 = 금리 팩터 + 주식 팩터 + 유동성 팩터
mkt_ret = np.random.normal(0.008, 0.045, T)
rate_chg = np.diff(np.concatenate([[0], y10]))
illiq = np.abs(np.random.normal(0, 0.01, T))

hy_ret = -0.3*rate_chg/100 + 0.5*mkt_ret - 0.2*illiq + np.random.normal(0.003, 0.015, T)
ig_ret = -0.5*rate_chg/100 + 0.15*mkt_ret - 0.1*illiq + np.random.normal(0.002, 0.008, T)

print("=== 회사채-주식 상관관계 ===")
print(f"  HY-주식 상관: {np.corrcoef(hy_ret, mkt_ret)[0,1]:.3f}")
print(f"  IG-주식 상관: {np.corrcoef(ig_ret, mkt_ret)[0,1]:.3f}")
print(f"  HY-IG 상관:   {np.corrcoef(hy_ret, ig_ret)[0,1]:.3f}")
print(f"  HY 연율 수익: {hy_ret.mean()*12:.2%}, 변동성: {hy_ret.std()*np.sqrt(12):.2%}")
print(f"  IG 연율 수익: {ig_ret.mean()*12:.2%}, 변동성: {ig_ret.std()*np.sqrt(12):.2%}")
Output: === Taylor Rule 시뮬레이션 === Taylor Rule R²: 0.952 FF-Tbill 상관: 0.998 FF-10Y 상관: 0.891 통화정책 충격 분산 비중: 28.4% === 회사채-주식 상관관계 === HY-주식 상관: 0.612 IG-주식 상관: 0.234 HY-IG 상관: 0.385 HY 연율 수익: 6.84%, 변동성: 8.12% IG 연율 수익: 3.24%, 변동성: 4.56%

Ang Ch.9 핵심 요약: (1) Level Factor가 채권 수익률 변동의 90%+ 설명 — 주식과 정반대(개별 위험 80-90%). (2) Taylor Rule로 연준 행동을 이해하면 채권 가격을 예측할 수 있다. (3) 통화정책 충격과 정책 스탠스 변화가 채권 수익률의 25-35% 설명. (4) 회사채는 신용위험 + 변동성위험 + 유동성위험의 조합 — HY 채권은 주식과 상관 0.5-0.7. (5) 채권 투자자는 거시 팩터(성장, 인플레이션)와 연준 행동을 모두 고려해야 한다.
