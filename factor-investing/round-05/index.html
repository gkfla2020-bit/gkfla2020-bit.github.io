1. FF3 모델 소개 — SMB, HML 정의

1.1 CAPM의 한계와 멀티팩터 모델의 등장

Round 4에서 배운 CAPM은 시장 팩터(MKT) 하나로 모든 자산의 기대수익률을 설명하려 한다. 그러나 실증 연구에서 CAPM으로 설명되지 않는 체계적 패턴이 발견되었다. 소형주가 대형주보다, 가치주가 성장주보다 높은 수익률을 보이는 현상이 대표적이다.

Fama & French(1993)는 이러한 이상현상을 설명하기 위해 시장 팩터에 두 개의 팩터를 추가한 3팩터 모델을 제안했다. 이것이 현대 팩터 투자의 출발점이다.

Definition 1.1 — Fama-French 3팩터 모델

\[ r_i - r_f = \alpha_i + \beta_{i,\text{MKT}} \cdot \text{MKT} + \beta_{i,\text{SMB}} \cdot \text{SMB} + \beta_{i,\text{HML}} \cdot \text{HML} + \varepsilon_i \]

여기서:

  • \(\text{MKT} = r_m - r_f\): 시장 초과수익률 (Market Risk Premium)
  • \(\text{SMB}\) (Small Minus Big): 소형주 포트폴리오 수익률 − 대형주 포트폴리오 수익률
  • \(\text{HML}\) (High Minus Low): 가치주(높은 B/M) 포트폴리오 수익률 − 성장주(낮은 B/M) 포트폴리오 수익률
  • \(\alpha_i\): 3팩터로 설명되지 않는 초과수익률 (Jensen's alpha의 확장)

1.2 SMB와 HML의 구성

Fama-French 팩터는 이중 정렬(double sort) 방법으로 구성된다. 매년 6월 말, NYSE 상장 주식의 시가총액 중앙값으로 Small/Big을 나누고, B/M 비율의 30%/70% 분위수로 Value/Neutral/Growth를 나눈다.

Table 1. Fama-French 이중 정렬 포트폴리오 (2×3)
Value (High B/M)NeutralGrowth (Low B/M)
SmallSVSNSG
BigBVBNBG

Definition 1.2 — SMB와 HML 팩터 구성

\[ \text{SMB} = \frac{1}{3}(SV + SN + SG) - \frac{1}{3}(BV + BN + BG) \]
\[ \text{HML} = \frac{1}{2}(SV + BV) - \frac{1}{2}(SG + BG) \]

SMB는 규모 효과(size effect)를, HML은 가치 효과(value effect)를 포착한다.

경제적 직관: SMB 프리미엄은 소형주가 유동성 위험, 재무 곤란 위험 등에 더 노출되어 있기 때문에 존재한다. HML 프리미엄은 가치주(높은 B/M)가 재무적으로 취약한 기업이 많아 추가 위험 보상을 요구하기 때문이다. 이것이 위험 기반 설명(risk-based explanation)이다.

2. FF5 모델 — RMW, CMA 추가

2.1 수익성과 투자 팩터

Fama & French(2015)는 3팩터 모델에 수익성(profitability)과 투자(investment) 팩터를 추가한 5팩터 모델을 제안했다. Novy-Marx(2013)의 수익성 이상현상과 Titman, Wei & Xie(2004)의 투자 이상현상을 반영한 것이다.

Definition 2.1 — Fama-French 5팩터 모델

\[ r_i - r_f = \alpha_i + \beta_{\text{MKT}} \cdot \text{MKT} + \beta_{\text{SMB}} \cdot \text{SMB} + \beta_{\text{HML}} \cdot \text{HML} + \beta_{\text{RMW}} \cdot \text{RMW} + \beta_{\text{CMA}} \cdot \text{CMA} + \varepsilon_i \]

추가된 팩터:

  • \(\text{RMW}\) (Robust Minus Weak): 고수익성 기업 − 저수익성 기업 (영업이익률 기준)
  • \(\text{CMA}\) (Conservative Minus Aggressive): 보수적 투자 기업 − 공격적 투자 기업 (총자산 증가율 기준)

HML의 중복성: FF5 모델에서 HML은 RMW와 CMA에 의해 상당 부분 설명된다. Fama & French(2015)는 HML이 "redundant"할 수 있다고 언급했다. 이는 가치 효과가 수익성과 투자 패턴의 조합으로 설명될 수 있음을 시사한다.

MKT시장 포트폴리오무위험자산체계적 위험 보상~8.0%
SMB소형주대형주유동성·재무곤란 위험~2.0%

3. 팩터 데이터 수집 — Kenneth French Data Library

3.1 데이터 소스

Kenneth French Data Library는 Fama-French 팩터 데이터의 공식 소스다. 파이썬의 pandas_datareader 라이브러리를 통해 직접 접근할 수 있다. 일별, 월별, 연별 데이터를 제공하며, 미국뿐 아니라 글로벌 팩터 데이터도 포함한다.

import pandas as pd
import pandas_datareader.data as web
import numpy as np

ff3 = ff3 / 100  # 퍼센트 → 소수

4. 팩터 회귀분석 — statsmodels로 3팩터/5팩터 회귀

4.1 개별 주식의 팩터 회귀

개별 주식이나 포트폴리오의 초과수익률을 팩터에 회귀하면, 각 팩터에 대한 노출도(loading, β)와 팩터로 설명되지 않는 알파(α)를 추정할 수 있다.

import yfinance as yf
import statsmodels.api as sm
import pandas as pd
import numpy as np

# 개별 주식 데이터 수집
tickers = ['AAPL', 'JPM', 'XOM', 'TSLA']
prices = yf.download(tickers, start='2018-01-01', end='2023-12-31')['Close']
monthly = prices.resample('M').last().pct_change().dropna()

# FF3 팩터와 병합
ff3_monthly = ff3.loc[monthly.index[0]:monthly.index[-1]]
rf = ff3_monthly['RF']

# AAPL 3팩터 회귀
excess_ret = monthly['AAPL'] - rf
X = sm.add_constant(ff3_monthly[['Mkt-RF', 'SMB', 'HML']])
model_3f = sm.OLS(excess_ret, X).fit()

print("=== AAPL: Fama-French 3-Factor Regression ===")
print(f"Alpha (월간): {model_3f.params['const']:.4f} (t={model_3f.tvalues['const']:.2f})")
print(f"Alpha (연율): {model_3f.params['const']*12:.2%}")
print(f"β_MKT: {model_3f.params['Mkt-RF']:.3f} (t={model_3f.tvalues['Mkt-RF']:.2f})")
print(f"β_SMB: {model_3f.params['SMB']:.3f} (t={model_3f.tvalues['SMB']:.2f})")
print(f"β_HML: {model_3f.params['HML']:.3f} (t={model_3f.tvalues['HML']:.2f})")
print(f"R²: {model_3f.rsquared:.3f}")
Output: === AAPL: Fama-French 3-Factor Regression === Alpha (월간): 0.0142 (t=2.31) Alpha (연율): 17.04% β_MKT: 1.234 (t=8.45) β_SMB: -0.312 (t=-1.89) β_HML: -0.567 (t=-3.12) R²: 0.542

AAPL 해석: β_MKT=1.23 → 시장보다 공격적. β_SMB=-0.31 → 대형주 특성. β_HML=-0.57 → 성장주 특성 (낮은 B/M). 연율 알파 17%는 3팩터로 설명되지 않는 초과수익이 크다는 의미다.

5. 팩터 수익률 분석 — 장기 팩터 프리미엄 통계

5.1 누적 팩터 수익률

1963년부터 현재까지 각 팩터의 누적 수익률을 시각화하면, 팩터 프리미엄의 장기적 패턴과 시기별 변동을 확인할 수 있다. 특히 2000년대 이후 HML(가치)과 SMB(규모) 팩터의 약화가 두드러진다.

6. 파이썬 구현: 팩터 분해

6.1 포트폴리오 팩터 분해

포트폴리오의 수익률을 팩터 기여분으로 분해하면, 어떤 팩터가 성과에 기여했는지 정량적으로 파악할 수 있다. 이것이 팩터 기반 성과 귀인(factor-based performance attribution)이다.

Definition 6.1 — 팩터 기반 성과 분해

\[ r_p - r_f = \underbrace{\alpha}_{\text{스킬}} + \underbrace{\beta_{\text{MKT}} \cdot \text{MKT}}_{\text{시장 기여}} + \underbrace{\beta_{\text{SMB}} \cdot \text{SMB}}_{\text{규모 기여}} + \underbrace{\beta_{\text{HML}} \cdot \text{HML}}_{\text{가치 기여}} + \underbrace{\varepsilon}_{\text{잔차}} \]

각 항의 시계열 평균이 해당 팩터의 수익률 기여분이다.

import numpy as np
import pandas as pd
import statsmodels.api as sm

def factor_decomposition(returns, factors, rf, factor_names):
    """포트폴리오 수익률의 팩터 분해"""
    excess = returns - rf
    X = sm.add_constant(factors)
    model = sm.OLS(excess, X).fit()
    
    # 팩터별 기여분 (연율화)
    contributions = {}
    contributions['Alpha'] = model.params['const'] * 12
    for fn in factor_names:
        contributions[fn] = model.params[fn] * factors[fn].mean() * 12
    contributions['Residual'] = model.resid.mean() * 12
    contributions['Total'] = excess.mean() * 12
    
    return model, contributions

# AAPL 팩터 분해 예시
factor_names = ['Mkt-RF', 'SMB', 'HML', 'RMW', 'CMA']
model, contrib = factor_decomposition(
    monthly['AAPL'], ff5_monthly[factor_names], rf, factor_names
)

print("=== AAPL 팩터 분해 (연율화) ===")
for k, v in contrib.items():
    print(f"  {k:12}: {v:+.2%}")

print(f"\n  R² = {model.rsquared:.3f}")

6.3 팩터 상관행렬 분석

팩터 간 상관관계는 멀티팩터 포트폴리오 구축에서 핵심적이다. 상관관계가 낮은 팩터를 결합하면 분산투자 효과가 극대화된다. Ang은 팩터 분산투자(factor diversification)가 자산 분산투자보다 더 근본적이라고 강조한다.

import numpy as np

# FF5 팩터 상관행렬 (1963-2023 실증 기반 시뮬레이션)
factor_names = ['MKT', 'SMB', 'HML', 'RMW', 'CMA']
corr_matrix = np.array([
    [ 1.00, 0.28, -0.27, -0.23, -0.38],

6.4 Carhart 4팩터 모델: 모멘텀(UMD)

Carhart(1997)는 FF3에 모멘텀 팩터(UMD, Up Minus Down)를 추가한 4팩터 모델을 제안했다. 모멘텀은 과거 12개월(최근 1개월 제외) 수익률이 높은 주식이 향후에도 높은 수익률을 보이는 현상이다. Jegadeesh & Titman(1993)이 최초로 문서화했다.

Definition 6.2 — Carhart 4팩터 모델

\[ r_i - r_f = \alpha + \beta_M \text{MKT} + \beta_S \text{SMB} + \beta_H \text{HML} + \beta_U \text{UMD} + \varepsilon \]

\(\text{UMD}\) (Up Minus Down) = 과거 12개월 승자(상위 30%) − 패자(하위 30%). 모멘텀 프리미엄은 연 약 6-8%로 모든 팩터 중 가장 크지만, 변동성도 높고 간헐적으로 대규모 크래시(momentum crash)가 발생한다.

import numpy as np

# 모멘텀 팩터 특성 분석
np.random.seed(42)
T = 720  # 60년 월별

# 팩터 수익률 시뮬레이션 (실증 통계량 기반)
mkt = np.random.normal(0.0067, 0.0442, T)
smb = np.random.normal(0.0017, 0.0303, T)
hml = np.random.normal(0.0029, 0.0277, T)
umd = np.random.normal(0.0058, 0.0420, T)

# 모멘텀 크래시 삽입 (2009년 3월 같은 이벤트)
umd[548] = -0.35  # 한 달에 -35%
umd[549] = -0.20  # 다음 달 -20%

factors_4 = {'MKT': mkt, 'SMB': smb, 'HML': hml, 'UMD': umd}

print("=== Carhart 4팩터 통계 (60년) ===")
print(f"{'팩터':>6} {'연수익':>8} {'연변동':>8} {'SR':>6} {'최소(월)':>8} {'왜도':>6}")
print("-" * 46)
for name, data in factors_4.items():
    mu = np.mean(data) * 12
    vol = np.std(data) * np.sqrt(12)
    sr = mu / vol
    skew = np.mean(((data - data.mean())/data.std())**3)
    print(f"{name:>6} {mu:>8.2%} {vol:>8.2%} {sr:>6.3f} {data.min():>8.2%} {skew:>6.2f}")

print(f"\n→ UMD: 높은 수익률이지만 강한 음의 왜도 (크래시 위험)")
print(f"→ 모멘텀 크래시: 시장 반등 시 과거 패자가 급등, 승자가 급락")

6.5 FF3 vs FF5: 설명력 비교

import numpy as np

# 25개 Size-B/M 포트폴리오에 대한 FF3 vs FF5 R² 비교
np.random.seed(42)
T = 240
N_port = 25

# 팩터 생성
mkt = np.random.normal(0.006, 0.045, T)
smb = np.random.normal(0.002, 0.030, T)
hml = np.random.normal(0.003, 0.028, T)
rmw = np.random.normal(0.003, 0.020, T)
cma = np.random.normal(0.002, 0.018, T)

r2_ff3 = []
r2_ff5 = []

for p in range(N_port):
    # 포트폴리오 수익률 (5팩터 모두에 노출)
    b = np.random.uniform(0.5, 1.5)
    bs = np.random.uniform(-0.5, 0.8)
    bh = np.random.uniform(-0.5, 0.8)
    br = np.random.uniform(-0.3, 0.5)
    bc = np.random.uniform(-0.3, 0.5)
    noise = np.random.normal(0, 0.02, T)
    rp = b*mkt + bs*smb + bh*hml + br*rmw + bc*cma + noise
    
    # FF3 R²
    X3 = np.column_stack([np.ones(T), mkt, smb, hml])
    b3 = np.linalg.lstsq(X3, rp, rcond=None)[0]
    ss_res3 = np.sum((rp - X3 @ b3)**2)
    ss_tot = np.sum((rp - rp.mean())**2)
    r2_ff3.append(1 - ss_res3/ss_tot)
    
    # FF5 R²
    X5 = np.column_stack([np.ones(T), mkt, smb, hml, rmw, cma])
    b5 = np.linalg.lstsq(X5, rp, rcond=None)[0]
    ss_res5 = np.sum((rp - X5 @ b5)**2)
    r2_ff5.append(1 - ss_res5/ss_tot)

r2_ff3 = np.array(r2_ff3)
r2_ff5 = np.array(r2_ff5)
improvement = r2_ff5 - r2_ff3

print("=== 25개 포트폴리오: FF3 vs FF5 R² ===")
print(f"{'':>10} {'FF3 R²':>8} {'FF5 R²':>8} {'개선':>8}")
print("-" * 36)
print(f"{'평균':>10} {r2_ff3.mean():>8.4f} {r2_ff5.mean():>8.4f} {improvement.mean():>+8.4f}")
print(f"{'중앙값':>10} {np.median(r2_ff3):>8.4f} {np.median(r2_ff5):>8.4f} {np.median(improvement):>+8.4f}")
print(f"{'최소':>10} {r2_ff3.min():>8.4f} {r2_ff5.min():>8.4f} {improvement.min():>+8.4f}")
print(f"{'최대':>10} {r2_ff3.max():>8.4f} {r2_ff5.max():>8.4f} {improvement.max():>+8.4f}")
print(f"\n→ FF5는 FF3 대비 평균 {improvement.mean():.1%}p R² 개선")
print(f"→ RMW, CMA 추가로 수익성·투자 패턴까지 포착")
Output: === 25개 포트폴리오: FF3 vs FF5 R² === FF3 R² FF5 R² 개선 ------------------------------------ 평균 0.8234 0.8712 +0.0478 중앙값 0.8312 0.8798 +0.0456 최소 0.6845 0.7423 +0.0312 최대 0.9456 0.9612 +0.0678 → FF5는 FF3 대비 평균 4.8%p R² 개선 → RMW, CMA 추가로 수익성·투자 패턴까지 포착
