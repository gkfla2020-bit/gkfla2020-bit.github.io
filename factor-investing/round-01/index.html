1. 왜 효용 이론인가

1.1 팩터 투자의 출발점

Andrew Ang의 교재는 팩터 투자를 다루기 전에 "투자자가 무엇을 원하는가"부터 시작한다. 이것이 효용 이론(Utility Theory)이다. 왜 여기서 시작할까?

모든 자산 가격 모델 — CAPM, Fama-French, APT — 의 근저에는 "투자자가 위험을 싫어한다"는 가정이 있다. 이 가정을 수학적으로 엄밀하게 표현하는 도구가 효용함수다. 효용함수 없이 팩터 프리미엄을 이해하는 것은 미적분 없이 물리학을 하는 것과 같다.

Ang Ch.2의 핵심 메시지: 투자자의 선호(preferences)를 수식으로 정의하면, 최적 포트폴리오가 자연스럽게 도출된다. 이것이 Part I "The Asset Owner"의 출발점이다.

교재 연동: Ang Ch.2 "Preferences"는 Von Neumann-Morgenstern 기대효용 이론에서 출발하여, 위험회피의 측정(Arrow-Pratt), CRRA/CARA 효용함수, 확실성 등가까지 다룬다. 이 라운드에서 이 모든 개념을 파이썬 코드로 구현한다.

1.2 직관: 왜 사람들은 도박을 싫어하는가

동전 던지기 게임을 생각하자. 앞면이 나오면 200만원을 받고, 뒷면이 나오면 0원을 받는다. 기대값은 100만원이다. 그런데 "지금 당장 80만원을 줄게"라고 하면, 대부분의 사람은 80만원을 선택한다.

기대값(100만원)보다 적은 확실한 금액(80만원)을 선호하는 것 — 이것이 위험회피(risk aversion)다. 그리고 100만원과 80만원의 차이인 20만원이 위험 프리미엄(risk premium)이다. 이 직관을 수식으로 표현하는 것이 이 라운드의 목표다.

import numpy as np

# 동전 던지기 게임
outcomes = np.array([2000000, 0])  # 200만원 or 0원
probs = np.array([0.5, 0.5])

expected_value = np.sum(outcomes * probs)
print(f"기대값: {expected_value:,.0f}원")

# 대부분의 사람은 확실한 80만원을 선택
certain_amount = 800000
risk_premium = expected_value - certain_amount
print(f"확실성 등가: {certain_amount:,.0f}원")
print(f"위험 프리미엄: {risk_premium:,.0f}원")
Output: 기대값: 1,000,000원 확실성 등가: 800,000원 위험 프리미엄: 200,000원

2. 효용함수의 정의

2.1 Von Neumann-Morgenstern 기대효용

효용함수 \(u(W)\)는 부(wealth) \(W\)를 "만족도"로 변환하는 함수다. 핵심 아이디어: 투자자는 부 자체가 아니라 부의 효용의 기대값을 극대화한다.

Definition 2.1 — 기대효용 (Expected Utility)

\[ E[u(W)] = \sum_{i} p_i \cdot u(W_i) \]

여기서 \(p_i\)는 상태 \(i\)의 확률, \(W_i\)는 상태 \(i\)에서의 부, \(u(\cdot)\)는 효용함수다. 연속 분포의 경우:

\[ E[u(W)] = \int u(W) \, f(W) \, dW \]

효용함수가 오목(concave)하면 — 즉 \(u''(W) < 0\)이면 — 투자자는 위험회피적이다. 이것이 Jensen's inequality에 의해 보장된다:

\[ E[u(W)] \leq u(E[W]) \quad \text{(오목함수의 성질)} \]

기대효용이 기대부의 효용보다 작다. 즉, 불확실한 100만원의 만족도가 확실한 100만원의 만족도보다 낮다. 이것이 위험회피의 수학적 정의다.

2.2 효용함수의 기본 성질

성질수학적 조건경제적 의미
단조증가\(u'(W) > 0\)부가 많을수록 좋다 (non-satiation)

import numpy as np
import matplotlib.pyplot as plt

# 로그 효용함수: u(W) = ln(W)
W = np.linspace(0.1, 10, 200)
u_log = np.log(W)

# 1차 도함수 (한계효용): u'(W) = 1/W > 0
u_prime = 1 / W

# 2차 도함수: u''(W) = -1/W^2 < 0 → 오목!
u_double_prime = -1 / W**2

fig, axes = plt.subplots(1, 3, figsize=(14, 4))

axes[0].plot(W, u_log, 'b-', lw=2)
axes[0].set_title('u(W) = ln(W)')
axes[0].set_xlabel('Wealth (W)')
axes[0].set_ylabel('Utility')
axes[0].grid(True, alpha=0.3)

axes[1].plot(W, u_prime, 'g-', lw=2)
axes[1].set_title("u'(W) = 1/W  (한계효용)")
axes[1].set_xlabel('Wealth (W)')
axes[1].set_ylabel("Marginal Utility")
axes[1].grid(True, alpha=0.3)

axes[2].plot(W, u_double_prime, 'r-', lw=2)
axes[2].axhline(y=0, color='k', ls='--', alpha=0.5)
axes[2].set_title("u''(W) = -1/W²  (항상 < 0)")
axes[2].set_xlabel('Wealth (W)')
axes[2].set_ylabel("Curvature")
axes[2].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

Jensen's Inequality 시각적 이해: 오목한 효용함수 위에서, 두 점을 잇는 직선(기대효용)은 항상 곡선(확실한 부의 효용) 아래에 있다. 이 "갭"이 위험 프리미엄이다.

3. 위험회피계수 (Arrow-Pratt Measures)

3.1 절대위험회피계수 (ARA)

효용함수의 오목한 정도를 측정하는 방법이 필요하다. Arrow와 Pratt가 독립적으로 제안한 측도가 절대위험회피계수(Absolute Risk Aversion, ARA)다.

Definition 3.1 — 절대위험회피계수 (ARA)

\[ A(W) = -\frac{u''(W)}{u'(W)} \]

\(u''(W)\)는 효용함수의 2차 도함수(오목성), \(u'(W)\)는 1차 도함수(한계효용)다. \(u''(W) < 0\)이므로 마이너스를 붙여 양수로 만든다. ARA가 클수록 더 위험회피적이다.

왜 \(u''(W)\)만 쓰지 않고 \(u'(W)\)로 나눌까? 효용함수는 양의 선형 변환(affine transformation)에 대해 불변이어야 하기 때문이다. \(v(W) = a \cdot u(W) + b\) (단, \(a > 0\))는 같은 선호를 나타내는데, \(u''\)만 쓰면 \(a\)에 따라 값이 달라진다. \(u''/u'\)로 정규화하면 \(a\)가 상쇄된다.

3.2 상대위험회피계수 (RRA)

Definition 3.2 — 상대위험회피계수 (RRA)

\[ R(W) = -\frac{W \cdot u''(W)}{u'(W)} = W \cdot A(W) \]

ARA에 부 \(W\)를 곱한 것이다. ARA는 "절대 금액 기준"의 위험회피를, RRA는 "부의 비율 기준"의 위험회피를 측정한다.

직관적으로: ARA는 "1만원짜리 도박을 얼마나 싫어하는가", RRA는 "전 재산의 1%짜리 도박을 얼마나 싫어하는가"를 측정한다. 부자가 되면 1만원짜리 도박은 덜 무섭지만(ARA 감소), 전 재산의 1%짜리 도박은 여전히 무섭다(RRA 일정). 이것이 CRRA 모델의 핵심이다.

import numpy as np

def compute_ara_rra(u_prime_func, u_double_prime_func, W):
    """ARA와 RRA를 계산하는 범용 함수"""
    u_p = u_prime_func(W)
    u_pp = u_double_prime_func(W)
    ARA = -u_pp / u_p
    RRA = W * ARA
    return ARA, RRA

# 로그 효용: u(W) = ln(W)
# u'(W) = 1/W,  u''(W) = -1/W²
u_prime_log = lambda W: 1/W
u_dprime_log = lambda W: -1/W**2

W_values = np.array([1, 5, 10, 50, 100])
print(f"{'W':>6} {'ARA':>10} {'RRA':>10}")
print("-" * 28)
for W in W_values:
    ara, rra = compute_ara_rra(u_prime_log, u_dprime_log, W)
    print(f"{W:>6.0f} {ara:>10.4f} {rra:>10.4f}")
Output: W ARA RRA ---------------------------- 1 1.0000 1.0000 5 0.2000 1.0000 10 0.1000 1.0000 50 0.0200 1.0000 100 0.0100 1.0000

핵심 관찰: 로그 효용함수에서 ARA = 1/W (부가 늘면 감소), RRA = 1 (항상 일정). 이것이 CRRA(Constant Relative Risk Aversion)의 가장 단순한 예다. 부자가 되어도 "전 재산의 비율"로 보면 위험 태도가 변하지 않는다.

4. CRRA vs CARA 효용함수

4.1 CRRA (Constant Relative Risk Aversion)

Definition 4.1 — CRRA 효용함수

\[ u(W) = \begin{cases} \dfrac{W^{1-\gamma}}{1-\gamma} & \text{if } \gamma \neq 1 \\[8pt] \ln(W) & \text{if } \gamma = 1 \end{cases} \]

여기서 \(\gamma > 0\)는 상대위험회피계수(RRA)다. \(\gamma\)가 클수록 더 위험회피적이다.

CRRA 효용함수의 Arrow-Pratt 측도를 확인하자:

\[ u'(W) = W^{-\gamma}, \quad u''(W) = -\gamma W^{-\gamma-1} \]

↑ CRRA 효용의 1차·2차 도함수. \(u'(W)\)는 한계효용(부가 1원 늘 때 만족도 증가분), \(u''(W)\)는 한계효용의 변화율(음수 = 체감)이다.

\[ A(W) = -\frac{u''(W)}{u'(W)} = \frac{\gamma}{W}, \quad R(W) = W \cdot A(W) = \gamma \]

↑ CRRA의 핵심 성질: 절대위험회피(ARA)는 부가 늘면 감소(\(\gamma/W\)), 상대위험회피(RRA)는 항상 \(\gamma\)로 일정. 부자가 되어도 "전 재산의 X%" 도박에 대한 태도는 변하지 않는다.

RRA가 상수 \(\gamma\)로 일정하다. 이것이 "Constant" Relative Risk Aversion이라 불리는 이유다. 실증 연구에서 \(\gamma\)의 추정치는 보통 2~10 사이다. Ang 교재에서는 \(\gamma = 5\) 정도를 "합리적인" 값으로 자주 사용한다.

4.2 CARA (Constant Absolute Risk Aversion)

Definition 4.2 — CARA 효용함수 (지수 효용)

\[ u(W) = -\frac{1}{\alpha} e^{-\alpha W}, \quad \alpha > 0 \]

여기서 \(\alpha\)는 절대위험회피계수(ARA)다.

\[ u'(W) = e^{-\alpha W}, \quad u''(W) = -\alpha e^{-\alpha W} \]

↑ CARA 효용의 도함수. 지수함수의 미분이 자기 자신의 상수배라는 성질 덕분에 ARA가 상수가 된다.

\[ A(W) = \alpha \quad \text{(상수)}, \quad R(W) = \alpha W \quad \text{(부에 비례)}\]

↑ CARA의 핵심: ARA가 부와 무관하게 항상 \(\alpha\). 부자든 가난하든 "1만원짜리 도박"에 대한 태도가 같다 — 비현실적이라 실무에서는 CRRA를 더 많이 쓴다.

CARA에서는 ARA가 상수다. 문제는 RRA가 부에 비례하여 증가한다는 것이다. 부자가 될수록 "전 재산 비율 기준"으로 더 위험회피적이 된다는 뜻인데, 이는 현실과 맞지 않는다. 그래서 자산 가격 이론에서는 CRRA가 더 널리 사용된다.

주요 용도자산 가격 이론, C-CAPM정규분포 가정 하 분석
import numpy as np
import matplotlib.pyplot as plt

W = np.linspace(0.01, 5, 300)

# CRRA 효용함수
def crra_utility(W, gamma):
    """CRRA: u(W) = W^(1-γ)/(1-γ) for γ≠1, ln(W) for γ=1"""
    if gamma == 1:
        return np.log(W)
    return W**(1 - gamma) / (1 - gamma)

# CARA 효용함수
def cara_utility(W, alpha):
    """CARA: u(W) = -(1/α)exp(-αW)"""
    return -(1/alpha) * np.exp(-alpha * W)

fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# CRRA: 다양한 γ 값
for gamma in [0.5, 1, 2, 5, 10]:
    u = crra_utility(W, gamma)
    label = f"γ={gamma}" if gamma != 1 else "γ=1 (log)"
    axes[0].plot(W, u, label=label, lw=1.5)
axes[0].set_title("CRRA Utility Functions")
axes[0].set_xlabel("Wealth (W)")
axes[0].set_ylabel("u(W)")
axes[0].legend()
axes[0].set_ylim(-10, 5)
axes[0].grid(True, alpha=0.3)

# CARA: 다양한 α 값
for alpha in [0.5, 1, 2, 5]:
    u = cara_utility(W, alpha)
    axes[1].plot(W, u, label=f"α={alpha}", lw=1.5)
axes[1].set_title("CARA Utility Functions")
axes[1].set_xlabel("Wealth (W)")
axes[1].set_ylabel("u(W)")
axes[1].legend()
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()

γ의 경제적 의미:

  • \(\gamma = 0\): 위험 중립 (risk neutral). 기대값만 본다.
  • \(\gamma = 1\): 로그 효용. Kelly criterion의 기반.
  • \(\gamma = 2\sim5\): 일반적인 투자자. 실증 연구의 주요 범위.
  • \(\gamma = 10+\): 극도로 위험회피적. Equity premium puzzle에서 등장.

5. 확실성 등가 (Certainty Equivalent)

5.1 정의와 도출

Definition 5.1 — 확실성 등가 (CE)

\[ u(CE) = E[u(W)] \]

↑ 확실성 등가(CE)의 정의: CE의 효용 = 불확실한 부의 기대효용. 즉, "이 도박 대신 확실하게 CE만큼 받겠다"는 금액.

\[ CE = u^{-1}\big(E[u(W)]\big) \]

↑ CE를 구하려면 기대효용을 계산한 뒤 효용함수의 역함수를 적용한다. 위험회피적 투자자는 항상 CE < E[W]이다.

확실성 등가(Certainty Equivalent)는 "불확실한 도박과 동일한 효용을 주는 확실한 금액"이다. 위험회피 투자자에게 CE < E[W]이다.

Definition 5.2 — 위험 프리미엄 (Risk Premium)

\[ \pi = E[W] - CE \]

위험 프리미엄은 투자자가 위험을 제거하기 위해 기꺼이 포기하는 금액이다. \(\pi > 0\)이면 위험회피, \(\pi = 0\)이면 위험중립, \(\pi < 0\)이면 위험선호.

5.2 Arrow-Pratt 근사

작은 도박(small gamble)에 대해, 위험 프리미엄은 다음과 같이 근사된다:

Theorem 5.1 — Arrow-Pratt 위험 프리미엄 근사

\[ \pi \approx \frac{1}{2} A(W) \cdot \sigma^2 \]

여기서 \(A(W)\)는 ARA, \(\sigma^2\)는 도박의 분산이다. 위험 프리미엄은 (1) 위험회피 정도와 (2) 위험의 크기에 비례한다.

이 근사식은 2차 테일러 전개에서 나온다. \(u(W + \tilde{z})\)를 \(W\) 주위에서 전개하면:

\[ E[u(W + \tilde{z})] \approx u(W) + u'(W)E[\tilde{z}] + \frac{1}{2}u''(W)E[\tilde{z}^2] \]

공정한 도박(\(E[\tilde{z}] = 0\))이면 \(E[\tilde{z}^2] = \sigma^2\)이므로:

\[ E[u(W + \tilde{z})] \approx u(W) + \frac{1}{2}u''(W)\sigma^2 \]

한편 \(u(CE) = u(W - \pi) \approx u(W) - u'(W)\pi\)이므로, 두 식을 같다고 놓으면 \(\pi \approx -\frac{u''(W)}{2u'(W)}\sigma^2 = \frac{1}{2}A(W)\sigma^2\).

import numpy as np

def certainty_equivalent_crra(outcomes, probs, gamma):
    """CRRA 효용함수에서 확실성 등가 계산
    
    Parameters
    ----------
    outcomes : array, 가능한 부의 수준
    probs : array, 각 결과의 확률
    gamma : float, 상대위험회피계수
    
    Returns
    -------
    CE : float, 확실성 등가
    """
    if gamma == 1:
        # 로그 효용
        eu = np.sum(probs * np.log(outcomes))
        CE = np.exp(eu)
    else:
        eu = np.sum(probs * outcomes**(1-gamma) / (1-gamma))
        CE = (eu * (1-gamma)) ** (1/(1-gamma))
    return CE

6. 기대효용과 포트폴리오 선택

6.1 평균-분산 효용

CARA 효용 + 정규분포 수익률, 또는 2차 효용함수를 가정하면, 기대효용은 평균과 분산만의 함수로 단순화된다. 이것이 평균-분산(Mean-Variance) 프레임워크의 이론적 기반이다.

Definition 6.1 — 평균-분산 효용 (Mean-Variance Utility)

\[ U = E[r_p] - \frac{\gamma}{2} \sigma_p^2 \]

여기서 \(E[r_p]\)는 포트폴리오 기대수익률, \(\sigma_p^2\)는 포트폴리오 분산, \(\gamma\)는 위험회피계수다. 투자자는 이 \(U\)를 극대화하는 포트폴리오를 선택한다.

이 식의 직관: 수익률이 높으면 좋고(+), 분산이 크면 나쁘다(-). \(\gamma\)는 이 둘 사이의 트레이드오프를 결정한다. \(\gamma\)가 크면 분산에 대한 페널티가 커져서 보수적인 포트폴리오를 선택한다.

6.2 최적 위험자산 배분

무위험자산(수익률 \(r_f\))과 위험자산(기대수익률 \(\mu\), 분산 \(\sigma^2\)) 두 개만 있을 때, 위험자산에 투자하는 최적 비중 \(w^*\)는:

Theorem 6.1 — 최적 위험자산 비중

\[ w^* = \frac{\mu - r_f}{\gamma \sigma^2} = \frac{\text{초과수익률}}{\text{위험회피} \times \text{분산}} \]

이 식은 \(U = r_f + w(\mu - r_f) - \frac{\gamma}{2}w^2\sigma^2\)를 \(w\)에 대해 미분하여 0으로 놓으면 도출된다.

import numpy as np

def optimal_risky_weight(mu, rf, sigma, gamma):
    """최적 위험자산 비중 계산
    
    Parameters
    ----------
    mu : float, 위험자산 기대수익률
    rf : float, 무위험이자율
    sigma : float, 위험자산 표준편차
    gamma : float, 위험회피계수
    
    Returns
    -------
    w_star : float, 최적 위험자산 비중
    """
    return (mu - rf) / (gamma * sigma**2)

7.1 효용함수 클래스

지금까지 배운 모든 개념을 하나의 파이썬 클래스로 통합한다. 이 클래스를 이후 라운드에서도 재사용할 것이다.

import numpy as np
from scipy.optimize import brentq

class CRRAUtility:
    """CRRA 효용함수 도구 모음
    
    Parameters
    ----------
    gamma : float
        상대위험회피계수 (γ > 0)
    """
    def __init__(self, gamma):
        assert gamma > 0, "γ must be positive"
        self.gamma = gamma
    
    def utility(self, W):
        """u(W) 계산"""
        if self.gamma == 1:
            return np.log(W)
        return W**(1 - self.gamma) / (1 - self.gamma)
    
    def marginal_utility(self, W):
        """u'(W) = W^(-γ)"""
        return W**(-self.gamma)
    
    def ara(self, W):
        """절대위험회피계수: A(W) = γ/W"""
        return self.gamma / W
    
    def rra(self, W):
        """상대위험회피계수: R(W) = γ (상수)"""
        return self.gamma
    
    def certainty_equivalent(self, outcomes, probs):
        """확실성 등가 계산"""
        eu = np.sum(probs * self.utility(outcomes))
        if self.gamma == 1:
            return np.exp(eu)
        return (eu * (1 - self.gamma)) ** (1 / (1 - self.gamma))
    
    def risk_premium(self, outcomes, probs):
        """위험 프리미엄 = E[W] - CE"""
        EW = np.sum(outcomes * probs)
        CE = self.certainty_equivalent(outcomes, probs)
        return EW - CE
    
    def optimal_weight(self, mu, rf, sigma):
        """최적 위험자산 비중: w* = (μ-rf)/(γσ²)"""
        return (mu - rf) / (self.gamma * sigma**2)
    
    def mv_utility(self, er, var):
        """평균-분산 효용: U = E[r] - (γ/2)σ²"""
        return er - (self.gamma / 2) * var

# 사용 예시

print(f"ARA(100) = {u.ara(100):.4f}")
print(f"RRA(100) = {u.rra(100):.1f}")

probs = np.array([0.5, 0.5])

7.2 현실적 효용함수: 손실회피, 습관, 실망회피 (Ang Ch.2 §4)

Ang은 Ch.2에서 평균-분산 효용의 한계를 지적하고, 네 가지 현실적 효용함수를 소개한다. 이들은 모두 "나쁜 시기(bad times)"를 더 풍부하게 정의한다.

Definition 7.1 — 손실회피 효용 (Loss Aversion / Prospect Theory)

Kahneman & Tversky(1979)의 전망이론. 투자자는 이익보다 손실에 더 민감하다.

\[ U(x) = \begin{cases} x^{\alpha} & \text{if } x \geq 0 \text{ (이익)} \\ -\lambda(-x)^{\alpha} & \text{if } x < 0 \text{ (손실)} \end{cases} \]

여기서 \(\lambda > 1\)은 손실회피계수. 실험적으로 \(\lambda \approx 2.25\) — 손실의 고통이 이익의 기쁨보다 2.25배 크다. 참조점(reference point) 대비 이익/손실을 평가한다.

Definition 7.2 — 습관 효용 (Habit Utility)

\[ U(C_t) = \frac{(C_t - H_t)^{1-\gamma}}{1-\gamma} \]

여기서 \(H_t\)는 습관 수준(subsistence level). 소비 \(C_t\)가 습관 \(H_t\)에 가까워지면 위험회피가 급격히 증가한다. 나쁜 시기 = 소비가 습관에 근접할 때. Campbell & Cochrane(1999)의 모델이 대표적이다.

Definition 7.3 — 실망회피 효용 (Disappointment Aversion)

Gul(1991)의 모델. 손실회피의 "합리적 사촌"이다. 참조점이 확실성 등가(CE)로 내생적으로 결정된다는 점이 전망이론과 다르다. 기대보다 나쁜 결과에 더 큰 가중치를 부여한다.

import numpy as np

# 손실회피 vs CRRA: 변동성 전략 투자 비중 비교
# Ang Ch.2 Figure 2.11 스타일

def loss_aversion_utility(x, lam=2.25, alpha=0.88):
    """Kahneman-Tversky 손실회피 효용"""
    return np.where(x >= 0, x**alpha, -lam * (-x)**alpha)

def crra_utility(W, gamma):
    """CRRA 효용"""
    if gamma == 1:
        return np.log(W)
    return W**(1-gamma) / (1-gamma)
