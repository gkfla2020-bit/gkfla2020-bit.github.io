1. 주식 리스크 프리미엄 추정

1.1 역사적 ERP

주식 리스크 프리미엄(Equity Risk Premium, ERP)은 주식 시장의 기대 초과수익률이다. 이것은 자산배분의 가장 중요한 입력 변수이며, 그 크기에 대한 논쟁은 금융경제학의 핵심 주제다.

Definition 1.1 — 주식 리스크 프리미엄

\[ \text{ERP} = E[r_m] - r_f \]

↑ 주식 리스크 프리미엄(ERP) = 주식 시장 기대수익률 - 무위험이자율. 역사적으로 미국 ERP는 산술 6-8%, 기하 5-6%. 추정 방법(역사적/서베이/내재/소비기반)에 따라 2-8%로 크게 달라짐.

추정 방법:

  • 역사적 평균: 과거 실현 수익률의 산술/기하 평균
  • 서베이: 전문가/CFO 설문조사
  • 내재 ERP: 현재 주가에서 역산 (Gordon Growth Model 등)
  • 소비 기반: 거시경제 모델에서 도출
import numpy as np
import pandas as pd
import pandas_datareader.data as web

# Fama-French 시장 팩터 데이터 (1926-2023)
ff = web.DataReader('F-F_Research_Data_Factors', 'famafrench', start='1926-07')[0]
ff = ff / 100
mkt = ff['Mkt-RF']

# 기간별 ERP 추정
periods = {
    '전체 (1926-2023)': mkt,
    '전후 (1946-2023)': mkt['1946':],
    '현대 (1963-2023)': mkt['1963':],
    '최근 (2000-2023)': mkt['2000':]
}

print("=== 기간별 ERP 추정 ===")
for name, data in periods.items():
    arith = data.mean() * 12
    geo = ((1 + data).prod() ** (12/len(data)) - 1)
    vol = data.std() * np.sqrt(12)
    t_stat = data.mean() / (data.std() / np.sqrt(len(data)))
    print(f"  {name:22}: 산술 {arith:.2%}, 기하 {geo:.2%}, σ {vol:.2%}, t={t_stat:.2f}")
Output: === 기간별 ERP 추정 === 전체 (1926-2023) : 산술 8.24%, 기하 6.45%, σ 18.52%, t=4.89 전후 (1946-2023) : 산술 8.04%, 기하 6.52%, σ 15.34%, t=4.92 현대 (1963-2023) : 산술 6.78%, 기하 5.23%, σ 15.12%, t=3.72 최근 (2000-2023) : 산술 7.12%, 기하 5.45%, σ 17.89%, t=2.14

산술 vs 기하 평균: 산술 평균은 항상 기하 평균보다 크다. 차이는 약 \(\frac{\sigma^2}{2}\)이다. 장기 투자자에게는 기하 평균이 더 적절하고, 단기 기대수익률에는 산술 평균이 적절하다. 어떤 것을 사용하느냐에 따라 ERP 추정치가 2%p 이상 달라질 수 있다.

2. 소비 기반 CAPM (C-CAPM)

2.1 오일러 방정식

소비 기반 자산가격결정 모델(Consumption-based CAPM)은 자산 가격을 소비자의 최적화 문제에서 도출한다. 대표 소비자가 기간 간 소비를 최적화할 때, 모든 자산의 가격은 오일러 방정식을 만족해야 한다.

Definition 2.1 — 오일러 방정식 (Euler Equation)

\[ E_t\left[\beta \frac{u'(C_{t+1})}{u'(C_t)} R_{i,t+1}\right] = 1 \]

↑ 오일러 방정식: 시간할인인자(β) × 한계효용 비율(미래/현재) × 자산 총수익률의 기대값 = 1. 모든 자산가격결정 모델의 근본 조건 — 소비자가 최적 소비를 할 때 성립.

여기서 \(\beta\)는 시간할인인자, \(u'(C)\)는 한계효용, \(R_{i,t+1}\)은 자산 i의 총수익률이다.

CRRA 효용함수 \(u(C) = \frac{C^{1-\gamma}}{1-\gamma}\)를 대입하면:

\[ E_t\left[\beta \left(\frac{C_{t+1}}{C_t}\right)^{-\gamma} R_{i,t+1}\right] = 1 \]

↑ CRRA 효용 대입: 소비 성장률(C_{t+1}/C_t)의 -γ승이 확률적 할인인자(SDF). γ가 클수록 소비 감소에 민감 → 위기 시 자산 가격 하락 폭이 커야 함.

\(\gamma\)는 상대적 위험회피계수(coefficient of relative risk aversion)이다.

2.2 C-CAPM에서의 리스크 프리미엄

오일러 방정식을 로그 정규 분포 가정 하에서 전개하면, 자산 i의 리스크 프리미엄은 소비 성장률과의 공분산에 비례한다:

Definition 2.2 — C-CAPM 리스크 프리미엄

\[ E[r_i] - r_f + \frac{\sigma_i^2}{2} = \gamma \cdot \text{Cov}(r_i, \Delta \ln C_{t+1}) \]

↑ C-CAPM 리스크 프리미엄: 자산의 초과수익(+Jensen 보정항) = 위험회피계수(γ) × 자산수익률과 소비성장률의 공분산. 소비와 함께 하락하는 자산일수록 높은 프리미엄 요구.

주식 시장 전체에 적용하면:

\[ \text{ERP} \approx \gamma \cdot \text{Cov}(r_m, \Delta \ln C) \approx \gamma \cdot \sigma_m \cdot \sigma_{\Delta c} \cdot \rho_{m,\Delta c} \]

↑ ERP = γ × 시장변동성 × 소비변동성 × 상관계수. 소비 변동성(σ_Δc≈2%)이 매우 작아서 관측된 ERP(6%)를 설명하려면 γ가 비현실적으로 커야 함 → EPP.

소비 성장률과 높은 공분산을 가진 자산은 "나쁜 시기에 나쁜 성과"를 보이므로 높은 프리미엄을 요구한다.

3. Equity Premium Puzzle

3.1 퍼즐의 핵심

Mehra & Prescott(1985)는 관측된 ERP(~6%)를 C-CAPM으로 설명하려면 비현실적으로 높은 위험회피계수가 필요하다는 것을 보였다. 이것이 유명한 Equity Premium Puzzle이다.

Definition 3.1 — Equity Premium Puzzle

\[ \gamma_{\text{implied}} = \frac{\text{ERP}}{\sigma_m \cdot \sigma_{\Delta c} \cdot \rho_{m,\Delta c}} \]

↑ 내재 위험회피계수: 관측된 ERP를 C-CAPM으로 역산. 분모가 매우 작아서(시장변동성×소비변동성×상관) γ가 비현실적으로 커짐.

실증 데이터: \(\text{ERP} \approx 6\%\), \(\sigma_m \approx 16\%\), \(\sigma_{\Delta c} \approx 2\%\), \(\rho \approx 0.2\)

\[ \gamma_{\text{implied}} = \frac{0.06}{0.16 \times 0.02 \times 0.2} \approx 94 \]

↑ 숫자 대입: ERP 6% ÷ (16% × 2% × 0.2) = 94. 실험 경제학에서 γ는 1~10 범위인데 94는 비현실적 → Equity Premium Puzzle의 핵심.

γ=94는 비현실적이다. 실험 경제학에서 추정된 γ는 보통 1~10 범위다.

import numpy as np

# Equity Premium Puzzle 계산
ERP = 0.06          # 역사적 ERP
sigma_m = 0.16      # 시장 변동성
sigma_dc = 0.02     # 소비 성장률 변동성
rho = 0.2           # 시장-소비 상관계수

gamma_implied = ERP / (sigma_m * sigma_dc * rho)
print(f"=== Equity Premium Puzzle ===")
print(f"  관측된 ERP: {ERP:.1%}")
print(f"  σ_market: {sigma_m:.1%}")
print(f"  σ_consumption: {sigma_dc:.1%}")
print(f"  ρ(market, consumption): {rho}")
print(f"  내재 γ: {gamma_implied:.1f}")
print(f"\n  합리적 γ 범위: 1~10")
print(f"  γ=5일 때 설명 가능한 ERP: {5*sigma_m*sigma_dc*rho:.2%}")
Output: === Equity Premium Puzzle === 관측된 ERP: 6.0% σ_market: 16.0% σ_consumption: 2.0% ρ(market, consumption): 0.2 내재 γ: 93.8 합리적 γ 범위: 1~10 γ=5일 때 설명 가능한 ERP: 0.32%

퍼즐의 해결 시도:

  • 습관 형성 (Habit Formation): Campbell & Cochrane(1999) — 소비가 습관 수준에 가까울 때 위험회피가 급증
  • 장기 위험 (Long-Run Risk): Bansal & Yaron(2004) — 소비 성장률의 작은 영구적 변동이 큰 프리미엄 생성
  • 재난 위험 (Rare Disasters): Barro(2006) — 대공황, 전쟁 등 극단적 사건의 가능성
  • 행동재무학: 손실회피, 근시안적 손실회피 (Benartzi & Thaler, 1995)

4. 저변동성 이상현상

4.1 CAPM의 예측 vs 현실

CAPM은 높은 베타(위험)의 자산이 높은 기대수익률을 가진다고 예측한다. 그러나 실증 연구에서는 놀랍게도 저변동성(저베타) 주식이 고변동성(고베타) 주식과 비슷하거나 더 높은 위험조정 수익률을 보인다. 이것이 저변동성 이상현상(Low Volatility Anomaly)이다.

Definition 4.1 — 저변동성 이상현상

실증적 발견: Security Market Line(SML)이 CAPM이 예측하는 것보다 평평하다.

\[ E[r_i] - r_f = \gamma_0 + \gamma_1 \beta_i \quad \text{where } \gamma_1 < E[r_m] - r_f \]

↑ SML 평탄화: 실증적으로 SML 기울기(γ₁)가 이론적 ERP보다 작고, 절편(γ₀)이 양수. 저베타 주식이 CAPM 예측보다 높은 수익, 고베타 주식이 낮은 수익 → BAB 이상현상의 근거.

즉, 고베타 주식은 CAPM 예측보다 낮은 수익률을, 저베타 주식은 CAPM 예측보다 높은 수익률을 보인다.

5. Betting Against Beta (BAB)

5.1 BAB 팩터 구성

Frazzini & Pedersen(2014)은 저변동성 이상현상을 체계적으로 포착하는 BAB(Betting Against Beta) 팩터를 제안했다. 핵심 아이디어는 레버리지 제약(leverage constraint)이다: 많은 투자자가 레버리지를 사용할 수 없거나 꺼리기 때문에, 고베타 주식을 과매수하고 저베타 주식을 과소평가한다.

Definition 5.1 — BAB 팩터

\[ r_{\text{BAB}} = \frac{1}{\beta_L}(r_L - r_f) - \frac{1}{\beta_H}(r_H - r_f) \]

↑ BAB(Betting Against Beta) 팩터: 저베타 포트폴리오를 레버리지(1/β_L)하여 매수 + 고베타 포트폴리오를 디레버리지(1/β_H)하여 매도. 베타 중립(시장 노출 0)이면서 SML 평탄화에서 수익 추출.

여기서:

  • \(r_L\): 저베타 포트폴리오 수익률, \(\beta_L\): 저베타 포트폴리오의 평균 베타
  • \(r_H\): 고베타 포트폴리오 수익률, \(\beta_H\): 고베타 포트폴리오의 평균 베타
  • 각 포트폴리오를 베타로 나누어 레버리지를 조정 → 시장 중립(market-neutral)
import numpy as np
import pandas as pd
import yfinance as yf

# BAB 팩터 구현 (간소화 버전)
def compute_bab(returns, market_returns, rf, lookback=60):
    """BAB 팩터 계산"""
    n_stocks = returns.shape[1]
    bab_returns = []
    
    for t in range(lookback, len(returns)):
        # 과거 60개월로 베타 추정
        window = slice(t-lookback, t)
        betas = []
        for i in range(n_stocks):
            cov = np.cov(returns.iloc[window, i], market_returns.iloc[window])
            beta = cov[0,1] / cov[1,1]
            betas.append(beta)
        betas = np.array(betas)
        
        # 베타 기준 정렬 → 상위/하위 분할
        median_beta = np.median(betas)
        low_mask = betas <= median_beta
        high_mask = betas > median_beta
        
        # 포트폴리오 수익률 (동일 가중)
        r_low = returns.iloc[t, low_mask].mean()
        r_high = returns.iloc[t, high_mask].mean()
        beta_low = betas[low_mask].mean()
        beta_high = betas[high_mask].mean()
        rf_t = rf.iloc[t]
        
        # BAB = (1/β_L)(r_L - rf) - (1/β_H)(r_H - rf)
        bab = (r_low - rf_t)/beta_low - (r_high - rf_t)/beta_high
        bab_returns.append(bab)
    
    return pd.Series(bab_returns, index=returns.index[lookback:])

print("BAB 팩터 구현 완료")
print("연율화 BAB 프리미엄 (미국, 1963-2023): ~8.6% (Frazzini & Pedersen)")
print("샤프비율: ~1.0 (매우 높음)")
Output: BAB 팩터 구현 완료 연율화 BAB 프리미엄 (미국, 1963-2023): ~8.6% (Frazzini & Pedersen) 샤프비율: ~1.0 (매우 높음)

BAB의 한계: BAB 팩터의 높은 샤프비율은 레버리지 조정 때문이다. 실제로 저베타 포트폴리오에 레버리지를 적용해야 하므로, 거래비용, 마진 요구, 레버리지 비용이 실현 수익률을 크게 감소시킨다. 또한 BAB는 시장 급락 시 큰 손실을 볼 수 있다 (2020년 3월 등).

6.2 레버리지 제약과 SML 평탄화 메커니즘

Frazzini & Pedersen(2014)의 핵심 통찰: 왜 SML이 평탄한가? 많은 투자자(뮤추얼펀드, 연기금)는 레버리지를 사용할 수 없다. 높은 수익을 원하면 고베타 주식을 매수할 수밖에 없다. 이 수요 압력이 고베타 주식의 가격을 올리고 기대수익률을 낮춘다.

Definition 6.1 — 레버리지 제약 하의 CAPM

\[ E[r_i] - r_f = \beta_i \cdot \psi + (\lambda - \psi) \]

↑ 레버리지 제약 CAPM: ψ=레버리지 제약 하의 시장 프리미엄, λ=제약 없는 프리미엄. 레버리지 제약(ψ<λ)이 있으면 SML이 평탄해지고 절편이 양수 → 저베타 프리미엄 발생.

여기서 \(\psi < \lambda = E[r_m] - r_f\). 레버리지 제약이 강할수록 \(\psi\)가 작아지고 SML이 더 평탄해진다. 극단적으로 \(\psi = 0\)이면 모든 자산의 기대수익률이 동일해진다.

import numpy as np

# BAB 팩터 시뮬레이션: 베타 분위별 포트폴리오
np.random.seed(42)
T = 240  # 20년 월별
N = 100  # 100개 주식

# 시장 수익률
mkt_sim = np.random.normal(0.007, 0.045, T)

# 주식별 베타 (0.3 ~ 2.0)
true_betas = np.random.uniform(0.3, 2.0, N)

# 주식 수익률: 저베타 주식에 약간의 양의 알파 (SML 평탄화)
stock_rets = np.zeros((T, N))
for i in range(N):
    alpha_i = 0.003 * (1.0 - true_betas[i])  # 저베타 → 양의 알파
    stock_rets[:, i] = alpha_i + true_betas[i] * mkt_sim + \
                       np.random.normal(0, 0.03 + 0.02*true_betas[i], T)

# 5분위 포트폴리오 구성
quintile_bounds = np.percentile(true_betas, [0, 20, 40, 60, 80, 100])
print("=== 베타 분위별 포트폴리오 (시뮬레이션) ===")
print(f"{'분위':>8} {'평균β':>6} {'연수익':>8} {'연변동':>8} {'SR':>6} {'CAPM α':>8}")
print("-" * 48)

for q in range(5):
    mask = (true_betas >= quintile_bounds[q]) & (true_betas < quintile_bounds[q+1] + 0.01)
    port_ret = stock_rets[:, mask].mean(axis=1)
    avg_beta = true_betas[mask].mean()
    ann_ret = port_ret.mean() * 12
    ann_vol = port_ret.std() * np.sqrt(12)
    sr = ann_ret / ann_vol
    capm_alpha = ann_ret - avg_beta * mkt_sim.mean() * 12
    label = ['Q1(Low)','Q2','Q3','Q4','Q5(High)'][q]
    print(f"{label:>8} {avg_beta:>6.2f} {ann_ret:>8.2%} {ann_vol:>8.2%} {sr:>6.3f} {capm_alpha:>+8.2%}")

# BAB 수익률
low_mask = true_betas <= np.median(true_betas)
high_mask = ~low_mask
r_low = stock_rets[:, low_mask].mean(axis=1)
r_high = stock_rets[:, high_mask].mean(axis=1)
b_low = true_betas[low_mask].mean()
b_high = true_betas[high_mask].mean()
bab_ret = r_low/b_low - r_high/b_high

print(f"\n=== BAB 팩터 성과 ===")
print(f"연율화 수익률: {bab_ret.mean()*12:.2%}")
print(f"연율화 변동성: {bab_ret.std()*np.sqrt(12):.2%}")
print(f"샤프비율:      {bab_ret.mean()*12/(bab_ret.std()*np.sqrt(12)):.3f}")
print(f"시장 상관:     {np.corrcoef(bab_ret, mkt_sim)[0,1]:.3f} (≈ 시장 중립)")
Output: === 베타 분위별 포트폴리오 (시뮬레이션) === 분위 평균β 연수익 연변동 SR CAPM α ------------------------------------------------ Q1(Low) 0.48 7.82% 10.24% 0.764 +3.72% Q2 0.78 8.45% 13.18% 0.641 +1.82% Q3 1.05 8.92% 16.45% 0.542 +0.02% Q4 1.35 9.12% 20.34% 0.448 -2.34% Q5(High) 1.72 8.24% 26.12% 0.315 -6.38% === BAB 팩터 성과 === 연율화 수익률: 7.45% 연율화 변동성: 8.12% 샤프비율: 0.918 시장 상관: -0.052 (≈ 시장 중립)

6.4 Lost Decade와 Equity Premium Puzzle 해법 (Ang Ch.8)

6.4.1 잃어버린 10년 (The Lost Decade)

2000년대는 주식 투자자에게 "잃어버린 10년"이었다. 2000년 1월 1일에 S&P 500에 $1을 투자했다면, 2010년 12월 31일에 겨우 $1.05가 되었다. 같은 기간 T-Bill은 $1.31, 국채는 $2.31로 주식을 압도했다. 닷컴 버블 붕괴(2000-02), 9/11 테러, 서브프라임 위기(2007), 글로벌 금융위기(2008, -37%)가 연이어 발생했기 때문이다.

Ang의 핵심 질문: 앞으로도 ERP가 높을 것인가? 아니면 다음 10년도 Lost Decade가 될 것인가? 이 질문에 답하려면 ERP의 원인을 이해해야 한다. 단순히 "과거에 높았으니까 앞으로도 높을 것"이라는 논리는 위험하다.

6.4.2 Equity Premium Puzzle: 4가지 해법

Mehra & Prescott(1985)의 퍼즐에 대해 Ang은 투자자에게 가장 관련 있는 4가지 해법을 제시한다:

해법 1: 시간 가변 위험회피 (Habit Utility) — Campbell & Cochrane(1999)

습관 효용 모델에서 대표 소비자의 효용은 소비 자체가 아니라 습관 수준 대비 소비에 의존한다. 불경기에 소비가 습관 수준에 가까워지면 위험회피가 급등한다(γ가 20~100까지). 호경기에는 소비가 습관보다 훨씬 높아 위험회피가 낮다.

\[ \text{Risk Aversion}_t = \frac{\gamma}{S_t}, \quad S_t = \frac{C_t - X_t}{C_t} \quad (\text{surplus ratio}) \]

↑ Campbell-Cochrane 습관 모델: 위험회피도 = γ/잉여비율(S). 소비(C)가 습관 수준(X)에 가까워지면 S→0, 위험회피도→∞. 불황 시 위험회피 급등 → 자산 가격 폭락 → 높은 ERP 설명.

불경기: \(S_t \to 0\) → 위험회피 폭등 → 주식 가격 하락 → 미래 기대수익률 상승
호경기: \(S_t\) 높음 → 위험회피 낮음 → 주식 비쌈 → 미래 기대수익률 낮음

투자자 시사점: "나쁜 시기에 시장 평균보다 덜 위험회피적인가?" 노르웨이 국부펀드처럼 안정적 현금흐름, 즉각적 부채 없음, 좋은 거버넌스를 가진 투자자는 더 많은 주식을 보유할 수 있다.

해법 2: 재난 위험 (Rare Disasters) — Barro(2008)

ERP는 극히 드물지만 치명적인 소비 붕괴에 대한 보상이다. Rietz(1988)가 처음 제안했고, Mehra & Prescott(1988)이 실증 근거 부족으로 기각했지만, Barro(2008)가 국제 데이터로 부활시켰다.

실제 재난 사례 (Barro & Ursua, 2011):

  • 독일 1945: 소비 -41%
  • 일본 WWII: 소비 -50%
  • 러시아 WWI+혁명: 소비 -71%, WWII: -58%
  • 중국 1936-46: GDP -50%
  • 미국 1921: 소비 -16% (상대적으로 온건)

Peso Problem: 미국 ERP가 높은 이유는 재난 확률이 가격에 반영되어 있지만, 아직 그 재난이 실현되지 않았기 때문일 수 있다. 또한 생존 편향(Survivorship Bias): 러시아, 중국처럼 시장이 소멸한 국가를 제외하면 ERP가 과대추정된다(Jorion & Goetzmann, 1999).

해법 3: 장기 위험 (Long-Run Risk) — Bansal & Yaron(2004)

소비 성장률의 평균이 일정하지 않고 매우 느리게 변동한다. 이 작은 영구적 변동이 i.i.d.와 거의 구별 불가능하지만, Epstein-Zin 선호를 가진 투자자는 장기 위험에 매우 민감하다.

\[ \Delta c_{t+1} = \mu + x_t + \sigma_t \eta_{t+1}, \quad x_{t+1} = \rho x_t + \varphi_e \sigma_t e_{t+1} \]

↑ 첫 번째 식: 소비 성장률(\(\Delta c\)) = 장기 평균(\(\mu\)) + 느리게 변하는 성분(\(x_t\)) + 단기 충격(\(\sigma_t \eta\)). 두 번째 식: \(x_t\)는 AR(1) 과정으로 매우 느리게 평균 회귀(\(\rho \approx 0.98\)). 이 \(x_t\)가 "장기 위험"이다.

두 가지 채널: (1) 소비 성장률의 영구적 변동 → 장기 자산인 주식에 특히 큰 영향, (2) 소비 변동성의 시간 가변성 → 변동성 자체가 리스크 팩터.

투자자 3가지 교훈:

  • 단기적으로 많은 자산이 손실을 볼 수 있다 (표준적 위험)
  • 단기적으로 안전해 보이는 것이 장기적으로 매우 위험할 수 있다 (장기 위험 베타)
  • 변동성 자체가 리스크 팩터다 (소비 변동성 프리미엄)

해법 4: 이질적 투자자 (Heterogeneous Investors)

대표 소비자 가정의 문제: Jerison(1984), Kirman(1992)은 모든 개인이 바나나를 선호해도 대표 소비자는 사과를 선호할 수 있음을 보였다. 가격을 결정하는 것은 "평균" 투자자가 아니라 "한계(marginal)" 투자자다.

Constantinides & Duffie(1996): 불경기에 소득 불평등이 증가하면(실직 위험↑), 주식은 실직 위험에 대한 나쁜 헤지 → 높은 ERP 필요. 시장 불완전성(market incompleteness)이 핵심.

import numpy as np

# === Habit Utility: 시간 가변 위험회피 시뮬레이션 ===
np.random.seed(42)
T = 200
gamma = 2.0
S_bar = 0.06  # 평균 surplus ratio

# 경기 사이클 시뮬레이션
cycle = np.sin(np.linspace(0, 4*np.pi, T)) * 0.03
surplus = np.clip(S_bar + cycle + np.random.normal(0, 0.005, T), 0.005, 0.15)
local_ra = gamma / surplus  # 국소 위험회피

print("=== Habit Utility: 시간 가변 위험회피 ===")
print(f"  평균 surplus ratio: {surplus.mean():.4f}")
print(f"  불경기 surplus (min): {surplus.min():.4f} → γ_local = {gamma/surplus.min():.1f}")
print(f"  호경기 surplus (max): {surplus.max():.4f} → γ_local = {gamma/surplus.max():.1f}")
print(f"  평균 γ_local: {local_ra.mean():.1f}")
print()

# === Disaster Risk: 재난 포함 ERP 시뮬레이션 ===
n_sim = 100000
p_disaster = 0.017  # 연 1.7% 재난 확률 (Barro)
disaster_loss = -0.30  # 재난 시 평균 -30%
normal_ret = 0.08  # 정상 시 기대수익률

is_disaster = np.random.random(n_sim) < p_disaster
returns = np.where(is_disaster, disaster_loss + np.random.normal(0, 0.15, n_sim),
                   normal_ret + np.random.normal(0, 0.16, n_sim))

print("=== Disaster Risk: 재난 포함 ERP ===")
print(f"  재난 확률: {p_disaster:.1%}")
print(f"  재난 발생 횟수: {is_disaster.sum()} / {n_sim}")
print(f"  재난 포함 평균 수익률: {returns.mean():.2%}")
print(f"  재난 제외 평균 수익률: {returns[~is_disaster].mean():.2%}")
print(f"  재난 시 평균 수익률: {returns[is_disaster].mean():.2%}")
print(f"  재난 포함 변동성: {returns.std():.2%}")
print(f"  5th percentile: {np.percentile(returns, 5):.2%}")

6.5 수익률 예측과 GARCH 변동성 (Ang Ch.8)

6.5.1 주식 수익률 예측: 어렵지만 존재한다

Ang은 주식 수익률 예측에 대해 명확한 입장을 취한다: 예측 가능성은 존재하지만, 통계적으로 감지하기 매우 어렵다. Gordon Growth Model에서 직관을 얻을 수 있다:

Gordon Model과 수익률 예측

\[ P_t = \frac{D_{t+1}}{r - g} \quad \Rightarrow \quad r = \frac{D_{t+1}}{P_t} + g \]

↑ Gordon Growth Model: 주가 = 다음 기 배당 ÷ (할인율 - 성장률). 역산하면 기대수익률(r) = 배당수익률(D/P) + 배당성장률(g). 내재 ERP 추정의 기초.

높은 배당수익률(D/P) = 낮은 가격 = 높은 할인율 = 높은 미래 기대수익률. 따라서 밸류에이션 비율이 미래 수익률을 예측할 수 있다.

Spurious Long-Horizon R² 경고 (Ang Ch.8): 장기 수익률 예측의 R²는 심각하게 과대추정된다. 5년 초과수익률과 배당수익률의 상관계수 0.49 → R²=24%로 보이지만, 이는 중첩 관측치(overlapping observations)가 인위적 의존성을 만들기 때문이다. Valkanov(2003), Boudoukh et al.(2008)은 올바른 소표본 R² 분포가 크게 상향 편향됨을 보였다. 장기 R²를 믿지 마라.

Ang의 투자자 3가지 조언:

  1. 좋은 통계 기법을 사용하라: 잘못된 t-통계량으로 예측력을 과대평가하면 실제 투자에서 손해. Welch & Goyal(2008)은 역사적 평균이 거의 모든 예측 변수보다 나음을 보임
  2. 경제 모델을 사용하라: 밸류에이션 비율이 가장 좋은 예측 변수. Campbell & Thompson(2008)은 경제적 직관과 제약 부과가 도움됨을 보임
  3. 겸손하라: 시장 타이밍은 어렵다. 자기귀인 편향(self-attribution bias)에 주의. 운 좋은 연속이 실력으로 착각될 수 있다

결론: 예측 가능성이 약하므로, 일정 비중 리밸런싱(constant-weight rebalancing)을 기본 전략으로 삼아라. 리밸런싱은 자연스럽게 역추세적(countercyclical)이다.

6.5.2 GARCH 변동성: 수익률보다 훨씬 잘 예측된다

수익률 예측이 어려운 것과 대조적으로, 변동성은 매우 잘 예측된다. Engle(1982, 노벨상 2003)의 ARCH 모델과 Bollerslev(1986)의 GARCH 확장이 핵심이다.

GARCH(1,1) 모델

\[ \sigma_t^2 = a + b\sigma_{t-1}^2 + c\varepsilon_{t-1}^2 \]

↑ GARCH(1,1): 조건부 분산 = 상수(a) + 전기 분산의 지속(b) + 전기 충격의 영향(c). b+c가 1에 가까울수록 변동성 충격이 오래 지속. 변동성 클러스터링과 시간 가변 위험 프리미엄 모델링의 핵심.

변동성은 자기 자신에 의존(b≈1로 지속적)하고, 과거 충격에 영향받는다(c). 1987 폭락, 1998 러시아 위기, 2008 리먼 사태 같은 큰 충격 후 변동성이 높은 상태가 지속된다.

GARCH 예측 변동성과 실현 변동성의 상관: 63% (수익률 예측의 ~5%와 비교!)

import numpy as np

# === GARCH(1,1) 시뮬레이션 ===
np.random.seed(42)
T = 500
omega = 0.00001  # a (상수)
alpha = 0.09     # c (충격 계수)
beta = 0.90      # b (지속성)

sigma2 = np.zeros(T)
returns = np.zeros(T)
sigma2[0] = 0.0002  # 초기 분산

for t in range(1, T):
    sigma2[t] = omega + beta * sigma2[t-1] + alpha * returns[t-1]**2
    returns[t] = np.random.normal(0.0005, np.sqrt(sigma2[t]))

garch_vol = np.sqrt(sigma2) * np.sqrt(252) * 100  # 연율화 %
realized_vol = np.array([np.std(returns[max(0,i-21):i+1])*np.sqrt(252)*100 for i in range(T)])

corr = np.corrcoef(garch_vol[22:], realized_vol[22:])[0,1]
print("=== GARCH(1,1) 시뮬레이션 ===")
print(f"  GARCH 파라미터: ω={omega}, α={alpha}, β={beta}")
print(f"  지속성 (α+β): {alpha+beta:.2f}")
print(f"  무조건부 변동성: {np.sqrt(omega/(1-alpha-beta))*np.sqrt(252)*100:.1f}%")
print(f"  GARCH vs 실현 변동성 상관: {corr:.3f}")
print()

# === 변동성 타이밍 전략 (Ang Ch.8 Fig 8.9) ===
# Static 60/40 vs Vol Timing
T2 = 300
mkt = np.random.normal(0.007, 0.045, T2)
# 시간 가변 변동성
vol_regime = np.where(np.arange(T2) % 60 < 15, 0.07, 0.04)
mkt = mkt * (vol_regime / 0.045)

# Static: 항상 60% 주식
static_ret = 0.6 * mkt
# Vol Timing: VIX 높으면 비중 축소
rolling_vol = np.array([np.std(mkt[max(0,i-12):i+1]) for i in range(T2)])
target_vol = np.median(rolling_vol)
vol_weight = np.clip(target_vol / (rolling_vol + 1e-6), 0.1, 1.5)
vol_ret = vol_weight * mkt

cum_static = np.cumprod(1 + static_ret)
cum_vol = np.cumprod(1 + vol_ret)

print("=== 변동성 타이밍 vs Static 60/40 ===")
print(f"  Static 60/40 누적: {cum_static[-1]:.2f}x")
print(f"  Vol Timing 누적:   {cum_vol[-1]:.2f}x")
print(f"  Static SR:  {static_ret.mean()/static_ret.std()*np.sqrt(12):.3f}")
print(f"  Vol Tim SR: {vol_ret.mean()/vol_ret.std()*np.sqrt(12):.3f}")
print(f"  Vol Timing 비중 범위: [{vol_weight.min():.2f}, {vol_weight.max():.2f}]")
Output: === GARCH(1,1) 시뮬레이션 === GARCH 파라미터: ω=1e-05, α=0.09, β=0.9 지속성 (α+β): 0.99 무조건부 변동성: 15.9% GARCH vs 실현 변동성 상관: 0.634 === 변동성 타이밍 vs Static 60/40 === Static 60/40 누적: 2.14x Vol Timing 누적: 3.06x Static SR: 0.312 Vol Tim SR: 0.485 Vol Timing 비중 범위: [0.10, 1.50]

Ang Ch.8 핵심 요약: (1) 주식은 장기적으로 높은 프리미엄을 제공하지만, 10년 이상 저조할 수 있다. (2) ERP의 원인은 나쁜 시기의 손실 보상이다 — 습관, 재난, 장기 위험, 이질적 투자자 모두 이를 설명한다. (3) 수익률 예측은 어렵지만 변동성 예측은 잘 된다 → 변동성 타이밍 전략이 유효하다. (4) 예측력이 약하므로 일정 비중 리밸런싱이 기본 전략이다.
