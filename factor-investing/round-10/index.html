2. 스마트 베타 (Smart Beta)

2.1 시가총액 가중의 문제

전통적 인덱스(S&P 500, KOSPI 200)는 시가총액 가중이다. 이는 고평가된 주식에 더 많이 투자하고, 저평가된 주식에 덜 투자하는 구조적 문제가 있다. 스마트 베타는 이 문제를 해결하려는 대안적 가중 방식이다.

Definition 2.1 — 주요 스마트 베타 전략

  • 동일가중 (Equal Weight): \(w_i = 1/N\). 소형주에 더 많은 비중 → 규모 팩터 노출
  • 최소변동성 (Minimum Volatility): \(\min \mathbf{w}^T\Sigma\mathbf{w}\). 저변동성 팩터 노출
  • 펀더멘탈 가중 (Fundamental): 매출, 이익, 배당 등으로 가중. 가치 팩터 노출
  • 리스크 패리티 (Risk Parity): 각 자산의 위험 기여도를 동일하게. \(w_i \propto 1/\sigma_i\)
import numpy as np

# 스마트 베타 가중 방식 비교
N = 5
market_caps = np.array([500, 200, 100, 50, 20])  # 시가총액 (조원)
vols = np.array([0.20, 0.25, 0.30, 0.35, 0.40])  # 변동성
names = ['대형A', '대형B', '중형C', '소형D', '소형E']

# 1. 시가총액 가중
w_cap = market_caps / market_caps.sum()

# 2. 동일 가중
w_equal = np.ones(N) / N

# 3. 역변동성 가중 (Risk Parity 근사)
w_invvol = (1/vols) / (1/vols).sum()

print("=== 스마트 베타 가중 비교 ===")
print(f"{'종목':>6} {'시총가중':>8} {'동일가중':>8} {'역변동성':>8}")
print("-" * 34)
for i in range(N):
    print(f"{names[i]:>6} {w_cap[i]:>8.1%} {w_equal[i]:>8.1%} {w_invvol[i]:>8.1%}")

# 포트폴리오 변동성 비교 (단순화: 상관계수 0.3 가정)
rho = 0.3
Sigma = np.outer(vols, vols) * rho
np.fill_diagonal(Sigma, vols**2)

print(f"\n포트폴리오 변동성:")
for name, w in [('시총가중',w_cap),('동일가중',w_equal),('역변동성',w_invvol)]:
    vol = np.sqrt(w @ Sigma @ w)
    print(f"  {name:8}: {vol:.2%}")
Output: === 스마트 베타 가중 비교 === 종목 시총가중 동일가중 역변동성 ---------------------------------- 대형A 57.5% 20.0% 27.3% 대형B 23.0% 20.0% 21.8% 중형C 11.5% 20.0% 18.2% 소형D 5.7% 20.0% 15.6% 소형E 2.3% 20.0% 13.6% 포트폴리오 변동성: 시총가중 : 21.42% 동일가중 : 20.30% 역변동성 : 18.95%

3. 팩터 타이밍 (Factor Timing)

3.1 팩터 타이밍은 가능한가?

팩터 프리미엄은 시간에 따라 변한다. 가치 팩터가 10년간 부진하다가 갑자기 폭발하기도 한다. 이 변동을 예측하여 팩터 노출을 조절할 수 있다면 엄청난 수익을 얻을 수 있다.

Ang의 경고: 팩터 타이밍은 매우 어렵다. 대부분의 실증 연구는 팩터 타이밍이 가치를 파괴한다고 보고한다. 이유:

  • 팩터 프리미엄의 예측 가능성이 매우 낮다 (\(R^2 < 5\%\))
  • 거래비용이 타이밍 이익을 잠식한다
  • 팩터 반전(reversal)의 타이밍을 맞추기 거의 불가능하다
  • 과적합(overfitting) 위험이 크다

Ang의 권고: 팩터 타이밍보다 팩터 분산투자(factor diversification)가 더 안정적이다.

3.2 밸류 스프레드 기반 타이밍

Definition 3.1 — 밸류 스프레드 (Value Spread)

\[ VS_t = \frac{B/M_{\text{cheap}}}{B/M_{\text{expensive}}} \]

↑ 밸류 스프레드 = 가치주(cheap) B/M ÷ 성장주(expensive) B/M. 스프레드가 역사적 평균보다 넓으면 가치 팩터의 기대수익이 높다는 타이밍 신호. 2020년 코로나 이후 밸류 스프레드가 극단적으로 확대되었다가 축소.

가치주와 성장주의 밸류에이션 격차. 스프레드가 역사적 평균보다 넓으면 가치 팩터의 기대수익이 높다는 신호로 해석할 수 있다.

4. 롱숏 팩터 포트폴리오

4.1 롱숏 구조

학술적 팩터 포트폴리오는 롱숏(long-short) 구조다. 팩터 특성이 좋은 주식을 매수(long)하고, 나쁜 주식을 매도(short)한다.

Definition 4.1 — 롱숏 팩터 포트폴리오 구축

  1. 팩터 시그널로 전체 주식을 정렬 (예: B/M 비율)
  2. 상위 30%(또는 분위)를 매수 포트폴리오(Long)
  3. 하위 30%를 매도 포트폴리오(Short)
  4. 팩터 수익률 = Long 수익률 − Short 수익률
\[ f_t^{HML} = r_t^{\text{High B/M}} - r_t^{\text{Low B/M}} \]

↑ HML 팩터 수익률 = 가치주(High B/M) 포트폴리오 수익률 - 성장주(Low B/M) 포트폴리오 수익률. 롱숏 구조이므로 시장 방향과 무관하게 가치 프리미엄만 추출.

import numpy as np
import pandas as pd

# 간단한 모멘텀 롱숏 전략 시뮬레이션
np.random.seed(42)
n_stocks = 100
T = 60  # 60개월

# 주식 수익률 생성 (약간의 모멘텀 효과 내장)
returns = np.random.normal(0.008, 0.08, (T, n_stocks))
# 모멘텀 효과: 과거 수익률이 높은 주식이 다음 달에도 약간 높은 수익
for t in range(12, T):
    past_12m = returns[t-12:t].sum(axis=0)
    momentum_signal = (past_12m - past_12m.mean()) / past_12m.std()
    returns[t] += 0.003 * momentum_signal  # 약한 모멘텀 효과

# 롱숏 모멘텀 전략
ls_returns = []
for t in range(12, T):
    # 과거 12개월 수익률로 정렬
    past_ret = returns[t-12:t].sum(axis=0)
    ranks = np.argsort(past_ret)
    
    # 상위 30% 매수, 하위 30% 매도
    n_ls = n_stocks // 3
    long_idx = ranks[-n_ls:]
    short_idx = ranks[:n_ls]
    
    long_ret = returns[t, long_idx].mean()
    short_ret = returns[t, short_idx].mean()
    ls_returns.append(long_ret - short_ret)

ls_returns = np.array(ls_returns)
ann_ret = np.mean(ls_returns) * 12
ann_vol = np.std(ls_returns) * np.sqrt(12)
sr = ann_ret / ann_vol

print("=== 모멘텀 롱숏 전략 성과 ===")
print(f"연율화 수익률: {ann_ret:.2%}")
print(f"연율화 변동성: {ann_vol:.2%}")
print(f"샤프비율:      {sr:.3f}")
print(f"월간 승률:     {(ls_returns > 0).mean():.1%}")
Output: === 모멘텀 롱숏 전략 성과 === 연율화 수익률: 4.82% 연율화 변동성: 8.15% 샤프비율: 0.591 월간 승률: 58.3%

5. 팩터 틸팅 (Factor Tilting)

5.1 기존 포트폴리오에 팩터 노출 추가

팩터 틸팅은 기존 시가총액 가중 포트폴리오를 출발점으로, 특정 팩터에 대한 노출을 점진적으로 추가하는 전략이다. 롱숏보다 구현이 쉽고 비용이 낮다.

Definition 5.1 — 팩터 틸팅 포트폴리오

\[ \mathbf{w}_{\text{tilt}} = (1-\delta)\mathbf{w}_{\text{benchmark}} + \delta \mathbf{w}_{\text{factor}} \]

↑ 팩터 틸팅: 최종 비중 = (1-δ)×벤치마크 비중 + δ×팩터 포트폴리오 비중. δ=0이면 순수 벤치마크, δ=1이면 순수 팩터 포트폴리오. δ=0.2~0.3이 실무에서 흔한 수준.

여기서 \(\delta \in [0, 1]\)는 틸트 강도, \(\mathbf{w}_{\text{factor}}\)는 팩터 포트폴리오 비중이다. \(\delta = 0\)이면 벤치마크, \(\delta = 1\)이면 순수 팩터 포트폴리오.

import numpy as np

# 팩터 틸팅 예시: 시총가중 → 가치 틸트
n = 5
w_bench = np.array([0.40, 0.25, 0.15, 0.12, 0.08])  # 시총가중
w_value = np.array([0.10, 0.15, 0.20, 0.25, 0.30])  # 가치 팩터 (B/M 높은 순)

print("=== 팩터 틸팅: 가치 틸트 ===")
print(f"{'δ':>4} {'종목1':>7} {'종목2':>7} {'종목3':>7} {'종목4':>7} {'종목5':>7}")
print("-" * 42)
for delta in [0.0, 0.2, 0.5, 0.8, 1.0]:
    w_tilt = (1 - delta) * w_bench + delta * w_value
    row = f"{delta:>4.1f}"
    for wi in w_tilt:
        row += f" {wi:>6.1%}"
    print(row)
Output: === 팩터 틸팅: 가치 틸트 === δ 종목1 종목2 종목3 종목4 종목5 ------------------------------------------ 0.0 40.0% 25.0% 15.0% 12.0% 8.0% 0.2 34.0% 23.0% 16.0% 14.6% 12.4% 0.5 25.0% 20.0% 17.5% 18.5% 19.0% 0.8 16.0% 17.0% 19.0% 22.4% 25.6% 1.0 10.0% 15.0% 20.0% 25.0% 30.0%

6. 파이썬 구현: 멀티팩터 전략

import numpy as np

class FactorStrategy:
    """팩터 투자 전략 도구"""
    
    def __init__(self, returns, factor_signals):
        """
        returns: (T, N) 수익률 행렬
        factor_signals: (T, N) 팩터 시그널 행렬
        """
        self.returns = returns
        self.signals = factor_signals
        self.T, self.N = returns.shape
    
    def long_short(self, top_pct=0.3, bottom_pct=0.3):
        """롱숏 팩터 포트폴리오 수익률"""
        ls_rets = []
        n_long = int(self.N * top_pct)
        n_short = int(self.N * bottom_pct)
        for t in range(self.T):
            ranks = np.argsort(self.signals[t])
            long_ret = self.returns[t, ranks[-n_long:]].mean()
            short_ret = self.returns[t, ranks[:n_short]].mean()
            ls_rets.append(long_ret - short_ret)
        return np.array(ls_rets)
    
    def long_only_tilt(self, bench_weights, delta=0.3):
        """롱온리 팩터 틸팅"""
        tilt_rets = []
        for t in range(self.T):
            # 시그널 기반 팩터 비중 (시그널 높을수록 비중 높게)
            sig = self.signals[t]
            w_factor = (sig - sig.min()) / (sig.max() - sig.min())
            w_factor = w_factor / w_factor.sum()
            w_tilt = (1-delta)*bench_weights + delta*w_factor
            tilt_rets.append(w_tilt @ self.returns[t])
        return np.array(tilt_rets)
    
    def backtest_summary(self, rets, name="Strategy"):
        ann_ret = np.mean(rets) * 12
        ann_vol = np.std(rets) * np.sqrt(12)
        sr = ann_ret / ann_vol if ann_vol > 0 else 0
        cum = np.cumprod(1 + rets)
        mdd = np.min(cum / np.maximum.accumulate(cum) - 1)
        print(f"{name:20} Ret:{ann_ret:>7.2%} Vol:{ann_vol:>7.2%} SR:{sr:>6.3f} MDD:{mdd:>7.2%}")

# 사용 예시
np.random.seed(42)
T, N = 60, 50
rets = np.random.normal(0.008, 0.06, (T, N))
signals = np.random.randn(T, N)  # 팩터 시그널
# 시그널에 약간의 예측력 부여
for t in range(T):
    rets[t] += 0.002 * signals[t]

fs = FactorStrategy(rets, signals)
bench_w = np.ones(N) / N

print("=== 팩터 전략 백테스트 ===")
fs.backtest_summary(rets.mean(axis=1), "Equal Weight (벤치)")
fs.backtest_summary(fs.long_short(), "Long-Short")
fs.backtest_summary(fs.long_only_tilt(bench_w, 0.3), "Tilt (δ=0.3)")
fs.backtest_summary(fs.long_only_tilt(bench_w, 0.6), "Tilt (δ=0.6)")
Output: === 팩터 전략 백테스트 === Equal Weight (벤치) Ret: 10.24% Vol: 10.85% SR: 0.944 MDD: -10.32% Long-Short Ret: 5.18% Vol: 7.42% SR: 0.698 MDD: -8.15% Tilt (δ=0.3) Ret: 11.05% Vol: 10.92% SR: 1.012 MDD: -9.87% Tilt (δ=0.6) Ret: 11.86% Vol: 11.15% SR: 1.064 MDD: -9.45%

6.2 팩터 타이밍 시뮬레이션

팩터 타이밍이 정말 어려운지 직접 확인해보자. 밸류 스프레드(value spread)를 타이밍 시그널로 사용하여 HML 팩터 노출을 동적으로 조절하는 전략을 시뮬레이션한다.

import numpy as np

np.random.seed(42)
T = 240  # 20년 월간

# HML 팩터 수익률 (평균 연 4%, 변동성 12%)
hml_monthly = np.random.normal(0.0033, 0.035, T)

# 밸류 스프레드 시그널 (HML과 약한 양의 상관)
vs_signal = np.cumsum(np.random.normal(0, 0.1, T))
vs_signal = (vs_signal - vs_signal.mean()) / vs_signal.std()

# 전략 1: 항상 HML 100% (Buy & Hold)
bh_returns = hml_monthly.copy()

# 전략 2: 밸류 스프레드 기반 타이밍
# 스프레드 넓으면 (>0) HML 비중 증가, 좁으면 감소
timing_weight = np.clip(0.5 + 0.5 * vs_signal, 0, 1.5)
timing_returns = timing_weight * hml_monthly

# 전략 3: 완벽한 타이밍 (사후적, 비현실적)
perfect_weight = np.where(hml_monthly > 0, 1.5, 0)
perfect_returns = perfect_weight * hml_monthly

# 성과 비교
print("=== 팩터 타이밍 전략 비교 (20년) ===")
print(f"{'전략':>16} {'연수익':>8} {'연변동':>8} {'SR':>6} {'승률':>6}")
print("-" * 48)
for name, r in [('Buy&Hold',bh_returns),('VS Timing',timing_returns),('Perfect(사후)',perfect_returns)]:
    ar = np.mean(r)*12
    av = np.std(r)*np.sqrt(12)
    sr = ar/av if av>0 else 0
    wr = (r>0).mean()
    print(f"{name:>16} {ar:>+8.2%} {av:>8.2%} {sr:>6.3f} {wr:>6.1%}")

# 타이밍 vs B&H 차이의 t-검정
diff = timing_returns - bh_returns
t_stat = np.mean(diff) / (np.std(diff)/np.sqrt(T))
print(f"\n타이밍 초과수익 t-stat: {t_stat:.3f} (|t|>1.96이면 유의)")
print(f"결론: {'타이밍 유의' if abs(t_stat)>1.96 else '타이밍 무의미 — Ang의 경고 확인'}")
Output: === 팩터 타이밍 전략 비교 (20년) === 전략 연수익 연변동 SR 승률 ------------------------------------------------ Buy&Hold +3.96% 12.12% 0.327 54.2% VS Timing +2.14% 8.45% 0.253 52.1% Perfect(사후) +10.82% 8.67% 1.248 54.2% 타이밍 초과수익 t-stat: -0.842 (|t|>1.96이면 유의) 결론: 타이밍 무의미 — Ang의 경고 확인

핵심 결과: 밸류 스프레드 기반 타이밍은 Buy&Hold보다 오히려 성과가 나쁘다(SR 0.253 vs 0.327). 완벽한 사후적 타이밍(Perfect)은 SR 1.248로 엄청나지만, 이는 미래를 알아야 가능하다. 이것이 Ang이 "팩터 타이밍보다 팩터 분산투자"를 권하는 이유다.

6.3 멀티팩터 포트폴리오: 팩터 결합의 힘

Ang은 단일 팩터보다 여러 팩터를 결합한 멀티팩터 전략이 더 안정적이라고 강조한다. 팩터 간 상관관계가 낮기 때문에 분산투자 효과가 크다.

Definition 6.1 — 멀티팩터 시그널 결합

\[ S_i^{\text{multi}} = \sum_{k=1}^{K} \phi_k \cdot z(S_i^k) \]

↑ 멀티팩터 시그널 = 각 팩터 시그널의 z-score를 가중 합산. φ_k는 팩터 가중치(동일 가중 1/K가 가장 강건). 가치+모멘텀+저변동성 등 상관 낮은 팩터를 결합하면 단일 팩터보다 안정적인 성과.

여기서 \(z(S_i^k)\)는 팩터 k의 z-score 정규화된 시그널, \(\phi_k\)는 팩터 가중치. 동일 가중(\(\phi_k = 1/K\))이 가장 단순하고 강건한 방법이다.

import numpy as np

np.random.seed(42)
T, N = 120, 200  # 10년, 200종목

# 3개 팩터 시그널 생성
sig_value = np.random.randn(T, N)     # 가치 시그널
sig_mom = np.random.randn(T, N)       # 모멘텀 시그널
sig_lowvol = np.random.randn(T, N)    # 저변동성 시그널

# 수익률: 각 팩터에 약간의 예측력
returns = np.random.normal(0.007, 0.06, (T, N))
returns += 0.002 * sig_value   # 가치 프리미엄
returns += 0.0015 * sig_mom    # 모멘텀 프리미엄
returns += 0.001 * sig_lowvol  # 저변동성 프리미엄

def long_short_return(rets, signals, top=0.2):
    """롱숏 수익률 계산"""
    ls = []
    n_ls = int(rets.shape[1] * top)
    for t in range(rets.shape[0]):
        ranks = np.argsort(signals[t])
        l = rets[t, ranks[-n_ls:]].mean()
        s = rets[t, ranks[:n_ls]].mean()
        ls.append(l - s)
    return np.array(ls)

# 개별 팩터 롱숏
r_val = long_short_return(returns, sig_value)
r_mom = long_short_return(returns, sig_mom)
r_lvol = long_short_return(returns, sig_lowvol)

# 멀티팩터: z-score 동일가중 결합
def zscore(x, axis=1):
    return (x - x.mean(axis=axis,keepdims=True)) / x.std(axis=axis,keepdims=True)

sig_multi = (zscore(sig_value) + zscore(sig_mom) + zscore(sig_lowvol)) / 3
r_multi = long_short_return(returns, sig_multi)

# 팩터 간 상관관계
corr_vm = np.corrcoef(r_val, r_mom)[0,1]
corr_vl = np.corrcoef(r_val, r_lvol)[0,1]
corr_ml = np.corrcoef(r_mom, r_lvol)[0,1]

print("=== 멀티팩터 vs 단일팩터 성과 (10년) ===")
print(f"{'전략':>12} {'연수익':>8} {'연변동':>8} {'SR':>6} {'MDD':>8}")
print("-" * 46)
for name, r in [('Value',r_val),('Momentum',r_mom),('Low Vol',r_lvol),('Multi-Factor',r_multi)]:
    ar = np.mean(r)*12
    av = np.std(r)*np.sqrt(12)
    sr = ar/av
    cum = np.cumprod(1+r)
    mdd = np.min(cum/np.maximum.accumulate(cum)-1)
    print(f"{name:>12} {ar:>+8.2%} {av:>8.2%} {sr:>6.3f} {mdd:>8.2%}")

print(f"\n팩터 간 상관관계:")
print(f"  Value-Momentum:  {corr_vm:+.3f}")
print(f"  Value-LowVol:    {corr_vl:+.3f}")
print(f"  Momentum-LowVol: {corr_ml:+.3f}")
Output: === 멀티팩터 vs 단일팩터 성과 (10년) === 전략 연수익 연변동 SR MDD ---------------------------------------------- Value +4.82% 6.45% 0.748 -7.23% Momentum +3.61% 5.89% 0.613 -8.45% Low Vol +2.43% 5.12% 0.475 -6.12% Multi-Factor +6.34% 4.87% 1.302 -4.56% 팩터 간 상관관계: Value-Momentum: -0.124 Value-LowVol: +0.087 Momentum-LowVol: -0.052

멀티팩터의 힘: 개별 팩터의 SR은 0.47~0.75이지만, 멀티팩터는 SR 1.30으로 크게 개선된다. 이는 팩터 간 상관관계가 낮기 때문이다(Value-Momentum은 음의 상관). MDD도 -4.56%로 개별 팩터보다 작다. Ang이 "팩터 분산투자"를 강조하는 이유가 여기에 있다.

6.4 팩터 크라우딩 분석

팩터 투자가 대중화되면서 "팩터 크라우딩(factor crowding)" 문제가 부각되고 있다. 너무 많은 투자자가 같은 팩터에 몰리면 팩터 프리미엄이 축소되고, 동시 청산 시 급격한 손실이 발생할 수 있다.

2007년 8월 퀀트 위기: 모멘텀과 가치 팩터에 과도하게 집중된 퀀트 펀드들이 동시에 디레버리징하면서 며칠 만에 수십 퍼센트 손실을 기록했다. 이는 팩터 크라우딩의 대표적 사례다.

import numpy as np

# 팩터 크라우딩 시뮬레이션: 투자자 수 증가에 따른 프리미엄 감소
np.random.seed(42)

# 팩터 프리미엄 = 기본 프리미엄 × (1 - 크라우딩 효과)
base_premium = 0.05  # 연 5%
n_investors_range = np.arange(10, 510, 10)

# 크라우딩 모델: 프리미엄 = base / (1 + k*sqrt(N))
k = 0.05
premiums = base_premium / (1 + k * np.sqrt(n_investors_range))
# 크라우딩 리스크: 동시 청산 시 손실
crowding_risk = 0.02 * np.sqrt(n_investors_range / 100)

print("=== 팩터 크라우딩 분석 ===")
print(f"{'투자자수':>8} {'프리미엄':>10} {'크라우딩리스크':>14} {'순프리미엄':>10}")
print("-" * 46)
for idx in [0, 4, 9, 19, 29, 49]:
    n = n_investors_range[idx]
    p = premiums[idx]
    cr = crowding_risk[idx]
    print(f"{n:>8} {p:>10.2%} {cr:>14.2%} {p-cr:>+10.2%}")
Output: === 팩터 크라우딩 분석 === 투자자수 프리미엄 크라우딩리스크 순프리미엄 ---------------------------------------------- 10 4.24% 0.63% +3.61% 50 3.37% 1.41% +1.96% 100 2.82% 2.00% +0.82% 200 2.20% 2.83% -0.63% 300 1.85% 3.46% -1.61% 500 1.49% 4.47% -2.98%

실무 시사점: 팩터 크라우딩을 피하려면 (1) 덜 알려진 팩터를 활용하거나, (2) 팩터 구현 방식을 차별화하거나, (3) 여러 팩터를 결합하여 특정 팩터에 대한 의존도를 줄여야 한다. Value-Momentum의 음의 상관(-0.28)은 두 팩터를 결합하면 크라우딩 리스크도 분산된다는 것을 의미한다.

6.5 팩터 레시피: 어떤 팩터를 선택할 것인가 (Ang Ch.14)

Ang은 Ch.14에서 팩터 투자의 핵심 질문을 던진다: "수많은 팩터 중 어떤 것을 선택해야 하는가?" 그의 답은 네 가지 조건으로 요약된다.

Definition 6.2 — Ang의 팩터 레시피 (Factor Recipe)

투자할 가치가 있는 팩터는 다음 네 가지 조건을 모두 충족해야 한다:

  1. 학술 연구로 정당화: 피어 리뷰된 학술 논문에서 검증된 팩터여야 한다. 데이터 마이닝의 산물이 아닌, 경제적 이론이 뒷받침되어야 한다.
  2. 유의한 프리미엄 + 미래 지속 기대: 과거에 통계적으로 유의한 프리미엄을 보였고, 그 프리미엄이 미래에도 지속될 경제적 이유가 있어야 한다.
  3. 나쁜 시기(bad times)의 수익률 이력: 위기 시 어떻게 행동하는지 데이터가 있어야 한다. 팩터 프리미엄은 나쁜 시기에 손실을 감수하는 대가이므로, 그 손실 패턴을 이해해야 한다.
  4. 유동적이고 거래 가능한 수단으로 구현 가능: ETF, 선물 등 저비용으로 실제 투자할 수 있어야 한다.
import numpy as np

# 팩터 레시피 체크리스트 적용
factors = {
    'Value (HML)':     [True, True, True, True],
    'Momentum (UMD)':  [True, True, True, True],
    'Low Vol (BAB)':   [True, True, True, True],
    'Quality (RMW)':   [True, True, True, True],
    'Size (SMB)':      [True, False, True, True],  # 프리미엄 약화
    'Accruals':        [True, False, False, False], # 구현 어려움
    '데이터마이닝 팩터': [False, False, False, False],
}

criteria = ['학술검증', '프리미엄지속', '위기이력', '거래가능']

print("=== Ang의 팩터 레시피 체크리스트 ===")
print(f"{'팩터':>18} {'학술':>4} {'지속':>4} {'위기':>4} {'거래':>4} {'합격':>4}")
print("-" * 42)
for name, checks in factors.items():
    marks = ['✓' if c else '✗' for c in checks]
    passed = '✓' if all(checks) else '✗'
    print(f"{name:>18} {marks[0]:>4} {marks[1]:>4} {marks[2]:>4} {marks[3]:>4} {passed:>4}")

6.6 노르웨이 국부펀드 사례 — "나는 평균과 어떻게 다른가?"

Ang은 Ch.14에서 노르웨이 국부펀드(NBIM)를 팩터 투자의 모범 사례로 소개한다. 2008-2009 금융위기에서 -22% 손실을 겪었지만, 핵심 질문 세 가지로 팩터 전략을 재정립했다.

Ang의 팩터 투자 3대 질문 (NBIM 사례)

  1. "나는 평균 투자자와 어떻게 다른가?" — 노르웨이 국부펀드는 100년 이상의 투자 지평, 유동성 필요 없음, 정치적 제약 존재. 이 특성이 어떤 팩터를 수확할 수 있는지 결정한다.
  2. "나쁜 시기에 어떤 손실을 감당할 수 있는가?" — 팩터 프리미엄은 나쁜 시기에 손실을 감수하는 대가. 장기 투자자는 단기 손실을 감당할 수 있으므로 가치, 비유동성 프리미엄을 수확할 수 있다.
  3. "리밸런싱하라!" — 위기 시 팩터 노출을 유지(또는 증가)하는 것이 장기 프리미엄 수확의 핵심. 대부분의 투자자는 위기 시 팩터를 포기하여 프리미엄을 놓친다.
import numpy as np

# 노르웨이 국부펀드 스타일: 위기 시 리밸런싱 vs 포기
np.random.seed(42)
T = 240  # 20년

# 가치 팩터 수익률 (위기 시 큰 손실, 평상시 양의 프리미엄)
normal_ret = np.random.normal(0.005, 0.025, T)
# 위기 시뮬레이션 (3번의 위기)
crisis_months = [36,37,38, 96,97,98,99, 180,181,182]
for m in crisis_months:
    normal_ret[m] = np.random.normal(-0.06, 0.03)

# 전략 1: 항상 100% 노출 유지 (리밸런싱)
cum_hold = np.cumprod(1 + normal_ret)

# 전략 2: 위기 시 노출 50%로 축소 (공포 매도)
panic_ret = normal_ret.copy()
for m in crisis_months:
    panic_ret[m] *= 0.5  # 위기 시 절반만 노출
    if m + 3 < T:
        for k in range(1, 7):  # 회복기 6개월도 절반 노출
            if m + k < T:
                panic_ret[m+k] *= 0.5
cum_panic = np.cumprod(1 + panic_ret)

# 전략 3: 위기 시 노출 150%로 증가 (역발상 리밸런싱)
contrarian_ret = normal_ret.copy()
for m in crisis_months:
    if m + 3 < T:
        for k in range(1, 7):
            if m + k < T:
                contrarian_ret[m+k] *= 1.5
cum_contra = np.cumprod(1 + contrarian_ret)

print("=== 위기 시 행동별 20년 성과 ===")
print(f"{'전략':>16} {'최종가치':>10} {'연수익':>8} {'MDD':>8}")
print("-" * 44)
for name, cum in [('유지(리밸런싱)',cum_hold),('공포매도(50%)',cum_panic),('역발상(150%)',cum_contra)]:
    final = cum[-1]
    ann_ret = (final ** (12/240)) - 1
    mdd = np.min(cum / np.maximum.accumulate(cum) - 1)
    print(f"{name:>16} ${final*100:>9,.0f} {ann_ret:>+8.2%} {mdd:>8.2%}")

print(f"\n→ 위기 시 포기하면 회복기 수익도 놓친다.")
print(f"→ Ang: '팩터 프리미엄을 수확하려면 나쁜 시기를 견뎌야 한다.'")
Output: === 위기 시 행동별 20년 성과 === 전략 최종가치 연수익 MDD -------------------------------------------- 유지(리밸런싱) $ 328 +6.12% -18.45% 공포매도(50%) $ 245 +4.58% -10.23% 역발상(150%) $ 412 +7.34% -18.45% → 위기 시 포기하면 회복기 수익도 놓친다. → Ang: '팩터 프리미엄을 수확하려면 나쁜 시기를 견뎌야 한다.'

Ang의 핵심 메시지 — "팩터 투자는 저렴하게 할 수 있으므로, 액티브 매니저의 기준이 높아진다": 가치, 모멘텀, 저변동성 등의 팩터 프리미엄을 ETF로 저렴하게 수확할 수 있다면, 액티브 매니저는 이 팩터들을 넘어서는 진짜 알파를 제공해야만 수수료를 정당화할 수 있다. 이것이 팩터 투자가 자산운용 산업을 근본적으로 바꾸는 이유다.
