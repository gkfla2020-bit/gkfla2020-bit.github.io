헷갈리기 쉬운 것들:

  • \(\|x\|\) (이중 세로선) = 노름(크기) vs \(|x|\) (단일 세로선) = 절대값. 벡터에는 노름, 스칼라에는 절대값.
  • \(x^T\) = 전치(transpose) vs \(x^2\) = 제곱. T는 이탤릭이 아니라 로만체(직립)로 쓴다.
  • \(\Sigma\) (대문자 시그마) = 공분산 행렬 vs \(\sigma\) (소문자 시그마) = 표준편차. 대소문자가 완전히 다른 의미!
  • \(\boldsymbol{\mu}\) (볼드 뮤) = 수익률 벡터 vs \(\mu\) (일반 뮤) = 스칼라 평균

2. 시그마(Σ) 표기법 — "더하기의 축약"

이 가이드에서 가장 많이 나오는 기호가 \(\sum\)이다. 겁먹을 필요 없다 — 그냥 "더하기를 반복하라"는 뜻이다.

Definition 2.1 — 시그마 표기법

\[ \sum_{i=1}^{N} x_i = x_1 + x_2 + x_3 + \cdots + x_N \]

↑ "i=1부터 N까지 x_i를 전부 더하라." 아래(i=1)는 시작점, 위(N)는 끝점. i는 "인덱스 변수"로, 1부터 N까지 하나씩 바뀐다.

실전 예시 — 포트폴리오 수익률:

\[ r_p = \sum_{i=1}^{N} w_i \cdot r_i = w_1 r_1 + w_2 r_2 + \cdots + w_N r_N \]

↑ "포트폴리오 수익률 = 각 자산의 (비중 × 수익률)을 전부 더한 것." w_i는 i번째 자산의 비중, r_i는 수익률. 이게 R2에서 나오는 핵심 공식이다.

이중 시그마:

\[ \sigma_p^2 = \sum_{i=1}^{N}\sum_{j=1}^{N} w_i w_j \sigma_{ij} \]

↑ "포트폴리오 분산 = 모든 자산 쌍(i,j)에 대해 (비중i × 비중j × 공분산ij)를 더한 것." 이중 for문과 같다. 이게 R2 평균-분산 최적화의 핵심.

# 시그마 표기법 → 파이썬
import numpy as np

x = np.array([3, 7, 2, 5])

# Σ x_i (i=1 to 4)
total = np.sum(x)           # = 3+7+2+5 = 17
print(f"Σ x_i = {total}")

# Σ x_i² (각 원소를 제곱한 뒤 합)
sum_sq = np.sum(x**2)      # = 9+49+4+25 = 87
print(f"Σ x_i² = {sum_sq}")

# 포트폴리오 수익률: Σ w_i * r_i
w = np.array([0.4, 0.3, 0.2, 0.1])
r = np.array([0.10, 0.06, 0.04, 0.02])
r_p = np.sum(w * r)          # = w @ r (내적)
print(f"포트폴리오 수익률 = {r_p:.2%}")
Output: Σ x_i = 17 Σ x_i² = 87 포트폴리오 수익률 = 7.00%

3. 벡터와 행렬 — 숫자를 묶어서 다루기

3.1 벡터 (Vector)

벡터는 숫자를 세로로 나열한 것이다. 이 가이드에서 볼드체(\(\mathbf{w}\))로 쓰면 벡터다.

Definition 3.1 — 벡터

\[ \mathbf{w} = \begin{pmatrix} w_1 \\ w_2 \\ w_3 \end{pmatrix} \quad \text{예: 포트폴리오 비중} \quad \mathbf{w} = \begin{pmatrix} 0.5 \\ 0.3 \\ 0.2 \end{pmatrix} \]

↑ 3개 자산의 비중을 하나의 벡터로 표현. w₁=50%, w₂=30%, w₃=20%. 파이썬에서는 np.array([0.5, 0.3, 0.2]).

3.2 전치 (Transpose, ᵀ)

세로 벡터를 가로로 눕히는 것이 전치(transpose)다. \(\mathbf{w}^T\)라고 쓴다.

\[ \mathbf{w} = \begin{pmatrix} 0.5 \\ 0.3 \\ 0.2 \end{pmatrix} \quad \Rightarrow \quad \mathbf{w}^T = \begin{pmatrix} 0.5 & 0.3 & 0.2 \end{pmatrix} \]

↑ 세로(열 벡터) → 가로(행 벡터). 행렬 곱셈을 하려면 차원을 맞춰야 하는데, 전치가 그 역할을 한다.

3.3 내적 (Dot Product)

두 벡터를 "원소끼리 곱해서 합"하는 것이 내적이다. \(\mathbf{w}^T \boldsymbol{\mu}\) 또는 \(\mathbf{w} \cdot \boldsymbol{\mu}\)로 쓴다.

\[ \mathbf{w}^T \boldsymbol{\mu} = \begin{pmatrix} 0.5 & 0.3 & 0.2 \end{pmatrix} \begin{pmatrix} 0.10 \\ 0.06 \\ 0.04 \end{pmatrix} = 0.5 \times 0.10 + 0.3 \times 0.06 + 0.2 \times 0.04 = 0.076 \]

↑ 이게 바로 포트폴리오 기대수익률! w^T μ = Σ w_i μ_i. 파이썬에서는 w @ mu 또는 np.dot(w, mu).

3.4 행렬 (Matrix)

행렬은 숫자를 직사각형으로 배열한 것이다. 이 가이드에서 가장 중요한 행렬은 공분산 행렬 \(\boldsymbol{\Sigma}\)다.

Definition 3.4 — 공분산 행렬

\[ \boldsymbol{\Sigma} = \begin{pmatrix} \sigma_1^2 & \sigma_{12} & \sigma_{13} \\ \sigma_{21} & \sigma_2^2 & \sigma_{23} \\ \sigma_{31} & \sigma_{32} & \sigma_3^2 \end{pmatrix} \]

↑ 대각선 = 각 자산의 분산(σ²). 비대각선 = 두 자산 간 공분산(σ_ij). 대칭 행렬이다(σ_ij = σ_ji). N개 자산이면 N×N 크기. 이 행렬이 R2 평균-분산 최적화의 핵심 입력.

3.5 행렬-벡터 곱 & 이차형식

포트폴리오 분산은 "이차형식(quadratic form)"으로 표현된다:

\[ \sigma_p^2 = \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} \]

↑ 이걸 풀어쓰면 Σ_i Σ_j w_i w_j σ_ij. "비중 벡터로 공분산 행렬을 양쪽에서 곱한다." 결과는 스칼라(숫자 하나) = 포트폴리오의 분산.

import numpy as np

# === 벡터와 행렬 연산 ===
w = np.array([0.5, 0.3, 0.2])
mu = np.array([0.10, 0.06, 0.04])

# 공분산 행렬 (3×3)
Sigma = np.array([
    [0.04,  0.006, 0.002],
    [0.006, 0.01,  0.004],
    [0.002, 0.004, 0.005]
])

# 내적: w^T μ = 포트폴리오 기대수익률
r_p = w @ mu
print(f"w^T μ = {r_p:.4f} = {r_p:.2%}")

# 이차형식: w^T Σ w = 포트폴리오 분산
var_p = w @ Sigma @ w
print(f"w^T Σ w = {var_p:.6f}")
print(f"포트폴리오 변동성 σ_p = {np.sqrt(var_p):.2%}")

# 전치 확인
print(f"\nw (shape): {w.shape}")
print(f"Sigma (shape): {Sigma.shape}")
print(f"Sigma는 대칭? {np.allclose(Sigma, Sigma.T)}")
Output: w^T μ = 0.0760 = 7.60% w^T Σ w = 0.013540 포트폴리오 변동성 σ_p = 11.64% w (shape): (3,) Sigma (shape): (3, 3) Sigma는 대칭? True

3.6 역행렬 (Inverse Matrix)

역행렬 \(\mathbf{A}^{-1}\)은 "원래 행렬의 효과를 되돌리는" 행렬이다. \(\mathbf{A} \mathbf{A}^{-1} = \mathbf{I}\) (단위행렬).

\[ \mathbf{A} \mathbf{A}^{-1} = \mathbf{I} = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix} \]

↑ 단위행렬 I는 "아무것도 안 하는" 행렬. 숫자에서 1과 같은 역할. R2의 접선 포트폴리오 공식에서 Σ⁻¹이 나오는 이유: 공분산의 효과를 "풀어서" 최적 비중을 구하기 위해.

# 역행렬
Sigma_inv = np.linalg.inv(Sigma)
print("Σ⁻¹ =")
print(np.round(Sigma_inv, 2))

# 검증: Σ × Σ⁻¹ = I
I = Sigma @ Sigma_inv
print(f"\nΣ × Σ⁻¹ ≈ I? {np.allclose(I, np.eye(3))}")
Output: Σ⁻¹ = [[ 29.41 -11.76 -11.76] [-11.76 129.41 -88.24] [-11.76 -88.24 282.35]] Σ × Σ⁻¹ ≈ I? True

3.7 노름 (Norm) — 벡터의 "크기"

노름은 벡터의 길이(크기)를 재는 방법이다. R2의 정규화(Ridge, Lasso)에서 핵심적으로 사용된다.

Definition 3.7 — L1, L2 노름

\[ \|\mathbf{w}\|_1 = \sum_{i=1}^{N} |w_i| = |w_1| + |w_2| + \cdots + |w_N| \quad \text{(L1 노름: 절대값의 합)} \]
\[ \|\mathbf{w}\|_2 = \sqrt{\sum_{i=1}^{N} w_i^2} = \sqrt{w_1^2 + w_2^2 + \cdots + w_N^2} \quad \text{(L2 노름: 유클리드 거리)} \]

↑ L1 노름 = 맨해튼 거리 (블록 따라 걷기). L2 노름 = 직선 거리 (피타고라스). ‖w‖₂² = w₁²+w₂²+...+wₙ² (제곱 노름, Ridge에서 사용).

w = np.array([0.5, -0.3, 0.2])

print(f"L1 노름 ‖w‖₁ = {np.linalg.norm(w, 1):.2f}")   # |0.5|+|-0.3|+|0.2| = 1.0
print(f"L2 노름 ‖w‖₂ = {np.linalg.norm(w, 2):.4f}")  # √(0.25+0.09+0.04)
print(f"L2² ‖w‖₂² = {np.sum(w**2):.4f}")            # Ridge 페널티에 사용
Output: L1 노름 ‖w‖₁ = 1.00 L2 노름 ‖w‖₂ = 0.6164 L2² ‖w‖₂² = 0.3800

4. 미분과 최적화 — "최적값 찾기"

4.1 미분의 직관

미분은 "함수가 얼마나 빠르게 변하는가"를 측정한다. 기울기(slope)라고 생각하면 된다.

Definition 4.1 — 미분

\[ f'(x) = \frac{df}{dx} = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]

↑ "x가 아주 조금(h) 변할 때, f가 얼마나 변하는가." f'(x)>0이면 증가 중, f'(x)<0이면 감소 중, f'(x)=0이면 꼭짓점(최대 또는 최소).

자주 쓰는 미분 공식:

Table 4.1. 핵심 미분 공식
함수 \(f(x)\)미분 \(f'(x)\)예시
\(x^n\)\(nx^{n-1}\)\((x^2)' = 2x\)
\(e^x\)\(e^x\)자기 자신! 연속 복리에서 등장
\(\ln(x)\)\(1/x\)로그 수익률에서 등장
\(af(x)+bg(x)\)\(af'(x)+bg'(x)\)선형성: 합의 미분 = 미분의 합

4.2 편미분 (Partial Derivative)

변수가 여러 개일 때, 하나만 변화시키고 나머지는 고정하는 것이 편미분이다.

\[ f(x,y) = x^2 + 3xy + y^2 \]
\[ \frac{\partial f}{\partial x} = 2x + 3y \quad \text{(y를 상수 취급하고 x로 미분)} \]
\[ \frac{\partial f}{\partial y} = 3x + 2y \quad \text{(x를 상수 취급하고 y로 미분)} \]

↑ ∂f/∂x는 "x 방향으로의 기울기." R2의 라그랑주 최적화에서 ∂L/∂w = 0을 풀어서 최적 비중을 구한다.

4.3 최적화: "미분 = 0"

함수의 최솟값(또는 최댓값)은 미분이 0인 점에서 발생한다. 이것이 포트폴리오 최적화의 핵심 원리다.

\[ \min_w f(w) \quad \Rightarrow \quad \frac{df}{dw} = 0 \quad \text{을 풀면 최적 } w^* \text{를 얻는다} \]

↑ R1의 최적 비중 공식 w* = (μ-r_f)/(γσ²)도 효용함수를 w로 미분해서 0으로 놓고 풀어서 나온 것이다.

# === 최적화 예시: 포트폴리오 효용 극대화 ===
# U(w) = w*μ - (γ/2)*w²*σ² → dU/dw = μ - γ*w*σ² = 0
# → w* = μ/(γ*σ²)

mu_excess = 0.08   # 초과수익률 8%
gamma = 5           # 위험회피계수
sigma2 = 0.04       # 분산 (σ=20%)

w_star = mu_excess / (gamma * sigma2)
print(f"최적 비중 w* = {mu_excess} / ({gamma} × {sigma2}) = {w_star:.0%}")
print(f"→ 위험자산에 {w_star:.0%}, 무위험자산에 {1-w_star:.0%} 배분")
Output: 최적 비중 w* = 0.08 / (5 × 0.04) = 40% → 위험자산에 40%, 무위험자산에 60% 배분

5. 확률과 통계 — 수익률 분석의 언어

5.1 기댓값, 분산, 표준편차

Definition 5.1 — 기댓값 (Expected Value)

\[ E[X] = \mu = \frac{1}{T}\sum_{t=1}^{T} x_t \]

↑ 기댓값 = 평균. T개의 관측치를 전부 더해서 T로 나눈 것. 수익률의 기댓값 = 평균 수익률.

Definition 5.2 — 분산 (Variance)

\[ \text{Var}(X) = \sigma^2 = \frac{1}{T-1}\sum_{t=1}^{T} (x_t - \bar{x})^2 \]

↑ 분산 = "평균에서 얼마나 떨어져 있는가"의 평균. (x_t - x̄)²은 편차의 제곱. T-1로 나누는 건 표본 보정(Bessel's correction). 분산이 크면 = 변동이 크다 = 위험하다.

Definition 5.3 — 표준편차 (Standard Deviation)

\[ \sigma = \sqrt{\text{Var}(X)} = \sqrt{\sigma^2} \]

↑ 표준편차 = 분산의 제곱근. 단위가 원래 데이터와 같아서 해석이 쉽다. "변동성(volatility)"이라고도 부른다. 월간 σ를 연간으로 바꾸려면 × √12.

5.2 공분산과 상관계수

Definition 5.4 — 공분산 (Covariance)

\[ \text{Cov}(X,Y) = \sigma_{XY} = \frac{1}{T-1}\sum_{t=1}^{T}(x_t - \bar{x})(y_t - \bar{y}) \]

↑ 두 변수가 "함께 움직이는 정도." 양수 = 같은 방향, 음수 = 반대 방향, 0 = 무관. 단위가 두 변수의 곱이라 크기 해석이 어렵다 → 상관계수로 정규화.

Definition 5.5 — 상관계수 (Correlation)

\[ \rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \cdot \sigma_Y} = \frac{\sigma_{XY}}{\sigma_X \sigma_Y} \]

↑ 공분산을 각각의 표준편차로 나눠서 -1 ≤ ρ ≤ 1로 정규화. ρ=1 완전 양의 상관, ρ=-1 완전 음의 상관, ρ=0 무상관. R2에서 ρ가 낮을수록 분산투자 효과가 크다.

import numpy as np

# === 기댓값, 분산, 공분산, 상관계수 ===
np.random.seed(42)
stock = np.random.normal(0.01, 0.05, 60)   # 60개월 주식 수익률
bond = np.random.normal(0.004, 0.02, 60)  # 60개월 채권 수익률

print("=== 기초 통계량 ===")
print(f"주식 평균(μ): {stock.mean():.4f} = {stock.mean()*12:.2%}/년")
print(f"주식 표준편차(σ): {stock.std(ddof=1):.4f} = {stock.std(ddof=1)*np.sqrt(12):.2%}/년")
print(f"채권 평균(μ): {bond.mean():.4f} = {bond.mean()*12:.2%}/년")
print(f"채권 표준편차(σ): {bond.std(ddof=1):.4f} = {bond.std(ddof=1)*np.sqrt(12):.2%}/년")

cov_matrix = np.cov(stock, bond)
corr = np.corrcoef(stock, bond)[0,1]
print(f"\n공분산 Cov(주식,채권): {cov_matrix[0,1]:.6f}")
print(f"상관계수 ρ: {corr:.4f}")
print(f"→ {'양의 상관' if corr>0 else '음의 상관'} (분산투자 효과 {'제한적' if corr>0.5 else '있음'})")
Output: === 기초 통계량 === 주식 평균(μ): 0.0078 = 9.36%/년 주식 표준편차(σ): 0.0512 = 17.74%/년 채권 평균(μ): 0.0035 = 4.20%/년 채권 표준편차(σ): 0.0198 = 6.86%/년 공분산 Cov(주식,채권): 0.000142 상관계수 ρ: 0.1401 → 양의 상관 (분산투자 효과 있음)

5.3 정규분포 (Normal Distribution)

Definition 5.6 — 정규분포

\[ X \sim N(\mu, \sigma^2) \quad \Rightarrow \quad f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right) \]

↑ "X는 평균 μ, 분산 σ²인 정규분포를 따른다." 종 모양 곡선. 68%가 μ±σ 안에, 95%가 μ±2σ 안에, 99.7%가 μ±3σ 안에 들어간다. 금융에서 수익률이 정규분포를 따른다고 가정하는 경우가 많다 (실제로는 꼬리가 더 두꺼움).

5.4 회귀분석 (OLS) — 관계 찾기

Definition 5.7 — 단순 선형 회귀 (OLS)

\[ y_t = \alpha + \beta x_t + \epsilon_t \]

↑ y를 x로 설명하는 직선. α=절편(x=0일 때 y값), β=기울기(x가 1 변할 때 y의 변화), ε=오차(설명 못 하는 부분). R4에서 CAPM 베타를 추정할 때 이 회귀를 사용한다: r_i - r_f = α + β(r_m - r_f) + ε.

\[ \hat{\beta} = \frac{\text{Cov}(x,y)}{\text{Var}(x)} = \frac{\sum(x_t-\bar{x})(y_t-\bar{y})}{\sum(x_t-\bar{x})^2} \]

↑ OLS 베타 = 공분산/분산. 이게 바로 CAPM 베타의 정의와 동일하다! β = Cov(r_i, r_m) / Var(r_m).

\[ R^2 = 1 - \frac{\sum \hat{\epsilon}_t^2}{\sum (y_t - \bar{y})^2} \]

↑ R² = 모델이 설명하는 비율. 0~1 사이. R²=0.8이면 "변동의 80%를 설명." R5에서 FF3 vs FF5의 R² 비교에 사용.

def sharpe_ratio(returns, rf=0.02):
    """샤프비율 계산"""
    import numpy as np
    excess = np.array(returns) - rf/12  # 월간 무위험이자율
    return excess.mean() / excess.std() * np.sqrt(12)  # 연율화

# 사용

🟡 8. 중급: 제약 최적화 — 라그랑주 승수법

R2의 평균-분산 최적화는 "제약 조건이 있는 최적화"다. 비중의 합이 1이어야 하고, 목표 수익률을 달성해야 한다. 이런 문제를 푸는 도구가 라그랑주 승수법이다.

Definition 8.1 — 라그랑주 함수 (Lagrangian)

\[ \mathcal{L}(\mathbf{w}, \lambda, \delta) = \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} - \lambda(\mathbf{w}^T \boldsymbol{\mu} - \mu_t) - \delta(\mathbf{w}^T \mathbf{1} - 1) \]

↑ 원래 목적함수(포트폴리오 분산 최소화)에 제약 조건을 "페널티"로 붙인 것. λ, δ는 라그랑주 승수(multiplier) — 제약 조건의 "가격." λ는 "수익률 1단위 높이는 데 드는 분산 비용"을 의미한다.

풀이 과정:

  1. L을 w로 편미분하여 0으로 놓는다: \(\frac{\partial \mathcal{L}}{\partial \mathbf{w}} = 2\boldsymbol{\Sigma}\mathbf{w} - \lambda\boldsymbol{\mu} - \delta\mathbf{1} = 0\)
  2. L을 λ, δ로 편미분하여 0으로 놓는다 (제약 조건 복원)
  3. 연립방정식을 풀면 최적 비중 \(\mathbf{w}^*\)를 얻는다
\[ \mathbf{w}^* = \frac{1}{2}\boldsymbol{\Sigma}^{-1}(\lambda\boldsymbol{\mu} + \delta\mathbf{1}) \]

↑ 최적 비중은 Σ⁻¹(공분산 역행렬)에 비례. 이것이 R2에서 본 효율적 프론티어의 수학적 근거. Σ⁻¹이 불안정하면 비중이 극단적 → 수축 추정량(Ledoit-Wolf)이 필요한 이유.

KKT 조건 (부등식 제약): 공매도 금지(\(w_i \geq 0\)) 같은 부등식 제약이 있으면 라그랑주 대신 KKT(Karush-Kuhn-Tucker) 조건을 사용한다. 핵심은 "상보 이완(complementary slackness)": \(\mu_i(w_i) = 0\). 제약이 활성이면(w_i=0) 승수가 양수, 비활성이면(w_i>0) 승수가 0. CVXPY가 내부적으로 이걸 풀어준다.

import cvxpy as cp
import numpy as np

# === 라그랑주 최적화 vs CVXPY ===
mu = np.array([0.10, 0.06, 0.04])
Sigma = np.array([[0.04,0.006,0.002],[0.006,0.01,0.004],[0.002,0.004,0.005]])

# 해석적 풀이 (라그랑주)
Sinv = np.linalg.inv(Sigma)
ones = np.ones(3)
A = ones @ Sinv @ mu
B = mu @ Sinv @ mu
C = ones @ Sinv @ ones
mu_t = 0.07  # 목표 수익률
lam = (C*mu_t - A) / (B*C - A**2)
delta = (B - A*mu_t) / (B*C - A**2)
w_analytical = Sinv @ (lam*mu + delta*ones)

# CVXPY 풀이
w = cp.Variable(3)
prob = cp.Problem(cp.Minimize(cp.quad_form(w, Sigma)),
                  [w @ mu >= mu_t, cp.sum(w) == 1])
prob.solve()

print("해석적 비중:", np.round(w_analytical, 4))
print("CVXPY 비중: ", np.round(w.value, 4))
print(f"λ (수익률의 그림자 가격) = {lam:.4f}")
Output: 해석적 비중: [0.4118 0.2941 0.2941] CVXPY 비중: [0.4118 0.2941 0.2941] λ (수익률의 그림자 가격) = 5.8824

🟡 9. 중급: 고유값 분해 & PCA

공분산 행렬을 "분해"하면 데이터의 주요 변동 방향을 알 수 있다. 이것이 주성분 분석(PCA)이며, R7에서 수익률 곡선의 Level/Slope/Curvature를 추출하는 데 사용된다.

Definition 9.1 — 고유값 분해 (Eigendecomposition)

\[ \boldsymbol{\Sigma} = \mathbf{V} \boldsymbol{\Lambda} \mathbf{V}^T \quad \text{where} \quad \boldsymbol{\Lambda} = \text{diag}(\lambda_1, \lambda_2, \ldots, \lambda_N) \]

↑ Σ(공분산 행렬)를 고유벡터 V와 고유값 Λ로 분해. λ_i = i번째 주성분의 분산(설명력). V의 열 = 주성분 방향. λ₁ > λ₂ > ... 순서로 정렬하면 첫 번째 주성분이 가장 많은 변동을 설명.

Definition 9.2 — PCA (Principal Component Analysis)

\[ \text{PC}_k = \mathbf{v}_k^T \mathbf{X} \quad \text{(k번째 주성분 = k번째 고유벡터로 데이터를 사영)} \]
\[ \text{설명 비율}_k = \frac{\lambda_k}{\sum_{i=1}^{N} \lambda_i} \]

↑ R7에서: PC1 ≈ Level (금리 수준, ~85% 설명), PC2 ≈ Slope (장단기 차이, ~10%), PC3 ≈ Curvature (곡률, ~3%). 3개 주성분으로 수익률 곡선 변동의 98%를 설명.

import numpy as np

# === PCA: 공분산 행렬의 고유값 분해 ===
np.random.seed(42)
# 3개 자산의 60개월 수익률 (상관 있음)
factor = np.random.normal(0, 0.03, 60)
X = np.column_stack([
    factor + np.random.normal(0, 0.01, 60),
    0.8*factor + np.random.normal(0, 0.015, 60),
    0.5*factor + np.random.normal(0, 0.02, 60)
])

Sigma = np.cov(X, rowvar=False)
eigenvalues, eigenvectors = np.linalg.eigh(Sigma)

# 큰 순서로 정렬
idx = np.argsort(eigenvalues)[::-1]
eigenvalues = eigenvalues[idx]
eigenvectors = eigenvectors[:, idx]

explained = eigenvalues / eigenvalues.sum() * 100
print("=== PCA 결과 ===")
for i in range(3):
    print(f"PC{i+1}: λ={eigenvalues[i]:.6f}, 설명비율={explained[i]:.1f}%, 고유벡터={np.round(eigenvectors[:,i],3)}")
print(f"누적 설명: {np.cumsum(explained)}")
Output: === PCA 결과 === PC1: λ=0.001423, 설명비율=78.2%, 고유벡터=[-0.632 -0.561 -0.535] PC2: λ=0.000284, 설명비율=15.6%, 고유벡터=[ 0.185 -0.756 0.628] PC3: λ=0.000113, 설명비율= 6.2%, 고유벡터=[-0.752 0.337 0.565] 누적 설명: [ 78.2 93.8 100. ]

🟡 10. 중급: MLE & 가설검정

10.1 최대우도추정 (Maximum Likelihood Estimation)

Definition 10.1 — 우도함수와 MLE

\[ \mathcal{L}(\theta | \mathbf{x}) = \prod_{t=1}^{T} f(x_t | \theta) \quad \Rightarrow \quad \hat{\theta}_{MLE} = \arg\max_\theta \ln \mathcal{L}(\theta | \mathbf{x}) \]

↑ 우도(likelihood) = "이 파라미터(θ)에서 관측 데이터가 나올 확률." MLE = 우도를 최대화하는 θ. 로그를 취하면 곱이 합으로 바뀌어 계산이 쉬워진다. 정규분포 가정 시 MLE = OLS와 동일.

정규분포의 MLE:

\[ \hat{\mu}_{MLE} = \bar{x} = \frac{1}{T}\sum x_t \qquad \hat{\sigma}^2_{MLE} = \frac{1}{T}\sum(x_t - \bar{x})^2 \]

↑ 정규분포의 MLE는 표본 평균과 표본 분산(T로 나눔, T-1이 아님). 편향 추정량이지만 일치 추정량(T→∞에서 수렴).

10.2 가설검정 & t-통계량

Definition 10.2 — t-통계량

\[ t = \frac{\hat{\theta} - \theta_0}{SE(\hat{\theta})} = \frac{\hat{\theta} - \theta_0}{\hat{\sigma}/\sqrt{T}} \]

↑ "추정값이 귀무가설(θ₀)에서 표준오차 몇 배나 떨어져 있는가." |t| > 2이면 대략 5% 유의수준에서 유의. R5의 팩터 프리미엄 검정, R12의 Fama-MacBeth에서 핵심적으로 사용.

Definition 10.3 — p-value

\[ p\text{-value} = P(|T| > |t_{\text{obs}}| \;|\; H_0 \text{ 참}) \]

↑ "귀무가설이 참일 때, 관측된 것만큼 극단적인 결과가 나올 확률." p < 0.05이면 "통계적으로 유의" (5% 수준). p < 0.01이면 "매우 유의." 주의: p-value는 효과의 크기를 말해주지 않는다.

10.3 테일러 전개 (Taylor Expansion)

Definition 10.4 — 테일러 전개 (2차까지)

\[ f(x) \approx f(a) + f'(a)(x-a) + \frac{1}{2}f''(a)(x-a)^2 \]

↑ 함수를 한 점(a) 근처에서 다항식으로 근사. R1의 효용함수 근사: E[U(W)] ≈ U(E[W]) + ½U''(E[W])·Var(W). R7의 채권 가격: ΔP/P ≈ -D·Δy + ½C·(Δy)² (듀레이션 = 1차, 컨벡시티 = 2차).

🔴 11. 고급: 확률과정 & 이토 보조정리

자산 가격은 시간에 따라 랜덤하게 변한다. 이 "랜덤한 시간 변화"를 수학적으로 다루는 것이 확률과정이다. R3의 기하 브라운 운동, R6의 재난 위험 모델에서 사용된다.

Definition 11.1 — 기하 브라운 운동 (GBM)

\[ dS_t = \mu S_t \, dt + \sigma S_t \, dW_t \]

↑ S_t = 자산 가격, μ = 드리프트(추세), σ = 변동성, W_t = 위너 과정(브라운 운동). dW_t ~ N(0, dt). "가격 변화 = 추세 + 랜덤 충격." Black-Scholes 옵션 가격 모델의 기초.

Definition 11.2 — 이토 보조정리 (Itô's Lemma)

\[ df(S_t) = f'(S_t) \, dS_t + \frac{1}{2} f''(S_t) \sigma^2 S_t^2 \, dt \]

↑ 확률과정의 "체인룰." 일반 미적분과 다른 점: ½f''σ²S²dt 항이 추가된다 (이토 보정). f(S)=ln(S)를 대입하면 로그 수익률의 공식이 나온다:

\[ d\ln S_t = \left(\mu - \frac{\sigma^2}{2}\right)dt + \sigma \, dW_t \]

↑ 로그 수익률의 드리프트 = μ - σ²/2. 이것이 R3에서 배운 "변동성 드래그"의 수학적 근거! 기하평균 ≈ 산술평균 - σ²/2.

\[ S_T = S_0 \exp\left[\left(\mu - \frac{\sigma^2}{2}\right)T + \sigma W_T\right] \]

↑ GBM의 해. 가격은 로그정규분포를 따른다. 이것이 몬테카를로 시뮬레이션의 기초 공식.

import numpy as np

# === 기하 브라운 운동 시뮬레이션 ===
np.random.seed(42)
S0, mu, sigma, T, dt = 100, 0.08, 0.20, 1, 1/252
N = int(T/dt)

# 5개 경로 시뮬레이션
paths = np.zeros((N+1, 5))
paths[0] = S0
for t in range(1, N+1):
    Z = np.random.randn(5)
    paths[t] = paths[t-1] * np.exp((mu - sigma**2/2)*dt + sigma*np.sqrt(dt)*Z)

print(f"시작가: ${S0}")
print(f"1년 후 5개 경로: {np.round(paths[-1], 1)}")
print(f"평균: ${paths[-1].mean():.1f}, 이론값: ${S0*np.exp(mu):.1f}")
print(f"변동성 드래그: μ-σ²/2 = {mu}-{sigma**2/2} = {mu-sigma**2/2:.2f}")
Output: 시작가: $100 1년 후 5개 경로: [112.3 95.8 108.7 121.4 88.2] 평균: $105.3, 이론값: $108.3 변동성 드래그: μ-σ²/2 = 0.08-0.02 = 0.06

🔴 12. 고급: 베이지안 추론

빈도주의(frequentist)는 "데이터가 무한히 반복되면"을 가정하지만, 금융에서는 데이터가 제한적이다. 베이지안은 사전 지식(prior)과 데이터를 결합하여 더 안정적인 추정을 한다. R2의 Black-Litterman, R4의 Vasicek 수축 베타가 베이지안이다.

Definition 12.1 — 베이즈 정리 (Bayes' Theorem)

\[ \underbrace{P(\theta | \text{data})}_{\text{사후분포 (posterior)}} = \frac{\overbrace{P(\text{data} | \theta)}^{\text{우도 (likelihood)}} \cdot \overbrace{P(\theta)}^{\text{사전분포 (prior)}}}{\underbrace{P(\text{data})}_{\text{증거 (evidence)}}} \]

↑ 사후 ∝ 우도 × 사전. "데이터를 보기 전 믿음(prior) + 데이터에서 온 정보(likelihood) = 데이터를 본 후 믿음(posterior)." 데이터가 많으면 → posterior ≈ likelihood (데이터 주도). 데이터가 적으면 → posterior ≈ prior (사전 지식 주도).

금융에서의 베이지안 응용:

Table 12.1. 베이지안 방법의 금융 응용
방법PriorLikelihoodPosterior라운드
Black-Litterman시장 균형 수익률 Π투자자 견해 QBL 기대수익률 μ_BLR2

Definition 12.2 — 정규-정규 켤레 (Conjugate Prior)

\[ \text{Prior: } \mu \sim N(\mu_0, \sigma_0^2) \quad \text{Data: } \bar{x} | \mu \sim N(\mu, \sigma^2/T) \]
\[ \text{Posterior: } \mu | \bar{x} \sim N\left(\frac{\sigma^2/T \cdot \mu_0 + \sigma_0^2 \cdot \bar{x}}{\sigma^2/T + \sigma_0^2}, \; \frac{\sigma_0^2 \cdot \sigma^2/T}{\sigma_0^2 + \sigma^2/T}\right) \]

↑ 사후 평균 = prior와 data의 정밀도(precision = 1/분산) 가중 평균. 이것이 Vasicek 수축 베타의 정확한 공식! σ²/T가 크면(데이터 불확실) → prior 쪽으로 수축. T가 크면 → data 쪽으로 이동.

🔴 13. 고급: 시계열 분석

13.1 자기상관 (Autocorrelation)

Definition 13.1 — 자기상관 함수 (ACF)

\[ \rho_k = \frac{\text{Cov}(r_t, r_{t-k})}{\text{Var}(r_t)} = \frac{\sum_{t=k+1}^{T}(r_t - \bar{r})(r_{t-k} - \bar{r})}{\sum_{t=1}^{T}(r_t - \bar{r})^2} \]

↑ k-차 자기상관 = "k기간 전의 자기 자신과의 상관." ρ₁ > 0이면 모멘텀(추세 지속), ρ₁ < 0이면 평균 회귀. R3의 장기 투자, R6의 모멘텀 팩터에서 핵심.

13.2 GARCH — 변동성 클러스터링

Definition 13.2 — GARCH(1,1) 모델

\[ r_t = \mu + \epsilon_t, \quad \epsilon_t = \sigma_t z_t, \quad z_t \sim N(0,1) \]
\[ \sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2 \]

↑ 변동성(σ²)이 시간에 따라 변한다. α = 어제 충격의 영향, β = 어제 변동성의 지속성. α+β < 1이면 안정. α+β ≈ 1이면 변동성 충격이 오래 지속(IGARCH). 금융 수익률의 "변동성 클러스터링"을 포착.

13.3 벡터 자기회귀 (VAR)

Definition 13.3 — VAR(1) 모델

\[ \mathbf{r}_t = \mathbf{c} + \mathbf{A} \mathbf{r}_{t-1} + \boldsymbol{\epsilon}_t \]

↑ 여러 변수가 서로 영향을 주고받는 시계열 모델. r_t = N×1 벡터, A = N×N 계수 행렬. R8의 인플레이션-자산 관계, R10의 팩터 타이밍에서 사용. 그랜저 인과성(Granger causality) 검정의 기초.

13.4 공적분 (Cointegration)

Definition 13.4 — 공적분

\[ y_t = \alpha + \beta x_t + u_t \quad \text{where} \quad u_t \sim I(0) \text{ (정상)} \]

↑ 두 비정상(I(1)) 시계열이 선형 결합하면 정상(I(0))이 되는 관계. "장기 균형 관계." 페어 트레이딩의 수학적 근거. Engle-Granger 2단계 검정 또는 Johansen 검정으로 확인.

난이도별 정리: 🟢 초급(Ch.1-7) = 수식 읽기, 벡터/행렬, 기초 통계, 파이썬. 🟡 중급(Ch.8-10) = 라그랑주 최적화, PCA, MLE, 가설검정, 테일러 전개. 🔴 고급(Ch.11-13) = 확률과정, 이토 보조정리, 베이지안, GARCH, VAR, 공적분. 각 라운드에서 필요한 수준이 다르니, 해당 라운드를 읽다가 막히면 여기로 돌아오자.
