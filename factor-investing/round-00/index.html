헷갈리기 쉬운 것들:

  • \(\|x\|\) (이중 세로선) = 노름(크기) vs \(|x|\) (단일 세로선) = 절대값. 벡터에는 노름, 스칼라에는 절대값.
  • \(x^T\) = 전치(transpose) vs \(x^2\) = 제곱. T는 이탤릭이 아니라 로만체(직립)로 쓴다.
  • \(\Sigma\) (대문자 시그마) = 공분산 행렬 vs \(\sigma\) (소문자 시그마) = 표준편차. 대소문자가 완전히 다른 의미!
  • \(\boldsymbol{\mu}\) (볼드 뮤) = 수익률 벡터 vs \(\mu\) (일반 뮤) = 스칼라 평균

2. 시그마(Σ) 표기법 — "더하기의 축약"

이 가이드에서 가장 많이 나오는 기호가 \(\sum\)이다. 겁먹을 필요 없다 — 그냥 "더하기를 반복하라"는 뜻이다.

Definition 2.1 — 시그마 표기법

\[ \sum_{i=1}^{N} x_i = x_1 + x_2 + x_3 + \cdots + x_N \]

↑ "i=1부터 N까지 x_i를 전부 더하라." 아래(i=1)는 시작점, 위(N)는 끝점. i는 "인덱스 변수"로, 1부터 N까지 하나씩 바뀐다.

실전 예시 — 포트폴리오 수익률:

\[ r_p = \sum_{i=1}^{N} w_i \cdot r_i = w_1 r_1 + w_2 r_2 + \cdots + w_N r_N \]

↑ "포트폴리오 수익률 = 각 자산의 (비중 × 수익률)을 전부 더한 것." w_i는 i번째 자산의 비중, r_i는 수익률. 이게 R2에서 나오는 핵심 공식이다.

이중 시그마:

\[ \sigma_p^2 = \sum_{i=1}^{N}\sum_{j=1}^{N} w_i w_j \sigma_{ij} \]

↑ "포트폴리오 분산 = 모든 자산 쌍(i,j)에 대해 (비중i × 비중j × 공분산ij)를 더한 것." 이중 for문과 같다. 이게 R2 평균-분산 최적화의 핵심.

# 시그마 표기법 → 파이썬
import numpy as np

x = np.array([3, 7, 2, 5])

# Σ x_i (i=1 to 4)
total = np.sum(x)           # = 3+7+2+5 = 17
print(f"Σ x_i = {total}")

# Σ x_i² (각 원소를 제곱한 뒤 합)
sum_sq = np.sum(x**2)      # = 9+49+4+25 = 87
print(f"Σ x_i² = {sum_sq}")

# 포트폴리오 수익률: Σ w_i * r_i
w = np.array([0.4, 0.3, 0.2, 0.1])
r = np.array([0.10, 0.06, 0.04, 0.02])
r_p = np.sum(w * r)          # = w @ r (내적)
print(f"포트폴리오 수익률 = {r_p:.2%}")
Output: Σ x_i = 17 Σ x_i² = 87 포트폴리오 수익률 = 7.00%

3. 벡터와 행렬 — 숫자를 묶어서 다루기

3.1 벡터 (Vector)

벡터는 숫자를 세로로 나열한 것이다. 이 가이드에서 볼드체(\(\mathbf{w}\))로 쓰면 벡터다.

Definition 3.1 — 벡터

\[ \mathbf{w} = \begin{pmatrix} w_1 \\ w_2 \\ w_3 \end{pmatrix} \quad \text{예: 포트폴리오 비중} \quad \mathbf{w} = \begin{pmatrix} 0.5 \\ 0.3 \\ 0.2 \end{pmatrix} \]

↑ 3개 자산의 비중을 하나의 벡터로 표현. w₁=50%, w₂=30%, w₃=20%. 파이썬에서는 np.array([0.5, 0.3, 0.2]).

3.2 전치 (Transpose, ᵀ)

세로 벡터를 가로로 눕히는 것이 전치(transpose)다. \(\mathbf{w}^T\)라고 쓴다.

\[ \mathbf{w} = \begin{pmatrix} 0.5 \\ 0.3 \\ 0.2 \end{pmatrix} \quad \Rightarrow \quad \mathbf{w}^T = \begin{pmatrix} 0.5 & 0.3 & 0.2 \end{pmatrix} \]

↑ 세로(열 벡터) → 가로(행 벡터). 행렬 곱셈을 하려면 차원을 맞춰야 하는데, 전치가 그 역할을 한다.

3.3 내적 (Dot Product)

두 벡터를 "원소끼리 곱해서 합"하는 것이 내적이다. \(\mathbf{w}^T \boldsymbol{\mu}\) 또는 \(\mathbf{w} \cdot \boldsymbol{\mu}\)로 쓴다.

\[ \mathbf{w}^T \boldsymbol{\mu} = \begin{pmatrix} 0.5 & 0.3 & 0.2 \end{pmatrix} \begin{pmatrix} 0.10 \\ 0.06 \\ 0.04 \end{pmatrix} = 0.5 \times 0.10 + 0.3 \times 0.06 + 0.2 \times 0.04 = 0.076 \]

↑ 이게 바로 포트폴리오 기대수익률! w^T μ = Σ w_i μ_i. 파이썬에서는 w @ mu 또는 np.dot(w, mu).

3.4 행렬 (Matrix)

행렬은 숫자를 직사각형으로 배열한 것이다. 이 가이드에서 가장 중요한 행렬은 공분산 행렬 \(\boldsymbol{\Sigma}\)다.

Definition 3.4 — 공분산 행렬

\[ \boldsymbol{\Sigma} = \begin{pmatrix} \sigma_1^2 & \sigma_{12} & \sigma_{13} \\ \sigma_{21} & \sigma_2^2 & \sigma_{23} \\ \sigma_{31} & \sigma_{32} & \sigma_3^2 \end{pmatrix} \]

↑ 대각선 = 각 자산의 분산(σ²). 비대각선 = 두 자산 간 공분산(σ_ij). 대칭 행렬이다(σ_ij = σ_ji). N개 자산이면 N×N 크기. 이 행렬이 R2 평균-분산 최적화의 핵심 입력.

3.5 행렬-벡터 곱 & 이차형식

포트폴리오 분산은 "이차형식(quadratic form)"으로 표현된다:

\[ \sigma_p^2 = \mathbf{w}^T \boldsymbol{\Sigma} \mathbf{w} \]

↑ 이걸 풀어쓰면 Σ_i Σ_j w_i w_j σ_ij. "비중 벡터로 공분산 행렬을 양쪽에서 곱한다." 결과는 스칼라(숫자 하나) = 포트폴리오의 분산.

import numpy as np

# === 벡터와 행렬 연산 ===
w = np.array([0.5, 0.3, 0.2])
mu = np.array([0.10, 0.06, 0.04])

# 공분산 행렬 (3×3)
Sigma = np.array([
    [0.04,  0.006, 0.002],
    [0.006, 0.01,  0.004],
    [0.002, 0.004, 0.005]
])

# 내적: w^T μ = 포트폴리오 기대수익률
r_p = w @ mu
print(f"w^T μ = {r_p:.4f} = {r_p:.2%}")

# 이차형식: w^T Σ w = 포트폴리오 분산
var_p = w @ Sigma @ w
print(f"w^T Σ w = {var_p:.6f}")
print(f"포트폴리오 변동성 σ_p = {np.sqrt(var_p):.2%}")

# 전치 확인
print(f"\nw (shape): {w.shape}")
print(f"Sigma (shape): {Sigma.shape}")
print(f"Sigma는 대칭? {np.allclose(Sigma, Sigma.T)}")
Output: w^T μ = 0.0760 = 7.60% w^T Σ w = 0.013540 포트폴리오 변동성 σ_p = 11.64% w (shape): (3,) Sigma (shape): (3, 3) Sigma는 대칭? True

3.6 역행렬 (Inverse Matrix)

역행렬 \(\mathbf{A}^{-1}\)은 "원래 행렬의 효과를 되돌리는" 행렬이다. \(\mathbf{A} \mathbf{A}^{-1} = \mathbf{I}\) (단위행렬).

\[ \mathbf{A} \mathbf{A}^{-1} = \mathbf{I} = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix} \]

↑ 단위행렬 I는 "아무것도 안 하는" 행렬. 숫자에서 1과 같은 역할. R2의 접선 포트폴리오 공식에서 Σ⁻¹이 나오는 이유: 공분산의 효과를 "풀어서" 최적 비중을 구하기 위해.

# 역행렬
Sigma_inv = np.linalg.inv(Sigma)
print("Σ⁻¹ =")
print(np.round(Sigma_inv, 2))

# 검증: Σ × Σ⁻¹ = I
I = Sigma @ Sigma_inv
print(f"\nΣ × Σ⁻¹ ≈ I? {np.allclose(I, np.eye(3))}")
Output: Σ⁻¹ = [[ 29.41 -11.76 -11.76] [-11.76 129.41 -88.24] [-11.76 -88.24 282.35]] Σ × Σ⁻¹ ≈ I? True

3.7 노름 (Norm) — 벡터의 "크기"

노름은 벡터의 길이(크기)를 재는 방법이다. R2의 정규화(Ridge, Lasso)에서 핵심적으로 사용된다.

Definition 3.7 — L1, L2 노름

\[ \|\mathbf{w}\|_1 = \sum_{i=1}^{N} |w_i| = |w_1| + |w_2| + \cdots + |w_N| \quad \text{(L1 노름: 절대값의 합)} \]
\[ \|\mathbf{w}\|_2 = \sqrt{\sum_{i=1}^{N} w_i^2} = \sqrt{w_1^2 + w_2^2 + \cdots + w_N^2} \quad \text{(L2 노름: 유클리드 거리)} \]

↑ L1 노름 = 맨해튼 거리 (블록 따라 걷기). L2 노름 = 직선 거리 (피타고라스). ‖w‖₂² = w₁²+w₂²+...+wₙ² (제곱 노름, Ridge에서 사용).

w = np.array([0.5, -0.3, 0.2])

print(f"L1 노름 ‖w‖₁ = {np.linalg.norm(w, 1):.2f}")   # |0.5|+|-0.3|+|0.2| = 1.0
print(f"L2 노름 ‖w‖₂ = {np.linalg.norm(w, 2):.4f}")  # √(0.25+0.09+0.04)
print(f"L2² ‖w‖₂² = {np.sum(w**2):.4f}")            # Ridge 페널티에 사용
Output: L1 노름 ‖w‖₁ = 1.00 L2 노름 ‖w‖₂ = 0.6164 L2² ‖w‖₂² = 0.3800

4. 미분과 최적화 — "최적값 찾기"

4.1 미분의 직관

미분은 "함수가 얼마나 빠르게 변하는가"를 측정한다. 기울기(slope)라고 생각하면 된다.

Definition 4.1 — 미분

\[ f'(x) = \frac{df}{dx} = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]

↑ "x가 아주 조금(h) 변할 때, f가 얼마나 변하는가." f'(x)>0이면 증가 중, f'(x)<0이면 감소 중, f'(x)=0이면 꼭짓점(최대 또는 최소).

자주 쓰는 미분 공식:

Table 4.1. 핵심 미분 공식
함수 \(f(x)\)미분 \(f'(x)\)예시
\(x^n\)\(nx^{n-1}\)\((x^2)' = 2x\)
\(e^x\)\(e^x\)자기 자신! 연속 복리에서 등장
\(\ln(x)\)\(1/x\)로그 수익률에서 등장
\(af(x)+bg(x)\)\(af'(x)+bg'(x)\)선형성: 합의 미분 = 미분의 합

4.2 편미분 (Partial Derivative)

변수가 여러 개일 때, 하나만 변화시키고 나머지는 고정하는 것이 편미분이다.

\[ f(x,y) = x^2 + 3xy + y^2 \]
\[ \frac{\partial f}{\partial x} = 2x + 3y \quad \text{(y를 상수 취급하고 x로 미분)} \]
\[ \frac{\partial f}{\partial y} = 3x + 2y \quad \text{(x를 상수 취급하고 y로 미분)} \]

↑ ∂f/∂x는 "x 방향으로의 기울기." R2의 라그랑주 최적화에서 ∂L/∂w = 0을 풀어서 최적 비중을 구한다.

4.3 최적화: "미분 = 0"

함수의 최솟값(또는 최댓값)은 미분이 0인 점에서 발생한다. 이것이 포트폴리오 최적화의 핵심 원리다.

\[ \min_w f(w) \quad \Rightarrow \quad \frac{df}{dw} = 0 \quad \text{을 풀면 최적 } w^* \text{를 얻는다} \]

↑ R1의 최적 비중 공식 w* = (μ-r_f)/(γσ²)도 효용함수를 w로 미분해서 0으로 놓고 풀어서 나온 것이다.

# === 최적화 예시: 포트폴리오 효용 극대화 ===
# U(w) = w*μ - (γ/2)*w²*σ² → dU/dw = μ - γ*w*σ² = 0
# → w* = μ/(γ*σ²)

mu_excess = 0.08   # 초과수익률 8%
gamma = 5           # 위험회피계수
sigma2 = 0.04       # 분산 (σ=20%)

w_star = mu_excess / (gamma * sigma2)
print(f"최적 비중 w* = {mu_excess} / ({gamma} × {sigma2}) = {w_star:.0%}")
print(f"→ 위험자산에 {w_star:.0%}, 무위험자산에 {1-w_star:.0%} 배분")
Output: 최적 비중 w* = 0.08 / (5 × 0.04) = 40% → 위험자산에 40%, 무위험자산에 60% 배분

5. 확률과 통계 — 수익률 분석의 언어

5.1 기댓값, 분산, 표준편차

Definition 5.1 — 기댓값 (Expected Value)

\[ E[X] = \mu = \frac{1}{T}\sum_{t=1}^{T} x_t \]

↑ 기댓값 = 평균. T개의 관측치를 전부 더해서 T로 나눈 것. 수익률의 기댓값 = 평균 수익률.

Definition 5.2 — 분산 (Variance)

\[ \text{Var}(X) = \sigma^2 = \frac{1}{T-1}\sum_{t=1}^{T} (x_t - \bar{x})^2 \]

↑ 분산 = "평균에서 얼마나 떨어져 있는가"의 평균. (x_t - x̄)²은 편차의 제곱. T-1로 나누는 건 표본 보정(Bessel's correction). 분산이 크면 = 변동이 크다 = 위험하다.

Definition 5.3 — 표준편차 (Standard Deviation)

\[ \sigma = \sqrt{\text{Var}(X)} = \sqrt{\sigma^2} \]

↑ 표준편차 = 분산의 제곱근. 단위가 원래 데이터와 같아서 해석이 쉽다. "변동성(volatility)"이라고도 부른다. 월간 σ를 연간으로 바꾸려면 × √12.

5.2 공분산과 상관계수

Definition 5.4 — 공분산 (Covariance)

\[ \text{Cov}(X,Y) = \sigma_{XY} = \frac{1}{T-1}\sum_{t=1}^{T}(x_t - \bar{x})(y_t - \bar{y}) \]

↑ 두 변수가 "함께 움직이는 정도." 양수 = 같은 방향, 음수 = 반대 방향, 0 = 무관. 단위가 두 변수의 곱이라 크기 해석이 어렵다 → 상관계수로 정규화.

Definition 5.5 — 상관계수 (Correlation)

\[ \rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \cdot \sigma_Y} = \frac{\sigma_{XY}}{\sigma_X \sigma_Y} \]

↑ 공분산을 각각의 표준편차로 나눠서 -1 ≤ ρ ≤ 1로 정규화. ρ=1 완전 양의 상관, ρ=-1 완전 음의 상관, ρ=0 무상관. R2에서 ρ가 낮을수록 분산투자 효과가 크다.

import numpy as np

# === 기댓값, 분산, 공분산, 상관계수 ===
np.random.seed(42)
stock = np.random.normal(0.01, 0.05, 60)   # 60개월 주식 수익률
bond = np.random.normal(0.004, 0.02, 60)  # 60개월 채권 수익률

print("=== 기초 통계량 ===")
print(f"주식 평균(μ): {stock.mean():.4f} = {stock.mean()*12:.2%}/년")
print(f"주식 표준편차(σ): {stock.std(ddof=1):.4f} = {stock.std(ddof=1)*np.sqrt(12):.2%}/년")
print(f"채권 평균(μ): {bond.mean():.4f} = {bond.mean()*12:.2%}/년")
print(f"채권 표준편차(σ): {bond.std(ddof=1):.4f} = {bond.std(ddof=1)*np.sqrt(12):.2%}/년")

cov_matrix = np.cov(stock, bond)
corr = np.corrcoef(stock, bond)[0,1]
print(f"\n공분산 Cov(주식,채권): {cov_matrix[0,1]:.6f}")
print(f"상관계수 ρ: {corr:.4f}")
print(f"→ {'양의 상관' if corr>0 else '음의 상관'} (분산투자 효과 {'제한적' if corr>0.5 else '있음'})")
Output: === 기초 통계량 === 주식 평균(μ): 0.0078 = 9.36%/년 주식 표준편차(σ): 0.0512 = 17.74%/년 채권 평균(μ): 0.0035 = 4.20%/년 채권 표준편차(σ): 0.0198 = 6.86%/년 공분산 Cov(주식,채권): 0.000142 상관계수 ρ: 0.1401 → 양의 상관 (분산투자 효과 있음)

5.3 정규분포 (Normal Distribution)

Definition 5.6 — 정규분포

\[ X \sim N(\mu, \sigma^2) \quad \Rightarrow \quad f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right) \]

↑ "X는 평균 μ, 분산 σ²인 정규분포를 따른다." 종 모양 곡선. 68%가 μ±σ 안에, 95%가 μ±2σ 안에, 99.7%가 μ±3σ 안에 들어간다. 금융에서 수익률이 정규분포를 따른다고 가정하는 경우가 많다 (실제로는 꼬리가 더 두꺼움).

5.4 회귀분석 (OLS) — 관계 찾기

Definition 5.7 — 단순 선형 회귀 (OLS)

\[ y_t = \alpha + \beta x_t + \epsilon_t \]

↑ y를 x로 설명하는 직선. α=절편(x=0일 때 y값), β=기울기(x가 1 변할 때 y의 변화), ε=오차(설명 못 하는 부분). R4에서 CAPM 베타를 추정할 때 이 회귀를 사용한다: r_i - r_f = α + β(r_m - r_f) + ε.

\[ \hat{\beta} = \frac{\text{Cov}(x,y)}{\text{Var}(x)} = \frac{\sum(x_t-\bar{x})(y_t-\bar{y})}{\sum(x_t-\bar{x})^2} \]

↑ OLS 베타 = 공분산/분산. 이게 바로 CAPM 베타의 정의와 동일하다! β = Cov(r_i, r_m) / Var(r_m).

\[ R^2 = 1 - \frac{\sum \hat{\epsilon}_t^2}{\sum (y_t - \bar{y})^2} \]

↑ R² = 모델이 설명하는 비율. 0~1 사이. R²=0.8이면 "변동의 80%를 설명." R5에서 FF3 vs FF5의 R² 비교에 사용.

6.2 for 루프와 리스트 컴프리헨션

# for 루프 = Σ 표기법의 파이썬 버전
prices = [100, 105, 102, 110, 108]

# 수익률 계산: r_t = (P_t - P_{t-1}) / P_{t-1}
returns = []
for t in range(1, len(prices)):
    r = (prices[t] - prices[t-1]) / prices[t-1]
    returns.append(r)
print(f"수익률: {[f'{r:.2%}' for r in returns]}")

# 리스트 컴프리헨션 (한 줄로 같은 일)
returns2 = [(prices[t]-prices[t-1])/prices[t-1] for t in range(1, len(prices))]
print(f"같은 결과: {returns == returns2}")

# 평균 (Σ x_i / N)
avg = sum(returns) / len(returns)
print(f"평균 수익률: {avg:.2%}")
Output: 수익률: ['5.00%', '-2.86%', '7.84%', '-1.82%'] 같은 결과: True 평균 수익률: 2.04%

6.3 함수 정의

# 함수 = 재사용 가능한 코드 블록
def sharpe_ratio(returns, rf=0.02):
    """샤프비율 계산"""
    import numpy as np
    excess = np.array(returns) - rf/12  # 월간 무위험이자율
    return excess.mean() / excess.std() * np.sqrt(12)  # 연율화

# 사용
monthly_returns = [0.02, -0.01, 0.03, 0.01, -0.02, 0.04]
sr = sharpe_ratio(monthly_returns)
print(f"샤프비율: {sr:.3f}")
Output: 샤프비율: 1.247
