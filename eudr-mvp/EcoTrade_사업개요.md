# EcoTrade — 사업개요

## One-liner

> EU 환경규제(EUDR·CBAM) 대응을 위한 AI 기반 통합 컴플라이언스 플랫폼

---

## 문제 (Problem)

### 한국 수출기업이 직면한 현실

1. **EUDR (2025.12.30 시행)**
   - EU에 팜유/목재/커피/코코아/대두/고무/소 관련 제품 수출 시, 공급지가 2020.12.31 이후 산림 파괴 없음을 증명해야 함
   - 증명 실패 시 EU 시장 진입 불가 + 매출액 4% 벌금

2. **CBAM (2026.1 본격 전환)**
   - 철강/알루미늄/시멘트/비료/전력/수소 수출 시, 제품 내재 탄소량에 대해 EU 탄소 가격 지불
   - 한국 탄소 가격과의 차액만큼 추가 비용 발생 → 가격 경쟁력 직격

### 현재 대응 상태

- 대기업: 자체 ESG팀 + 유럽 컨설팅펌 고용 (비용 수억)
- 중소기업: **아무것도 못 하고 있음** — 규제 내용도 모르고, 어디서 시작할지도 모름
- 한국 내 전문 플랫폼: **없음**

---

## 솔루션 (Solution)

### 모듈 1: EUDR 산림검증 엔진

```
공급지 GPS 좌표 입력
→ 위성 이미지 수집 (Sentinel-2, 10m)
→ AI 분석 (GAN 기반 계절성 보정 + 시계열 변화 탐지)
→ 산림 파괴 여부 자동 판정
→ Due Diligence Statement 자동 생성
```

**핵심 AI 기술: GAN 기반 계절성 보정**

| 문제 | 기존 방법의 한계 | EcoTrade 접근 |
|------|-----------------|---------------|
| 열대림 건기/우기 NDVI 차이 | 단순 이미지 차분 → 계절 변화를 파괴로 오인 | GAN으로 계절별 정상 상태 생성 → 실제 관측과 비교하여 비정상(파괴) 탐지 |
| 구름 피복 | 구름 아래 데이터 소실 | GAN으로 구름 제거/복원 |
| 시계열 연속성 | 16일 주기 관측 → 중간 공백 | GAN으로 관측 공백 보간(interpolation) |

동작 원리:
1. 과거 3~5년 위성 시계열로 해당 지점의 **정상 계절 패턴** 학습
2. GAN이 "만약 파괴가 없었다면 현재 어떤 모습일지" 생성
3. 생성된 이미지 vs 실제 관측 이미지 비교 → 차이가 계절성으로 설명 불가능하면 = 파괴

### 모듈 2: CBAM 탄소비용 시뮬레이터

```
수출입 서류 업로드 (BL, Invoice, CO, 공정명세서)
→ AI OCR 문서 인식 (Fine-tuned 모델)
→ 제품별 내재 탄소량 자동 산정
→ EU ETS 가격 vs 한국 K-ETS 가격 차이 계산
→ 추가 비용 산출 + 절감 시나리오 시뮬레이션
```

**핵심 AI 기술: Fine-tuned OCR**

| 기능 | 설명 |
|------|------|
| 서류 자동 인식 | B/L, Commercial Invoice, Certificate of Origin, 공정명세서 등 다양한 양식 |
| 필드 추출 | HS Code, 원산지, 중량, 제조공정, 원자재 정보 자동 추출 |
| 배출계수 매핑 | 추출된 정보 → IPCC/EU 기본값 또는 실측값 매핑 |
| 다국어 | 한국어/영어/중국어 혼재 서류 처리 |

### 모듈 3: 통합 대시보드 & 리포팅

- 공급망 전체 리스크 맵 (지도 시각화)
- 연도별 산림 파괴 지수 추이
- CBAM 비용 전망 (탄소가격 시나리오별)
- EU 제출용 보고서 자동 생성 (DDS, CBAM Declaration)
- 경고 시스템: 공급지 리스크 변동 시 알림

## 기술 차별화

### 왜 단순 이미지 차분이 안 되는가

```
열대림 NDVI 시계열:

건기:  ████████░░ (NDVI ~0.55)
우기:  ██████████ (NDVI ~0.80)
파괴:  ███░░░░░░░ (NDVI ~0.25)
```

단순 차분: 건기 → 우기 변화를 "산림 회복"으로, 우기 → 건기를 "산림 파괴"로 오인할 수 있음.

**GAN 기반 접근:**
- Conditional GAN이 해당 좌표의 계절·기후 조건을 입력받아 "정상 상태" 이미지 생성
- 정상 상태 vs 실제 관측의 잔차(residual)만 분석
- 계절성 제거 후 순수한 토지 전용(land use change)만 탐지

### 선행 연구 기반

- 연구실 선배 논문: GAN 생성 이미지 해석을 통한 위성 이미지 변화 탐지
- Conditional GAN / Pix2Pix 기반 시계열 정상화
- EUDR 맥락에서의 응용은 아직 상용화 사례 없음 → **선점 기회**

## 비즈니스 모델

### 타겟 고객

| 세그먼트 | 규모 | 페인포인트 | 예시 |
|----------|------|-----------|------|
| EUDR 대상 수출기업 | ~500개사 | 공급지 검증 불가 | 팜유 수입/가공, 목재·가구, 식품(커피/코코아) |
| CBAM 대상 수출기업 | ~2,000개사 | 탄소비용 예측 불가 | 포스코, 현대제철, 쌍용C&E, 알루미늄 가공업체 |
| 중소 무역회사 | ~5,000개사 | 규제 자체를 모름 | EU향 중간재 수출 중소기업 |

### 수익 모델

```
1. SaaS 구독 (월정액)
   - Basic: 공급지 5개, 월 50만원
   - Pro: 공급지 50개 + CBAM, 월 300만원
   - Enterprise: 무제한 + API + 전용 지원, 협의

2. 컨설팅 (초기 셋업)
   - 공급망 매핑 + 위험 평가: 1,000~3,000만원/건
   - Due Diligence 체계 구축: 2,000~5,000만원/건

3. 보고서 발급 (건당)
   - EUDR DDS 생성: 건당 30~50만원
   - CBAM Declaration 생성: 건당 50~100만원
```

## 로드맵

### Phase 1 (0~6개월): MVP + 초기 고객

- [ ] EUDR 위성검증 MVP (MODIS/Sentinel-2 + 규칙 기반 판정)
- [ ] CBAM 비용 시뮬레이터 MVP (수동 입력 → 비용 산출)
- [ ] 파일럿 고객 3~5개사 확보
- [ ] 정부 과제 지원 (중기부 TIPS, 산업부 탄소중립)

### Phase 2 (6~12개월): AI 고도화

- [ ] GAN 모델 학습 (계절성 보정 + 구름 제거)
- [ ] AI OCR 파인튜닝 (한국 수출입 서류 특화)
- [ ] Sentinel-2 10m 정밀 분석 파이프라인
- [ ] 자동 보고서 생성 시스템

### Phase 3 (12~24개월): 확장

- [ ] EU Forest Observatory 연동
- [ ] 실시간 모니터링 + 알림 시스템
- [ ] ASEAN 수출기업 확장 (베트남, 인도네시아)
- [ ] 추가 규제 모듈 (EU Battery Regulation, CSRD 등)

## 요약

EcoTrade는 "EU에 수출하려면 반드시 해야 하는데, 어떻게 해야 할지 모르는" 한국 기업에게
**AI로 자동화된 규제 대응 솔루션**을 제공한다.

GAN 기반 위성 분석으로 EUDR 산림검증의 정확도를 높이고,
AI OCR로 CBAM 서류 처리를 자동화하여,
한국 수출기업의 EU 시장 가격경쟁력을 지킨다.
