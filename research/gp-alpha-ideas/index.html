Research Ideas

Genetic Programming 기반 알파 탐색의 학술 확장 방향

Quality-Diversity, Cross-Market Transfer, Surrogate-Assisted Evolution, and Beyond
Ha Rim Jung
February 15, 2026
Overview
기존에 구현한 Island Model GP + SHADE + MAP-Elites + Surrogate 기반 알파 자동 탐색 시스템을 출발점으로, 퀀트 파이낸스 도메인에서 학술적으로 확장 가능한 연구 방향을 정리한다. 각 아이디어는 기존 문헌에서의 공백(gap), 핵심 가설, 실험 설계, 기대 기여를 포함한다. 선정 기준은 세 가지다: (1) 기존 퀀트 GP 문헌에서 다뤄지지 않은 희소성, (2) 실제 트레이딩 시스템에 적용 가능한 실용성, (3) 기존 프레임워크를 깨는 창의성.
Keywords: Genetic Programming, Quality-Diversity, MAP-Elites, Cross-Market Transfer, Surrogate-Assisted Optimization, Alpha Discovery, Portfolio Construction, Regime Detection, Minimum Description Length

1. Quality-Diversity Alpha Portfolios via MAP-Elites

High Novelty Rare Practical GECCO / IEEE CEC

Gap

기존 GP 기반 알파 탐색은 단일 목적(Sharpe ratio 최대화)에 집중한다. 그러나 실제 포트폴리오 구성에서는 서로 상관관계가 낮은 다양한 알파가 필요하다. 단일 목적 최적화는 구조적으로 유사한 고성능 알파를 반복 생성하며, 이들의 self-correlation이 높아 포트폴리오 편입 시 분산 효과가 제한된다.

핵심 가설

MAP-Elites의 행동 기술자(Behavior Descriptor)를 (Sharpe, Turnover, Factor Exposure)로 확장하면, 단일 목적 GP 대비 포트폴리오 수준의 Sharpe ratio가 유의하게 높아진다.

방법론

행동 공간을 3차원으로 확장한다: \(\mathcal{B} = (\text{Sharpe}_{10}, \text{Turnover}_{5}, \text{FactorExposure}_{4})\). 각 차원의 구간 수를 아래첨자로 표기하면 총 200셀의 아카이브가 된다. Factor Exposure는 Fama-French 5-factor 중 지배적 팩터의 인덱스로 이산화한다.

\[ \text{Portfolio Sharpe} = \frac{\mathbf{w}^\top \boldsymbol{\mu}}{\sqrt{\mathbf{w}^\top \boldsymbol{\Sigma} \mathbf{w}}} \]

여기서 \(\boldsymbol{\Sigma}\)는 MAP-Elites 아카이브에서 선택된 알파들의 수익률 공분산 행렬이다. 아카이브의 각 셀에서 최고 개체를 추출한 후, 평균-분산 최적화로 포트폴리오 가중치 \(\mathbf{w}\)를 결정한다.

실험 설계

(1) 비교군: 단일 목적 GP → top-N 선택 vs MAP-Elites → Markowitz 선택
(2) 평가: 포트폴리오 Sharpe, Maximum Drawdown, 알파 간 평균 상관계수
(3) 데이터: US equities TOP3000, 2010-2025, IS/OS 분할

기대 기여

Quality-Diversity 알고리즘이 퀀트 포트폴리오 구성에 적용된 최초 사례. 진화 연산 커뮤니티와 퀀트 커뮤니티 양쪽에 어필 가능.

2. Cross-Market Alpha Transfer via Island Model GP

High Novelty Rare Practical Quantitative Finance / Financial Innovation

Gap

GP 기반 트레이딩 규칙 탐색은 거의 예외 없이 단일 시장에서 수행된다. 한 시장에서 발견된 알파가 다른 시장에서도 작동하는지, 어떤 구조적 특성이 시장 간 전이 가능한지에 대한 체계적 연구가 부재하다. 이는 글로벌 퀀트 펀드의 핵심 질문임에도 불구하고 진화 연산 문헌에서 다뤄진 적이 없다.

핵심 가설

Island Model의 각 섬을 서로 다른 시장(USA, CHN, EUR, ASI)에 할당하고 이주(migration)를 통해 알파를 교환하면, 단일 시장 GP 대비 Out-of-Sample 성과가 개선되며, 전이 가능한 알파의 구조적 특성(사용 필드, 연산자 패턴)을 식별할 수 있다.

방법론

4개 섬을 USA, CHN, EUR, ASI에 각각 할당한다. 이주 시 Ring Topology 대신 Adaptive Topology를 도입한다: 이주 성공률(받는 섬에서의 성과 개선)이 높은 경로의 이주 확률을 증가시킨다.

\[ P(\text{migrate}_{i \to j}) = \frac{\text{success}_{i \to j} + \epsilon}{\sum_{k \neq j} (\text{success}_{k \to j} + \epsilon)} \]

전이 가능성(Transferability) 지표를 정의한다:

\[ T(\alpha) = \frac{1}{|\mathcal{M}|} \sum_{m \in \mathcal{M}} \mathbb{1}\left[\text{Sharpe}_m(\alpha) > \tau_m\right] \]

여기서 \(\mathcal{M}\)은 시장 집합, \(\tau_m\)은 시장 \(m\)의 제출 기준 Sharpe이다. \(T(\alpha) = 1\)이면 모든 시장에서 통과하는 "범용 알파"이다.

실험 설계

(1) 비교군: 단일 시장 GP × 4 vs Cross-Market Island GP
(2) 분석: 전이 성공률, 범용 알파의 구조 분석 (어떤 필드/연산자가 시장 불변인지)
(3) 가설 검증: 펀더멘탈 기반 알파가 기술적 알파보다 전이 가능성이 높은가

기대 기여

GP 기반 금융 전략 탐색에서 cross-market transfer를 다룬 최초 연구. 글로벌 퀀트 운용사의 실무적 질문에 직접 답하는 구조.

3. Surrogate-Assisted GP with Fidelity Control for Expensive Alpha Evaluation

Medium Novelty Practical Applied Soft Computing / Swarm and Evolutionary Computation

Gap

알파 시뮬레이션은 건당 3~15초가 소요되는 고비용 평가(expensive evaluation)이다. Surrogate-Assisted Evolutionary Optimization(SAEO)은 공학 최적화에서 활발히 연구되지만, 금융 GP에 적용된 사례는 극히 드물다. 특히 수식의 구조적 특징을 입력으로 하는 surrogate 모델의 설계와, 실제 평가와 surrogate 평가의 동적 배분(fidelity control)이 미탐색 영역이다.

핵심 가설

수식의 구조적 특징(연산자 빈도, 필드 카테고리, 트리 깊이, 윈도우 통계)으로 구성된 20차원 특징 벡터 기반 surrogate가, 실제 Sharpe의 순위를 Spearman \(\rho > 0.6\)으로 예측할 수 있으며, 하위 40% 필터링으로 API 호출을 40% 절감하면서 최종 포트폴리오 성과를 유지할 수 있다.

방법론

Multi-fidelity 프레임워크를 도입한다. 저비용 surrogate(XGBoost)와 고비용 실제 평가(BRAIN API)를 동적으로 배분한다:

\[ \text{Evaluate}(\alpha) = \begin{cases} f_{\text{surrogate}}(\alpha) & \text{if } \hat{\sigma}(\alpha) < \theta_{\text{conf}} \\ f_{\text{real}}(\alpha) & \text{otherwise} \end{cases} \]

여기서 \(\hat{\sigma}(\alpha)\)는 surrogate의 예측 불확실성이다. 불확실성이 높은 수식은 실제 평가로 보내고, 확실히 나쁜 수식은 surrogate만으로 필터링한다. 학습 데이터가 축적됨에 따라 \(\theta_{\text{conf}}\)를 적응적으로 조정한다.

실험 설계

(1) 비교군: 순수 GP vs Static Surrogate (고정 40% 필터) vs Adaptive Fidelity Control
(2) 평가: 동일 API 호출 예산 하에서의 최종 Sharpe, 발견된 제출 가능 알파 수
(3) Surrogate 정확도: Spearman \(\rho\), NDCG@10, 필터링 정밀도/재현율

기대 기여

SAEO를 금융 GP에 적용한 체계적 연구. API 호출 비용이 병목인 모든 플랫폼(WorldQuant, Quantopian 후속, QuantConnect 등)에 일반화 가능.

4. Regime-Aware Genetic Programming: 시장 국면별 알파 전환 시스템

High Novelty Rare Practical Journal of Financial Data Science / Expert Systems with Applications

Gap

대부분의 GP 기반 알파는 전체 기간에 대해 단일 수식을 최적화한다. 그러나 금융 시장은 비정상(non-stationary)이며, 특정 알파는 특정 국면(bull, bear, high-vol, low-vol)에서만 작동한다. 국면 전환을 GP 프레임워크에 내재화한 연구는 존재하지 않는다.

핵심 가설

Hidden Markov Model(HMM)로 식별된 시장 국면별로 독립적인 알파 풀을 진화시키고, 국면 전환 시 알파를 동적으로 교체하면, 단일 알파 대비 drawdown이 유의하게 감소하며 Sharpe가 개선된다.

방법론

2단계 시스템을 구성한다. 1단계에서 HMM(\(K=3\sim5\) states)으로 시장 국면을 식별한다. 관측 변수는 (수익률, 변동성, 거래량 변화율, 신용 스프레드)이다.

\[ s_t = \arg\max_{k} \; P(S_t = k \mid \mathbf{x}_{1:t}), \quad \mathbf{x}_t = (r_t, \sigma_t, \Delta v_t, \text{spread}_t) \]

2단계에서 각 국면 \(k\)에 대해 독립적인 Island를 할당하여 알파를 진화시킨다. 국면 \(k\)의 학습 데이터는 해당 국면에 속하는 기간만 사용한다. 실시간 운용 시 현재 국면 \(s_t\)에 해당하는 알파 풀에서 최고 알파를 선택한다.

\[ \alpha^*_t = \arg\max_{\alpha \in \mathcal{A}_{s_t}} \; \text{Score}(\alpha) \]

실험 설계

(1) 비교군: 전체 기간 단일 GP vs 국면별 GP + HMM 전환
(2) 국면 수 \(K\)에 대한 민감도 분석
(3) 국면 전환 지연(lag)이 성과에 미치는 영향
(4) 백테스트: 2010-2020 IS, 2020-2025 OS

기대 기여

GP와 regime detection의 결합은 기존 문헌에 부재. 실무적으로 drawdown 관리가 핵심인 기관 투자자에게 직접적 가치를 제공.

5. MDL-Guided Bloat Control: 정보이론 기반 수식 복잡도 정규화

Medium Novelty Rare Genetic Programming and Evolvable Machines / Evolutionary Computation

Gap

GP의 고질적 문제인 bloat(수식 비대화)는 금융 도메인에서 과적합과 직결된다. 기존 접근(Parsimony Pressure, 깊이 제한)은 수식의 의미적 복잡도를 반영하지 못한다. 노드 수가 같아도 사용하는 필드 수, 윈도우 파라미터 수, 변수 할당 수에 따라 실질적 복잡도가 크게 다르다.

핵심 가설

Minimum Description Length(MDL) 원리에 기반한 복잡도 측정이, 노드 수 기반 Parsimony Pressure 대비 IS/OS 일관성을 유의하게 개선한다.

방법론

수식의 기술 길이(Description Length)를 정보이론적으로 정의한다:

\[ L(H) = n_{\text{op}} \cdot \log_2 |\mathcal{O}| + n_{\text{field}} \cdot \log_2 |\mathcal{F}| + n_{\text{param}} \cdot \log_2 |\mathcal{W}| + n_{\text{var}} \cdot C_{\text{var}} \]

여기서 \(|\mathcal{O}|\)는 연산자 집합 크기, \(|\mathcal{F}|\)는 필드 집합 크기, \(|\mathcal{W}|\)는 윈도우 값 집합 크기, \(C_{\text{var}}\)는 변수 할당의 정보 비용이다. MDL 기반 적합도는:

\[ \text{Fitness}_{\text{MDL}}(\alpha) = \text{Sharpe}(\alpha) - \lambda \cdot L(H_\alpha) \]

\(\lambda\)는 복잡도 페널티 강도로, 교차 검증으로 결정한다.

실험 설계

(1) 비교군: 깊이 제한 only vs 노드 수 Parsimony vs MDL 정규화
(2) 평가: IS Sharpe, OS Sharpe, IS/OS 비율, 수식 해석 가능성(인간 평가)
(3) \(\lambda\) 민감도 분석 및 적응적 \(\lambda\) 스케줄링

기대 기여

MDL을 GP bloat control에 적용한 연구는 소수 존재하나, 금융 도메인에서의 실증은 부재. IS/OS gap 문제에 대한 이론적으로 정당화된 해법.

6. Semantic Backpropagation for Financial GP: 의미 기반 역전파 초기화

High Novelty Rare GECCO / PPSN / IEEE Transactions on Evolutionary Computation

Gap

GP의 초기 개체군은 대부분 완전 랜덤이거나 수작업 템플릿에 의존한다. Semantic Backpropagation(SB)은 목표 출력에서 역방향으로 각 노드의 "원하는 의미(desired semantics)"를 계산하여 서브트리를 라이브러리에서 매칭하는 기법이다. 심볼릭 회귀에서 성과가 입증되었으나, 금융 시계열에 적용된 사례가 없다.

핵심 가설

목표 수익률 벡터를 desired semantics로 설정하고, 각 서브트리의 출력 벡터를 실제 주가 데이터로 계산하여 매칭하면, 랜덤 초기화 대비 초기 세대의 평균 적합도가 유의하게 높아지고, 수렴 속도가 2~3배 빨라진다.

방법론

서브트리 라이브러리 \(\mathcal{L}\)을 구축한다. 각 서브트리 \(t_i\)에 대해 의미 벡터 \(\mathbf{s}_i = t_i(\mathbf{X})\)를 사전 계산한다. 여기서 \(\mathbf{X}\)는 학습 기간의 주가 데이터이다.

\[ t^* = \arg\min_{t_i \in \mathcal{L}} \; \| \mathbf{s}_i - \mathbf{s}_{\text{desired}} \|_2 \]

루트 노드의 desired semantics는 목표 수익률 벡터(예: 다음 날 수익률의 cross-sectional rank)이다. 이진 연산자의 경우, 한 자식의 semantics가 결정되면 다른 자식의 desired semantics를 역산할 수 있다:

\[ \mathbf{s}_{\text{desired}}^{\text{right}} = \mathbf{s}_{\text{desired}}^{\text{parent}} - \mathbf{s}_{\text{left}} \quad (\text{for } +) \]

실험 설계

(1) 비교군: 랜덤 초기화 vs 전략 템플릿 vs Semantic Backpropagation
(2) 평가: 세대별 수렴 곡선, 동일 API 예산 하 최종 성과
(3) 라이브러리 크기 \(|\mathcal{L}|\)에 대한 민감도

기대 기여

SB를 금융 GP에 적용한 최초 연구. 초기화 품질이 전체 탐색 효율에 미치는 영향을 정량화. 고비용 평가 환경에서 특히 가치가 높음.

7. Alpha Decay Prediction via Expression Structural Features

High Novelty Rare Practical Journal of Financial Economics / Quantitative Finance

Gap

알파의 수명(alpha decay)은 퀀트 운용의 핵심 리스크이다. 제출 시점에 높은 Sharpe를 보이던 알파가 수 개월 내에 성과가 소멸하는 현상은 보편적이다. 그러나 알파의 구조적 특성(수식 복잡도, 사용 필드, 연산자 패턴)과 수명 사이의 관계를 체계적으로 분석한 연구는 부재하다.

핵심 가설

수식의 구조적 특징(MDL 복잡도, 팩터 노출, 데이터 빈도, 윈도우 길이 분포)으로 알파의 반감기(half-life)를 예측할 수 있으며, 단순한 수식이 복잡한 수식보다 긴 수명을 가진다.

방법론

GP로 생성된 대량의 알파(수천 개)에 대해 시간에 따른 Sharpe 감쇠를 추적한다. 반감기를 다음과 같이 정의한다:

\[ t_{1/2} = \min \left\{ t : \text{Sharpe}_{[0,t]} < \frac{1}{2} \text{Sharpe}_{[0, T_{\text{IS}}]} \right\} \]

수식의 구조적 특징 벡터 \(\mathbf{f}(\alpha)\)와 반감기 \(t_{1/2}\) 사이의 관계를 생존 분석(Cox Proportional Hazards)으로 모델링한다:

\[ h(t | \mathbf{f}) = h_0(t) \exp(\boldsymbol{\beta}^\top \mathbf{f}(\alpha)) \]

실험 설계

(1) 데이터: GP로 생성된 알파 5,000개 이상, 각각 3년 이상의 OS 추적
(2) 특징: MDL 복잡도, 트리 깊이, 필드 카테고리 분포, 윈도우 평균/분산, 팩터 베타
(3) 검증: 예측된 장수명 알파 vs 단수명 알파의 실제 OS 성과 비교

기대 기여

알파 수명을 사전 예측하는 프레임워크. 제출 전 수명 예측으로 단기 소멸 알파를 필터링하여 포트폴리오 안정성 향상. 퀀트 실무에 즉시 적용 가능.

종합 비교

표 1: 연구 아이디어 비교
#주제희소성실용성난이도타겟 저널/학회
1QD Alpha Portfolios (MAP-Elites)높음높음중GECCO, IEEE CEC
2Cross-Market Transfer (Island)높음높음중Quant Finance, Financial Innovation
3Surrogate-Assisted GP중높음중Applied Soft Computing, SWEVO
4Regime-Aware GP (HMM)높음높음상JFDS, Expert Systems
5MDL Bloat Control중중중GPEM, Evolutionary Computation
6Semantic Backpropagation높음중상GECCO, PPSN, IEEE TEVC
7Alpha Decay Prediction높음높음상JFE, Quantitative Finance

1, 2, 4번은 기존 문헌에서 다뤄진 적이 없는 조합이라 novelty가 가장 높다. 3, 5번은 기존 기법의 금융 도메인 적용이라 상대적으로 안전한 선택이다. 7번은 데이터 축적이 필요하므로 장기 프로젝트에 적합하다.

이 아이디어들은 상호 배타적이지 않다. 예를 들어 1(MAP-Elites) + 4(Regime-Aware)를 결합하면 "국면별 다양성 최적화"라는 더 강력한 프레임워크가 된다. 2(Cross-Market) + 7(Decay Prediction)을 결합하면 "시장 간 전이 가능하면서 수명이 긴 알파"를 선별하는 시스템이 된다.
