## 이 글이 다루는 것

연령대별 소비 데이터를 비교 분석해 페르소나 클러스터를 만들고, 각 페르소나에 맞는 금융상품 추천 시나리오를 시각화한 프로토타입.

## 입력 데이터

- "소비 데이터 기반 '금융 페르소나' 맞춤형 서비스 제안하기 - 연령대별 비교.csv"
- 구매·결제·채널·카테고리 통계의 연령대별 비교

## 페르소나 설계

- 실제 소비 비중을 기반으로 4~5개 페르소나로 클러스터링.
- 페르소나마다 가장 부족한 금융 상품 카테고리(저축·투자·보험·결제)를 식별.
- 하나금융 상품군 매트릭스에 매핑해 추천 시나리오 작성.

## UI

- 모바일 우선. 페르소나 카드 → 상품 카드로 이어지는 한 손 스크롤.
- QR 시연용 정적 페이지 — 발표 자리에서 핸드폰으로 즉시 접속 가능.
