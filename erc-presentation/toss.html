§2.1 Risk Contributions
리스크 기여도 — 이 수식이 의미하는 것
$$\sigma_i(x) = \frac{x_i \cdot (\Sigma x)_i}{\sigma(x)}$$
기호의미실제로 뭘 보는가
$(\Sigma x)_i$자산 i와 포트폴리오의 공분산"이 자산이 포트폴리오와 얼마나 같이 움직이는가"
$x_i$현재 비중"이 자산에 얼마나 넣었는가"
$\sigma_i(x)$총 리스크 기여도"이 자산이 포트폴리오 변동성에 실제로 얼마나 기여하는가"
오일러 정리: $\sigma(x) = \sum_i \sigma_i(x)$
포트폴리오 전체 리스크 = 각 자산 RC의 합 (근사 아님, 수학적 등식)
주식 50% + 채권 50% 투자하면?
돈은 반반인데
리스크 기여도는 94 : 6
주식이 하루 2% 움직이고 채권이 0.3% 움직이면, 포트폴리오 손익의 거의 전부가 주식에서 옴
"분산 투자"라고 생각했는데 실제로는 주식 올인
ERC 조건
$$c_i = \frac{\sigma_i}{\sigma} = \frac{1}{n} \quad \forall\, i$$

§2.2 세 전략 비교
1/n vs MV vs ERC — 주가가 움직이면 어떻게 되는가
1/n 동일가중
조건: $x_i = 1/n$
"모든 자산에 같은 돈"
  • 주식 −2%, 채권 +0.3% → 포트폴리오 −0.85%
  • 손실의 거의 전부가 주식에서
  • 자본의 민주주의, 리스크의 독재
MV 최소분산
조건: $\partial\sigma/\partial x_i = \partial\sigma/\partial x_j$
"추가 1원 넣어도 리스크 증가 같다"
  • 채권에 94% 집중
  • 상승장에서도 +0.4%밖에 안됨
  • 사실상 채권 펀드
ERC 리스크 균등
조건: $x_i(\Sigma x)_i = x_j(\Sigma x)_j$
"각 자산의 리스크 기여도가 같다"
  • 주식 20%, 채권 80%
  • 위기 시 한쪽에 안 몰림
  • 리스크의 민주주의
수치 예시: 주식(σ=20%) + 채권(σ=5%), ρ=0

§3.1 2자산 닫힌 해
2자산 ERC = 변동성 역수 가중
$$x_1^* = \frac{\sigma_2}{\sigma_1 + \sigma_2}$$
변동성이 큰 자산은 비중 ↓, 작은 자산은 비중 ↑
예시: 삼성전자(σ=30%) + 미국 국채(σ=7%)
삼성전자 = 7/(30+7) = 19%
미국 국채 = 30/(30+7) = 81%
삼성전자가 하루 3% 움직이고 국채가 0.7% 움직이면,
같은 리스크 기여를 하려면 삼성전자를 적게 넣어야 함

상관 추정이 틀려도 ERC 비중은 변하지 않음

§3.2 등상관 케이스
등상관 → 변동성 역수 가중 (최적화 불필요)
$$x_i^* = \frac{\sigma_i^{-1}}{\sum_j \sigma_j^{-1}}$$
모든 자산 쌍의 상관이 같으면 (ρᵢⱼ = ρ ∀i≠j)
→ ERC 비중 = 변동성의 역수에 비례
실제 데이터로 보면
ETFσ1/σERC 비중 (근사)
IEF6.76%0.148높음 (~30%)
GLD18.45%0.054중간 (~9%)
VNQ33.19%0.030낮음 (~2%)
IEF가 하루 0.4% 움직이고 VNQ가 2% 움직이면,
IEF에 많은 돈, VNQ에 적은 돈 → "충격의 크기"를 같게
학술적 발견
ρ → −1/(n−1) 일 때 MV = ERC
"모든 자산이 최대한 반대로 움직이는" 극단적 분산 상황
→ 분산 효과 극대화 시 "리스크 최소화" = "리스크 균등화"
ERC는 "분산 효과가 극대화된 MV"로 해석 가능
등변동성 + 이상관 케이스 (식 4')
다른 자산과의 상관이 낮은 자산일수록 비중 ↑
금(GLD)이 주식과 거의 무상관(2.8%)이라
ERC에서 높은 비중을 받는 이유
→ ERC는 변동성만 보는 게 아니라 상관 구조도 활용
§3.3-3.4 일반적 경우
베타 역수 가중 & 수치 최적화
$$x_i^* = \frac{\beta_i^{-1}}{\sum_j \beta_j^{-1}}$$
$\beta_i = (\Sigma x^*)_i / \sigma(x^*)$
"자산 i가 ERC 포트폴리오와 얼마나 같이 움직이는가"
의미: 주가가 어떻게 움직이면?
  • 포트폴리오 빠질 때 같이 많이 빠지는 자산 (높은 β) → 비중 ↓
  • 덜 빠지거나 반대로 움직이는 자산 (낮은 β) → 비중 ↑
ETFβ 수준ERC 비중이유
EWY (한국)높음2.3%포트폴리오와 같이 움직임
GLD (금)낮음9.2%독립적으로 움직임
IEF (국채)매우 낮음29.8%반대로 움직임
수치 최적화: 논문 식 (6)
$$f(x) = \sum_i\sum_j\left(x_i(\Sigma x)_i - x_j(\Sigma x)_j\right)^2$$
"모든 자산 쌍의 RC 차이를 0으로 만들어라"
f(x*)=0 → 모든 자산의 RC가 같다 = ERC 조건
  • scipy SLSQP, 초기값 1/n
  • 131회 리밸런싱 모두 수렴
  • f(x*) < 10⁻²⁰
식 (7) 볼록 변환
전역 최적 보장하지만 구현 복잡
→ 논문 저자들도 식 (6) 선호
→ 실무에서도 식 (6)이 표준
def erc_objective(x, cov): sigma = np.sqrt(x @ cov @ x) rc = x * (cov @ x) / sigma return sum((rc[i]-rc[j])**2 for i in range(n) for j in range(n)) res = minimize(erc_objective, x0=1/n, method='SLSQP', bounds=[(0.001,1)]*n)
§3.5 최적성
ERC는 언제 최적인가?
모든 자산의 기대수익률이 같을 때
→ ERC = Maximum Sharpe Ratio
ERC의 암묵적 가정:
"기대수익률을 모르면
다 같다고 가정하는 게 합리적"
  • 주식이 연 8% 오를지 5% 오를지 아무도 모름
  • 잘못된 추정으로 MV 폭발하는 것보다
  • "다 같다"고 가정하고 리스크만 균등화하는 게 안전
ERC의 철학
"미래 수익률은 모르지만,
리스크는 측정할 수 있다.
그러니 리스크만 관리하자."
Bridgewater All Weather가 이 철학을 구현
"경제가 어떻게 변하든 리스크가 균등하면 견딜 수 있다"
2008 위기: All Weather −3.9% vs 60/40 −30%

논문의 핵심 기여 5가지
  • ① 수학적 정의: σᵢ(x) = σⱼ(x) ∀i,j
  • ② 닫힌 해: 2자산 x₁=σ₂/(σ₁+σ₂), 상관에 무관

Q&A
예상 질문 & 답변
Q1: 논문은 ERC 최고인데 왜 내 데이터는 MV?
백테스트 시작 시점 차이. 2008 위기 직전 시작 → MV 채권 집중 유리. 구조적 성질은 동일.
Q2: ERC가 MV보다 항상 좋은가?
아니다. 동질적 자산(주식만)에서는 MV 유리. ERC는 이질적 자산(주식+채권+대체)에서 강하다.
Q3: Risk Parity와 ERC는 같은가?
ERC = Risk Parity의 수학적 정의. 실무에서 Risk Parity = ERC + 레버리지. Bridgewater All Weather가 대표적.
Q4: MV가 IEF에 57.6% 집중하는 건 문제?
맞다. 가중치 Gini 82%. 단일 자산에 57% 넣는 것은 실무적으로 수용 불가. ERC는 Gini 38%로 훨씬 균등.
Q5: 60/40과 ERC의 차이는?
60/40: 자본 6:4, 리스크 110:−10 → 사실상 주식 올인. ERC: 리스크 50:50 → 진짜 분산.
Q6: 레버리지 적용하면?
ERC×1.74배 → 1/n과 동일 변동성에서 수익률 6.04% > 4.71%. Bridgewater 방식.
