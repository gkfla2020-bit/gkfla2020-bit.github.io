핵심 아이디어: "리스크를 균등하게 나누자"

피자를 4명이 나눠 먹는다고 합시다. 1/n은 "같은 크기로 자르자" — 매운 조각도 같은 크기. MV는 "가장 순한 조각만 한 사람에게" — 나머지는 굶음. ERC는 "매운 정도가 같도록 자르자" — 매운 건 작게, 순한 건 크게.

논문의 핵심 제안: 자본(돈)을 균등하게 나누는 대신, 각 자산이 포트폴리오 전체 리스크에 기여하는 양을 균등하게 만들자. 변동성이 큰 자산은 비중을 줄이고, 작은 자산은 비중을 높여서.

리스크 기여도(Risk Contribution)란?

"각 자산이 포트폴리오 전체 리스크에 얼마나 기여하는가"를 정확히 측정하는 지표

1/n 포트폴리오 (13개 ETF, 각 7.69%)

돈은 균등한데 리스크는 극도로 불균등!

ERC 포트폴리오

모든 자산이 정확히 7.69%씩 리스크 기여

오일러 정리: 포트폴리오 전체 리스크 = 각 자산의 리스크 기여도의 합 (근사가 아니라 정확한 등식)
→ 이 분해가 가능하기 때문에 "리스크를 균등하게 나누자"는 전략이 수학적으로 정의될 수 있다.

논문의 발견 ①: 2자산 ERC는 놀랍도록 단순하다

\[x_1^* = \frac{\sigma_2}{\sigma_1 + \sigma_2}, \quad x_2^* = \frac{\sigma_1}{\sigma_1 + \sigma_2}\]

변동성이 큰 자산은 비중을 줄이고, 작은 자산은 비중을 높인다. 그게 전부.

예시: 삼성전자(σ=30%) + 미국 국채(σ=7%)

19%
삼성전자 (변동성 큼 → 적게)
81%
미국 국채 (변동성 작음 → 많이)
시소 균형과 같습니다. 무거운 사람(변동성 큰 자산)은 중심에 가까이, 가벼운 사람은 멀리 앉아야 균형이 맞습니다.

핵심: 상관계수에 무관!

ERC(실선)는 수평선, MV(점선)는 상관에 따라 크게 변동

왜 중요한가: 상관계수는 추정하기 가장 어려운 파라미터다. 주식-채권 상관은 2000년대에 음수였지만 1990년대에는 양수였다. MV는 이 불안정한 상관에 민감하지만, ERC는 (2자산에서) 완전히 면역이다.

논문의 발견 ②: 일반적 경우 — "베타 역수 가중"

3자산 이상에서는 상관도 영향을 미친다. 하지만 해석은 여전히 직관적:

\[x_i^* \propto \frac{1}{\beta_i} \quad \text{(포트폴리오에 대한 베타의 역수)}\]
직관: 포트폴리오가 빠질 때 같이 많이 빠지는 자산(높은 β) → 비중 ↓
포트폴리오와 독립적으로 움직이는 자산(낮은 β) → 비중 ↑

논문이 말하는 것: ERC는 MV의 "리스크 최소화"와 1/n의 "완벽한 분산" 사이의 최적 타협점이다. 리스크를 조금 더 감수하는 대가로, 극단적 집중을 피하고 거래비용을 절감한다.

논문의 발견 ④: ERC는 언제 "최적"인가?

모든 자산의 기대수익률이 같다고 가정하면, ERC = 최고 샤프 비율 포트폴리오

왜 이 가정이 합리적인가?

기대수익률 추정의 어려움

  • Merton(1980): 유의미한 추정에 수백 년 데이터 필요
  • 0.1% 추정 오류 → MV 가중치 50% 이상 변동
  • DeMiguel et al.(2009): 14개 최적화 모델 중 1/n이 out-of-sample 최고
"미래 수익률은 모르지만, 리스크는 측정할 수 있다. 그러니 리스크만 관리하자." — ERC의 철학

공분산 추정은 상대적으로 안정적

파라미터추정 난이도필요 데이터
기대수익률 μ매우 어려움수백 년
변동성 σ상대적 쉬움1~3년
상관 ρ중간2~5년

→ ERC는 μ 불필요, σ와 ρ만 사용 → 추정 오류에 강건

Bridgewater All Weather: "경제 환경이 어떻게 변하든 리스크가 균등하면 포트폴리오가 견딜 수 있다" — Ray Dalio

왜 이질적 자산에서 ERC가 이기는가?

  • MV: 채권에 80%+ 집중 → 주식 상승장 수혜 못 받음 → 수익률 희생
  • 1/n: 주식에 과다 리스크 배분 → 위기 시 폭락 (−34%)
  • ERC: 채권의 안정성 + 주식의 성장성 균형 → 위기 방어 + 상승 참여
MV는 "비 올 때 우산" — 비(위기) 올 때 최고지만 맑은 날 짐이 됨.
1/n은 "항상 반팔" — 여름(상승장) 최고지만 겨울(위기)에 얼어 죽음.
ERC는 "날씨에 맞게 옷 입기" — 극단적으로 좋지는 않지만 어떤 날씨에서든 합리적.

하단 (RC): ERC(굵은 선) = 항상 7.69%로 수평선. 1/n(점선) = EEM 16%, TLT −1.3%로 극도로 불균등
→ 가중치가 아무리 변해도 리스크 기여도는 항상 균등 — 이것이 ERC의 본질

ERC는 "올라운더" 선수와 같습니다. 공격(상승장)에서 최고는 아니지만 수비(위기)도 합니다. 수비 전문(MV)만큼 방어적이지는 않지만 공격도 참여합니다. 어떤 경기(시장 환경)에서든 합리적인 성과를 냅니다.

결론: 이 논문이 말하는 것

"We propose an alternative approach based on equalizing risk contributions from the various components of the portfolio." — p.18

논문의 5가지 핵심 기여

  1. 수학적 정의: "모든 자산의 리스크 기여도가 같다" = ERC
  2. 닫힌 해: 2자산에서 변동성 역수 가중, 상관에 무관
  3. 수치 최적화: 직접 최적화(식 6) + 볼록 변환(식 7)
  4. 부등식: σ_MV ≤ σ_ERC ≤ σ_1/n (항상 성립)
  5. 실증: 이질적 자산군에서 ERC 샤프 최고 (0.67)

내 실증이 확인한 것

부등식 ✓ | RC Gini ≈ 0 ✓ | MV 극단 집중 ✓ | ERC 턴오버 절반 ✓ | ERC > 1/n ✓

한 문장 요약: "자본이 아니라 리스크를 균등하게 나누면, 어떤 시장 환경에서든 합리적인 성과를 낼 수 있다."

역사적 의의: Bridgewater All Weather ($150B+), AQR Risk Parity, Invesco Balanced-Risk 등 수조 달러 규모 전략의 수학적 기초. Google Scholar 인용 1,500+.

Q&A 예상 질문

Q: 왜 내 데이터에서는 MV가 최고?

백테스트 시작 시점 차이. 2008 위기 직전 → MV의 채권 집중이 극도로 유리.

Q: ERC가 항상 좋은가?

아닙니다. 동질적 자산군(주식만)에서는 MV가 더 좋습니다. ERC는 이질적 자산군에서 강합니다.

Q: Risk Parity = ERC?

ERC는 수학적 정의. 실무에서 Risk Parity = ERC + 레버리지 (Bridgewater All Weather).

Q: MV의 IEF 57.6% 집중은 문제?

맞습니다. 가중치 Gini 82%. 단일 자산 57% 집중은 실무적 수용 불가. ERC는 38%로 훨씬 균등.

Q: 레버리지 적용하면?

ERC × 1.74배 → 1/n과 동일 변동성에서 수익률 6.04% > 4.71%. Bridgewater 방식.

Q: 한국에서 구현 가능?

KODEX 200 + TIGER 미국채10년선물 + KODEX 골드선물로 간단한 ERC 구현 가능.

실무 시사점 & 마무리

  • 멀티에셋 포트폴리오 → ERC 적극 활용 (이질적 자산군에서 1/n을 지배)
  • MV의 함정 → 샤프 높지만 IEF 57.6% 집중은 실무적 수용 불가
  • 거래비용 → ERC 턴오버 = MV의 절반
  • 위기 대응 → ERC는 리스크 균등화로 자동 방어 (별도 타이밍 모델 불필요)
  • 레버리지 → ERC × 1.74 = 동일 변동성에서 1/n보다 높은 수익률
